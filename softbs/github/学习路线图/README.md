# 学习路线图

## 使用顺序

1. 完成[公共环境与模型配置](公共环境与模型配置.md)。
2. 按[第一批学习路线](第一批学习路线.md)依次学习五个工具。
3. 每完成一篇教程，更新 [`../学习进度/总进度.md`](../学习进度/总进度.md)。
4. 新增项目时使用[新增工具模板](新增工具模板.md)。

## 完成标准

- 能解释工具解决的问题和不适用场景。
- 能在 WSL Ubuntu 中从零安装并验证。
- 能在 VSCode 与 Codex 工作流中使用。
- 能在 OpenRouter、ChatNVIDIA 和 Ollama 之间按优先级切换。
- 能保存输出、日志、评估结果与复盘记录。
