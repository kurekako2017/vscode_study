# GitHub 学生版工具概览

参考：GitHub Education — Student Developer Pack
https://education.github.com/pack

概述：
- GitHub 学生开发包为已通过学生认证的用户提供多家厂商的免费或优惠资源（云资源、IDE、域名、设计工具、课程等），适合用来做学习、练习和作品集项目。

你最常会用到的类别：
- 域名：Name.com、Namecheap（通常提供域名优惠或赠送域名，具体以当期官方页面为准）
- 云与主机：DigitalOcean、Vercel、Netlify（用于部署网站、API 和小型服务）
- 开发者工具：JetBrains、GitHub Pro、Codespaces（用于编码、远程开发和提升效率）
- CI/CD 与部署：GitHub Actions、Travis/Circle（用于自动构建、测试和发布）
- 抓取与数据：Zyte Scrapy Cloud（用于运行和管理网络爬虫，适合大规模采集与自动化调度）
- 设计与原型：Figma、Canva（用于页面设计、原型图和演示素材）
- 学习资源：在线课程、训练营、教程优惠（用于系统学习和提升技能）

这些工具是用来做什么的：
- 域名工具：申请自己的项目域名，让作品集、个人站点或小项目看起来更完整。
- 云主机工具：把网站、后端接口、数据库等部署到公网，方便访问和展示。
- 开发者工具：提升写代码效率，例如更强的 IDE、在线开发环境或云端工作区。
- CI/CD 工具：自动完成测试、构建和部署，减少手工操作。
- 抓取平台：专门运行和管理网络爬虫，适合批量抓取、调度和自动化采集。
- 设计工具：制作页面原型、图形素材和演示文档。
- 学习资源：帮助你更快入门新技术或补足某个方向的基础。

主要类别与示例（以官方页面为准）：
- 云与主机：DigitalOcean（额度）、Netlify / Vercel（托管优惠）
- 开发者工具：JetBrains（免费许可证）、GitHub Pro、Codespaces 优惠
- CI/CD 与部署：Travis/Circle/GitHub Actions（加速或额度）
- 域名与证书：Name.com、Namecheap（域名优惠或赠送域名）、SSL 证书优惠
- 抓取与数据：Zyte Scrapy Cloud（网络爬虫托管与自动化处理）
- 设计与原型：Canva / Figma（教育优惠或付费功能）
- 学习资源：Front-end/Back-end 教程、在线课程优惠

如何申请：
1. 准备在校证明（学校邮箱或学生证照片等），并登录你的 GitHub 账户。
2. 打开官方页面并点击“Get benefits” / “学生包申请”，按要求填写并上传证明。
3. 等待审核通过（通常几小时到几天）。审批通过后，可在页面内查看并领取各项优惠。

使用建议：
- 先查看你确实需要的类别（例如：需要云主机就先领取 DigitalOcean Credit，想做个人站点就先看 Name.com / Namecheap 域名权益）。
- 每项优惠通常有使用期限或额度限制，使用前阅读对应厂商条款。
- 在仓库内添加 `README` 或 `docs` 记录已领取的服务与用途，方便复现与项目管理。
