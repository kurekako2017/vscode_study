## 标记说明

| 标记 | 含义 |
| --- | --- |
| ❌ | 未开始或尚未验收 |
| ⏳ | 进行中 |
| ✅ | 已验收 |

## 状态规则

| 状态 | 使用条件 |
| --- | --- |
| 未开始 | 尚未进入安装或学习 |
| 学习中 | 任一阶段为 ⏳，或部分阶段已完成 |
| 已完成 | 安装、学习、实战全部 ✅；如计划集成 AI-Learn，集成也应完成 |
| 已放弃 | 经实战证明不匹配需求，并在《AI工具决策记录》中写明原因 |

## 新工具自动追加规则

本看板是新增工具流程的必更项。每次按 [`学习路线图/新增工具模板.md`](../学习路线图/新增工具模板.md) 新增工具时，同步追加下面这一行：

```md
| <工具名> | ❌ | ❌ | ❌ | ❌ | 未开始 |
```

不从看板删除已停止学习的工具；将状态改为“已放弃”，保留决策历史。
