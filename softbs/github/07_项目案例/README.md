# 07 项目案例

## 目录职责

放置已完成验收的跨工具项目案例。单一工具的学习案例仍属于 `03_工具实战`；只有产生 AI-Learn 可复用能力的联合项目才进入本目录。

## 候选案例

- [项目案例索引](./项目案例索引.md)
- [Student Pages Demo](./student-pages-demo/README.md)

1. OpenHands + Codex：AI-Learn Bug 复现、修复、测试和独立审查。
2. Codebase Memory MCP + OpenHands：基于代码图的变更影响分析。
3. RAGFlow + AI-Learn：带引用的研发知识助手。
4. browser-use + RAGFlow：经审核的公开文档增量采集与索引。
5. OpenClaw + 上述能力：带工具白名单和人工批准的多 Agent 控制平面。

## 案例验收要求

- 有可重复的输入、环境和执行命令。
- 有功能、质量、延迟、成本和安全指标。
- 有日志、追踪或审计证据。
- 有人工审批点和回滚步骤。
- 已填写学习复盘并更新状态看板。
