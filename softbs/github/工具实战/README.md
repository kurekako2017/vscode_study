## 学习原则

- 先在测试仓库、测试账号和限制域名中运行。
- 所有 Agent 的第一次执行都保留确认步骤。
- 每次只改一个变量：环境、工具、模型或任务。
- 先记录基线，再优化速度、质量和成本。
