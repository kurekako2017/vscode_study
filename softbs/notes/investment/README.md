# Investment 投资研究资料库

本目录用于集中管理投资研究学习路线、工具调研资料和未来项目设计。

> 本目录内容仅用于学习和研究，不构成投资建议。不得在文档中保存证券账户密码、API Key 或其他敏感凭据。

## 目录结构

| 目录 | 用途 | 当前内容 |
| --- | --- | --- |
| [`roadmap/`](roadmap/) | 学习路线、总体指南和阶段规划 | 2 份文档 |
| [`research/`](research/) | 工具、框架和开源项目调研 | 1 份文档 |
| [`projects/`](projects/) | 可落地的系统设计与实战项目 | 2 份文档 |

## 推荐入口

1. 先阅读 [投资体系导航](投资体系导航.md)，了解整体关系。
2. 再查看 [INDEX](INDEX.md)，按分类进入具体文档。
3. 学习顺序建议采用“路线图 → 工具研究 → 项目设计 → 实战开发”。

## 维护规则

- 新文档先判断是路线图、研究资料还是项目资料，再放入对应目录。
- 同一文档涉及多个主题时，只保留一个主文件，通过 `INDEX.md` 建立交叉入口。
- 移动或重命名文件前，先检查相对链接并获得用户确认。
- 不保存真实投资账户信息、密钥或个人交易记录。
- 不自动执行 Git 提交或推送。
