# 消費税辞典

## 使用说明

本辞典用于快速理解消费税术语和 freee 中的常见入口。菜单以2026年6月确认的官方帮助为参考，方案或版本不同可能改变显示位置。

| 用语 | 日文解释 | 中文解释 | freee中的对应位置 |
| --- | --- | --- | --- |
| 課税事業者 | 消費税の申告・納税義務を負う事業者 | 需要申报缴纳消费税的经营者 | freee会計：消費税の基本設定；freee申告：`消費税 → 基本情報` |
| 免税事業者 | 原則として消費税の納税義務が免除される事業者 | 原则上免除消费税纳税义务的经营者 | freee会計：消費税的事业者区分设置 |
| 課税売上 | 消費税が課される売上取引 | 需要计算销项消费税的销售 | 取引登记的税区分；`決算申告 → 消費税区分別表` |
| 課税仕入 | 仕入税額控除の対象となり得る仕入取引 | 可能用于进项税额抵扣的采购 | 取引登记的税区分；`マスタ・口座 → 税区分` |
| 標準税率10% | 原則的な消費税等の税率 | 一般商品和服务使用的10%合计税率 | 取引登记 → 税率 `10%` |
| 軽減税率8% | 一定の飲食料品等に適用される税率 | 符合条件的饮食料品等使用的8%税率 | 取引登记 → 税率 `8%（軽）` |
| 旧税率8% | 経過措置等で使用される旧税率 | 经过措施等适用的旧8%税率 | 旧税率对应税区分；不能与8%轻减混用 |
| 税区分 | 取引の消費税上の性質を示す分類 | 表示交易消费税性质的分类 | `マスタ・口座 → 税区分` |
| インボイス | 適格請求書発行事業者が交付する適格請求書 | 满足法定记载要求的合格发票 | 取引、ファイルボックス、取引先的登录编号与控除区分 |
| 仕入税額控除 | 売上税額から一定の仕入税額を控除する仕組み | 从销项税额中扣除符合条件的进项税额 | freee申告消費税计算、消費税区分別表 |
| 控80・控50 | 免税事業者等からの仕入れに係る経過措置の控除割合 | 从免税经营者等采购时的阶段性抵扣比例 | `課対仕入（控80/控50）` 等税区分 |
| 原則課税 | 売上税額から実際の仕入税額を控除して計算する方式 | 按实际销项和进项税额计算 | freee会計消费税设置；freee申告 `消費税 → 基本情報` |
| 簡易課税 | みなし仕入率を用いて納付税額を計算する制度 | 使用视同进项率计算税额的制度 | freee会計消费税课税方式；需要事前届出确认 |
| 2割特例 | 一定のインボイス登録事業者向け負担軽減措置 | 面向符合条件的登记经营者的阶段性减负措施 | freee申告消费税基本信息；适用条件需按年度确认 |
| 仮受消費税 | 売上時に預かった消費税を管理する科目 | 管理销售时收取的消费税 | freee会計总账、试算表（税抜经理） |
| 仮払消費税 | 仕入時に支払った消費税を管理する科目 | 管理采购时支付的消费税 | freee会計总账、试算表（税抜经理） |
| 未払消費税 | 申告で確定した未納の消費税を管理する科目 | 管理已确定但尚未缴纳的消费税 | freee会計总账、BS、决算振替 |
