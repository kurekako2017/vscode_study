# 法人税辞典

## 使用说明

本辞典用于理解法人决算和法人税申告中的主要术语。实际表格、税率和支持功能必须按事业年度与 freee 当前方案确认。

| 用语 | 日文解释 | 中文解释 | freee中的对应位置 |
| --- | --- | --- | --- |
| 決算書 | 事業年度の財政状態と経営成績を示す書類 | 反映年度财务状态和经营成果的报表 | freee会計：`決算申告`；freee申告：`法人税 → 決算書` |
| 貸借対照表（BS） | 決算日時点の資産・負債・純資産を示す表 | 反映决算日资产、负债和净资产 | freee会計：レポート、決算書 |
| 損益計算書（PL） | 一定期間の収益・費用・利益を示す表 | 反映期间收入、费用和利润 | freee会計：レポート、決算書 |
| 当期純利益 | 会計上の最終的な当期利益 | 会计口径的本期净利润 | PL、freee申告法人税的決算数据联动 |
| 所得金額 | 会計利益を税務調整した課税所得 | 对会计利润进行税务调整后的所得 | freee申告：法人税申告書、別表四 |
| 益金 | 法人税法上、所得計算に算入する収益 | 法人税法上计入所得的收益 | freee申告：別表四及相关別表 |
| 損金 | 法人税法上、所得計算から控除する費用・損失 | 法人税法上允许扣除的费用或损失 | freee申告：別表四及相关別表 |
| 加算 | 会計利益に税務上加える調整 | 税务上调增会计利润 | freee申告：別表四 |
| 減算 | 会計利益から税務上差し引く調整 | 税务上调减会计利润 | freee申告：別表四 |
| 留保 | 税務調整が将来解消し得る形で社内に残る区分 | 可能在未来转回、留在企业内部的调整 | freee申告：別表四、別表五（一） |
| 社外流出 | 配当等、法人外へ流出し原則留保されない区分 | 已流出企业、通常不形成以后转回的分类 | freee申告：別表四 |
| 別表一 | 所得と法人税額を集約する申告書 | 汇总所得和法人税额的主表 | freee申告：`法人税 → 申告書` |
| 別表四 | 会計利益から税務所得を計算する明細書 | 从会计利润调整为税务所得的表 | freee申告：`法人税 → 申告書` |
| 別表五（一） | 利益積立金額・資本金等の異動を管理する明細書 | 管理税务净资产及资本变动的表 | freee申告：`法人税 → 申告書` |
| 別表五（二） | 租税公課の発生・納付・未納を管理する明細書 | 管理税金发生、缴纳和未缴状态的表 | freee申告：`法人税 → 申告書` |
| 中間申告 | 事業年度途中に行う法人税等の申告 | 事业年度中途进行的预缴申报 | freee申告：`法人税 → 申告一覧`；方案限制需确认 |
| 確定申告 | 事業年度終了後に税額を確定する申告 | 年度结束后确定税额的申报 | freee申告：`法人税 → 申告一覧` |
| 防衛特別法人税 | 令和8年4月以後の適用関係を確認すべき新税目 | 需要按适用年度确认的新法人税相关税目 | freee申告最新法人税样式；旧年度画面可能不显示 |
