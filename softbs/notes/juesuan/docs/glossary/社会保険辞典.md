# 社会保険辞典

## 使用说明

本辞典用于理解健康保险、厚生年金、工资和年末调整中的常见术语。社会保险等级、保险料和提交条件应以当年度日本年金机构、加入健康保险组织及 freee 官方信息为准。

| 用语 | 日文解释 | 中文解释 | freee中的对应位置 |
| --- | --- | --- | --- |
| 被保険者 | 健康保険・厚生年金保険の適用を受ける人 | 参加健康保险、厚生年金的人员 | freee人事労務：従業員信息、社会保険设置 |
| 標準報酬月額 | 保険料計算の基礎となる等級化された報酬月額 | 用于计算社会保险费的标准报酬等级 | freee人事労務：従業員的社会保险信息 |
| 算定基礎届 | 4～6月の報酬を届け出て定時決定を行う書類 | 根据4月至6月报酬重新决定标准报酬的表格 | freee人事労務：算定基礎届作成画面（方案不同可能改变） |
| 定時決定 | 毎年1回標準報酬月額を見直す手続 | 每年定期重新决定标准报酬月额 | freee人事労務：算定基础届流程 |
| 月額変更届 | 固定的賃金の大幅変動時に随時改定する届書 | 固定工资显著变化时进行随时改定的表格 | freee人事労務：月額変更届作成画面 |
| 随時改定 | 定時決定を待たず標準報酬月額を変更する手続 | 不等待年度定时决定而调整标准报酬 | freee人事労務：月额变更流程 |
| 固定的賃金 | 基本給・固定手当等、継続的に一定額で支給する賃金 | 基本工资、固定津贴等持续固定支付项目 | freee人事労務：給与規定、手当设置 |
| 支払基礎日数 | 報酬支払の基礎となった日数 | 用于判断报酬月份是否纳入计算的天数 | freee人事労務：算定基础届对象月份资料 |
| 健康保険料 | 健康保険制度のための保険料 | 健康保险费用 | freee人事労務：給与明細、社会保険控除 |
| 厚生年金保険料 | 厚生年金制度のための保険料 | 厚生年金保险费 | freee人事労務：給与明細、社会保険控除 |
| 事業主負担 | 会社が負担する社会保険料部分 | 企业承担的社会保险费 | freee人事労務：給与計算；freee会計联动仕訳 |
| 被保険者負担 | 給与から控除する本人負担部分 | 从工资中扣除的个人承担部分 | freee人事労務：給与明細的控除 |
| 年末調整 | 年間の給与所得税の過不足を精算する手続 | 年末结算全年工资所得税差额 | freee人事労務：`年末調整` |
| 源泉徴収票 | 年間給与・控除・源泉税額を記載する書類 | 记载全年工资、扣除和预扣税额的凭证 | freee人事労務：年末調整完成后输出；`設定 → 書類` 等 |
| 給与支払報告書 | 給与支払者が市区町村へ提出する書類 | 工资支付者向市区町村提交的报告 | freee人事労務输出、eLTAX提交 |
| 特別徴収 | 住民税を給与から控除し会社が納入する制度 | 公司从工资代扣并缴纳居民税的制度 | freee人事労務：住民税控除设置、給与明細 |
