# 消費税申告完整案例

## 1. 实例目的

演示原则课税下如何按税率整理课税销售、课税仕入和消费税概算，并准备法人消费税确定申告。

> 本例是学习用概算。正式申告需要区分国税与地方消费税、课税标准、端数处理、仕入税额控除、インボイス、中间缴纳及各种调整，不能直接用本例数字申告。

## 2. 公司与申告前提

| 项目 | 案例条件 |
| --- | --- |
| 法人 | 本案例的一人 IT・SES 株式会社 |
| 课税期间 | 2025-05-01～2026-04-30 |
| 课税方式 | 原则课税 |
| 经理方式 | 税込经理 |
| インボイス | 假设必要资料齐全 |
| 中间缴纳 | 100,000円 |

## 3. 税率别业务数据

### 课税销售

| 税率 | 税抜销售 | 消费税等概算 |
| --- | ---: | ---: |
| 10% | 8,000,000円 | 800,000円 |
| 8%轻减 | 1,000,000円 | 80,000円 |
| 合计 | 9,000,000円 | 880,000円 |

### 课税仕入

| 税率 | 税抜仕入 | 消费税等概算 |
| --- | ---: | ---: |
| 10% | 3,000,000円 | 300,000円 |
| 8%轻减 | 500,000円 | 40,000円 |
| 合计 | 3,500,000円 | 340,000円 |

### 学习用概算

```text
销售消费税等概算          880,000
－ 仕入消费税等概算       340,000
= 差额概算                540,000
－ 中间缴纳               100,000
= 差引缴纳额             440,000
```

440,000円不是申告书最终税额。正式计算必须按申告书和付表分别计算消费税、地方消费税及端数。

## 4. freee准备步骤

1. 确认课税事业者状态和课税方式。
2. 确认全部交易的税区分。
3. 分别集计10%、8%轻减及必要时旧8%。
4. 确认适格请求书和账簿保存要求。
5. 排除不能进行仕入税额控除的项目。
6. 确认中间申告和中间缴纳。
7. 输出消费税集计资料。

## 5. 申告资料制作

原则课税的一般用申告通常需要根据实际情况准备：

- 消费税及地方消费税确定申告书
- 课税标准额等内訳书（第二表）
- 税率别消费税额计算表及相关付表
- 仕入税额控除相关明细
- 中间缴纳资料

### 操作顺序

1. 把税率别销售转入申告基础资料。
2. 计算国税消费税部分。
3. 反映可控除仕入税额。
4. 反映贷倒、返还、调整对象固定资产等适用项目。
5. 计算地方消费税。
6. 反映中间缴纳。
7. 确认差引税额。
8. 通过 e-Tax 发送。
9. 保存接收通知并完成缴纳。

## 6. 未払消费税处理

申告税额确定后，根据公司的经理方式确认决算仕訳。概念示例：

```text
申告确定时：确认仮受消费税、仮払消费税与未払消费税之间的振替
实际缴纳时：借方 未払消费税 / 贷方 普通预金
```

具体金额和科目组合必须与 freee 设置、申告书及中间缴纳一致。

## 7. 验收清单

- [ ] 课税方式和届出状态正确
- [ ] 10%、8%轻减、旧8%已区分
- [ ] インボイス和账簿保存要件已确认
- [ ] 不可控除仕入没有误计
- [ ] 中间缴纳已反映
- [ ] 付表和申告书一致
- [ ] freee集计与申告数据一致
- [ ] 接收通知和缴纳记录已保存

## 8. 常见错误

- 直接用含税销售乘税率作为最终申告税额。
- 把8%轻减和旧8%混在一起。
- 所有仕入税额都自动全额控除。
- 忘记中间缴纳。
- 发送完成后忘记实际缴纳。
