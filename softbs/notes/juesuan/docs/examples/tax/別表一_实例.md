# 別表一完整填写案例

## 1. 实例目的

別表一用于汇总法人税课税所得、计算税额、控除额、中间缴纳额和最终应缴金额。

> 本案例按 2025-05-01 至 2026-04-30 事业年度的一人 IT・SES 法人设计。正式提交前必须用该年度国税庁样式和申告软件复算。

| `事業内容` | 业务 | IT・SES、系统开发 |

| `青色申告` | 蓝色申告 | 本案例设为承认有效，执行时核对届出 |

## 4. 填写数据

| 日文项目 | 中文解释 | 金额 | 来源 |
| --- | --- | ---: | --- |
| `当期利益` | 会计税后纯利益 | 699,700円 | PL |
| `加算合計` | 税务加算合计 | 350,300円 | 別表四 |
| `減算合計` | 税务减算合计 | 20,000円 | 別表四 |
| `所得金額` | 法人税所得 | 1,030,000円 | `699,700 + 350,300 - 20,000` |
| `法人税額` | 法人税 | 154,500円 | 本案例适用税率计算 |
| `控除税額` | 控除税额 | 0円 | 別表六等；本例无 |
| `中間申告税額` | 中间缴纳额 | 50,000円 | 缴纳记录、別表五（二） |

## 5. 填写顺序

1. 填写法人名称、法人编号、所在地和事业年度。
2. 从別表四转入所得金额或欠损金额。
3. 使用当前年度规则计算法人税额。
4. 从相关别表转入税额控除。
5. 填写中间申告和中间缴纳金额。
6. 计算差引确定法人税额。
7. 确认地方法人税及当年度新增税目栏位。
8. 与別表五（二）的当期发生和缴纳状态核对。
9. 与 e-Tax 计算结果进行最终比较。

### freee 会计数据准备步骤

1. 在 freee 打开 `決算申告` → `決算書`。
2. 将事业年度设为 2025-05-01 至 2026-04-30。
3. 输出确定版貸借対照表和損益計算書。
4. 确认損益計算書当期纯利益为 699,700円。
5. 打开 `会計帳簿` → `総勘定元帳`，确认法人税等 300,300円。
6. 保存中间缴纳 50,000円的银行明细和仮払法人税等记录。
7. 将 freee 决算书与別表四所得 1,030,000円核对。
8. 确认未払法人税等 250,300円与別表五（二）一致。

## 6. e-Tax 软件填写步骤

1. 打开目标年度法人税申告数据。
2. 选择 `別表一` 的中小普通法人用样式。
3. 输入法人名称、法人番号、纳税地和事业年度。
4. 确认资本金 5,000,000円和股份资料与登記、別表五（一）一致。
5. 从別表四联动所得金额 1,030,000円。
6. 确认法人税额计算结果 154,500円。
7. 输入控除税额 0円。
8. 输入中间申告税额 50,000円。
9. 确认差引确定法人税额 104,500円。
10. 确认地方法人税额 15,800円。
11. 检查当年度是否出现防卫特别法人税等适用栏位。
12. 执行形式检查并修正全部错误。
13. 与別表五（二）和未払法人税等总账核对。

## 7. 表间联动

```text
PL 当期纯利益
  ↓
別表四：税务加算、减算
  ↓
別表一：所得与税额
  ↓
別表五（二）：税金发生、缴纳、未缴
```

- [ ] 事业年度与决算书一致
- [ ] 所得金额与別表四一致
- [ ] 中间缴纳与缴纳记录一致
- [ ] 税额与別表五（二）一致

- [ ] 已确认防卫特别法人税适用与否
- [ ] e-Tax形式检查无错误
