# 別表一 填写实例（Mock）

## 1. 实例目的

別表一用于汇总法人税课税所得、计算税额、控除额、中间缴纳额和最终应缴金额。

> 本实例全部使用虚构数据，只说明资料流向。表格栏位、税率和防卫特别法人税适用关系必须按实际事业年度使用国税厅最新样式确认。

## 4. Mock 数据流

| 项目 | Mock 金额 | 来源 |
| --- | ---: | --- |
| 会计当期纯利益 | 900,000円 | PL |
| 税务加算合计 | 350,000円 | 別表四 |
| 税务减算合计 | 20,000円 | 別表四 |
| 所得金额 | 1,230,000円 | `900,000 + 350,000 - 20,000` |
| 法人税额 | 184,500円 | 假设由当年度申告软件计算 |
| 控除税额 | 0円 | 別表六等；本例无 |
| 中间缴纳额 | 50,000円 | 缴纳记录、別表五（二） |
| Mock 差引应缴 | 134,500円 | `184,500 - 50,000` |

`184,500円` 不是法定税率计算结论，只是演示数字。实际税额还可能受税率区分、控除、留保金课税、地方法人税及防卫特别法人税等影响。

## 5. 填写顺序

1. 填写法人名称、法人编号、所在地和事业年度。
2. 从別表四转入所得金额或欠损金额。
3. 使用当前年度规则计算法人税额。
4. 从相关别表转入税额控除。
5. 填写中间申告和中间缴纳金额。
6. 计算差引确定法人税额。
7. 确认地方法人税及当年度新增税目栏位。
8. 与別表五（二）的当期发生和缴纳状态核对。
9. 与 e-Tax 计算结果进行最终比较。

## 6. 表间联动

```text
PL 当期纯利益
  ↓
別表四：税务加算、减算
  ↓
別表一：所得与税额
  ↓
別表五（二）：税金发生、缴纳、未缴
```

## 7. 验收清单

- [ ] 事业年度与决算书一致
- [ ] 所得金额与別表四一致
- [ ] 中间缴纳与缴纳记录一致
- [ ] 税额与別表五（二）一致
- [ ] 使用实际年度对应的最新样式
- [ ] 已确认防卫特别法人税适用与否
- [ ] e-Tax形式检查无错误
