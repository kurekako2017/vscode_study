# 別表五（二）完整填写案例

## 1. 实例目的

別表五（二）用于管理法人税、地方法人税、住民税、事业税等租税公课的发生、缴纳和期末未缴状态。

本案例事业年度为 2025-05-01 至 2026-04-30。

## 2. 前置资料

- 前期別表五（二）
- 未払法人税等总账
- 普通预金缴纳记录
- 中间申告及中间缴纳资料
- 当期法人税和地方税申告结果

## 4. 账簿联动示例

```text
前期未缴税金支付：借方 未払法人税等 / 贷方 普通预金
当期税金确定　　：借方 法人税、住民税及事业税 / 贷方 未払法人税等
```

具体科目和发生时点应与公司的经理方式、申告确定时间一致。

## 5. 填写顺序

1. 从前期別表五（二）取得期首未缴税额。
2. 从银行和纳税证明确认前期税金的实际缴纳。
3. 输入当期中间申告和中间缴纳。
4. 从別表一及地方税申告结果取得当期确定税额。
5. 计算期末未缴金额。
6. 与未払法人税等总账核对。
7. 确认損金不算入税金是否正确反映到別表四。

## 6. 决算计提与 freee 分录

当期中间缴纳 50,000円已记入仮払法人税等，决算时确认：

| 借方（日文） | 金额 | 贷方（日文） | 金额 |
| --- | ---: | --- | ---: |
| `法人税、住民税及び事業税` | 300,300円 | `仮払法人税等` | 50,000円 |
|  |  | `未払法人税等` | 250,300円 |

1. 打开 freee `取引入力` → `振替伝票`。
2. 将发生日设为 2026-04-30。
3. 借方输入法人税、住民税及事业税 300,300円。
4. 贷方输入仮払法人税等 50,000円。
5. 贷方输入未払法人税等 250,300円。
6. 登记后确认借贷一致。
7. 将税费 300,300円加算至別表四。

## 7. 別表五（二）逐项填写

1. 从前期表转入期首未缴税金 200,000円。
2. 录入本期实际缴纳前期税金 200,000円，确认期首残额归零。
3. 录入法人税当期发生 154,500円。
4. 录入法人税中间缴纳 50,000円。
5. 计算法人税期末未缴 104,500円。
6. 录入地方法人税 15,800円。
7. 录入法人住民税 80,700円。
8. 录入法人事业税 36,000円。
9. 录入特别法人事业税 13,300円。
10. 确认当期发生合计 300,300円。
11. 确认期末未缴合计 250,300円。
12. 与 freee 未払法人税等总账核对。

## 9. 验收清单

- [ ] 期首未缴与前期表一致
- [ ] 实际缴纳与银行记录一致
- [ ] 中间缴纳没有遗漏或重复
- [ ] 当期税额与別表一、地方税申告一致
- [ ] 期末未缴与 BS 总账一致
- [ ] 新增税目已按实际年度确认

## 10. 常见错误

- 把前期税金缴纳再次计入当期税金费用。
- 忘记中间缴纳，导致应缴金额重复。
- 法人税申告完成后没有更新未払法人税等。
- 別表五（二）与別表四、別表一金额不一致。
