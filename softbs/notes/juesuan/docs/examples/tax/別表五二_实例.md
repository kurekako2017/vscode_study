# 別表五（二）填写实例（Mock）

## 1. 实例目的

別表五（二）用于管理法人税、地方法人税、住民税、事业税等租税公课的发生、缴纳和期末未缴状态。

## 2. 前置资料

- 前期別表五（二）
- 未払法人税等总账
- 普通预金缴纳记录
- 中间申告及中间缴纳资料
- 当期法人税和地方税申告结果

## 3. Mock 场景

### 前期税金

| 税目 | 期首未缴 | 当期缴纳 | 残额 |
| --- | ---: | ---: | ---: |
| 前期法人税等 | 200,000円 | 200,000円 | 0円 |

### 当期税金

| 税目 | Mock 当期发生 | 中间缴纳 | 期末未缴 |
| --- | ---: | ---: | ---: |
| 法人税 | 184,500円 | 50,000円 | 134,500円 |
| 地方法人税 | 18,000円 | 0円 | 18,000円 |
| 法人住民税 | 70,000円 | 0円 | 70,000円 |
| 法人事业税等 | 0円 | 0円 | 0円 |

金额全部为 Mock，不代表任何地区、年度或实际税率。防卫特别法人税等新增税目应按适用年度追加确认。

## 4. 账簿联动示例

```text
前期未缴税金支付：借方 未払法人税等 / 贷方 普通预金
当期税金确定　　：借方 法人税、住民税及事业税 / 贷方 未払法人税等
```

具体科目和发生时点应与公司的经理方式、申告确定时间一致。

## 5. 填写顺序

1. 从前期別表五（二）取得期首未缴税额。
2. 从银行和纳税证明确认前期税金的实际缴纳。
3. 输入当期中间申告和中间缴纳。
4. 从別表一及地方税申告结果取得当期确定税额。
5. 计算期末未缴金额。
6. 与未払法人税等总账核对。
7. 确认損金不算入税金是否正确反映到別表四。

## 6. 表间联动

- 法人税确定额、中间缴纳 → [別表一实例](別表一_实例.md)
- 税金損金不算入调整 → [別表四实例](別表四_实例.md)
- 期末未缴税金 → BS 的未払法人税等

## 7. 验收清单

- [ ] 期首未缴与前期表一致
- [ ] 实际缴纳与银行记录一致
- [ ] 中间缴纳没有遗漏或重复
- [ ] 当期税额与別表一、地方税申告一致
- [ ] 期末未缴与 BS 总账一致
- [ ] 新增税目已按实际年度确认

## 8. 常见错误

- 把前期税金缴纳再次计入当期税金费用。
- 忘记中间缴纳，导致应缴金额重复。
- 法人税申告完成后没有更新未払法人税等。
- 別表五（二）与別表四、別表一金额不一致。
