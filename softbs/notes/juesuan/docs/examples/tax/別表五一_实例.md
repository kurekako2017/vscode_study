# 別表五（一）填写实例（Mock）

## 1. 实例目的

別表五（一）管理税务上的利益积立金额和资本金等金额，可以理解为“税务上的纯资产变动明细”。

本表必须从前期申告书连续结转，不能每年从零猜测。

## 2. 前置资料

- 前期別表五（一）
- 当期 BS 的纯资产部分
- 当期 [別表四实例](別表四_实例.md)
- 股东会利润处分资料
- 资本金变更资料

## 3. Mock 期首数据

| 项目 | 期首 Mock 金额 |
| --- | ---: |
| 繰越損益金 | 700,000円 |
| 减价償却超额 | 100,000円 |
| 资本金等金额 | 5,000,000円 |

## 4. Mock 当期变动

| 项目 | 减少 | 增加 | 期末 Mock 金额 | 依据 |
| --- | ---: | ---: | ---: | --- |
| 繰越損益金 | 0円 | 900,000円 | 1,600,000円 | 当期纯利益，假设无配当 |
| 减价償却超额 | 20,000円 | 0円 | 80,000円 | 別表四本期认容 |
| 资本金等金额 | 0円 | 0円 | 5,000,000円 | 本期无增减资 |

表格上的正负方向和填写栏位必须按实际年度样式确认。本表只演示“期首 + 增减 = 期末”的连续性。

## 5. 填写顺序

1. 从前期別表五（一）复制期末金额作为当期期首。
2. 与当期 BS 的纯资产项目核对。
3. 从別表四提取当期留保增加和留保取崩。
4. 反映配当、增资、减资等资本交易。
5. 计算各项目期末金额。
6. 确认利益积立金额合计和资本金等金额。
7. 保存本期表格，作为翌期期首依据。

## 6. 与別表四联动示例

```text
前期减价償却超额留保：100,000
本期別表四减算认容：   20,000
期末减价償却超额：     80,000
```

如果別表四已经减算20,000，但別表五（一）仍保留100,000，则表间不一致。

## 7. 验收清单

- [ ] 期首金额与前期表一致
- [ ] 当期纯利益与决算书一致
- [ ] 留保增减与別表四一致
- [ ] 资本金与登记者项及 BS 一致
- [ ] 期首加减变动等于期末
- [ ] 本期完成版已保存供翌期结转

## 8. 常见错误

- 没有前期表，直接把所有期首金额填零。
- 把社外流出项目放进留保余额。
- 忘记反映前期减价償却超额的本期认容。
- 只看会计 BS，不核对税务留保项目。
