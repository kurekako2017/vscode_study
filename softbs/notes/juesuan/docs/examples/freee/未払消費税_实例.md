# 未払消費税 实例（占位版）

## 学习目标

- 理解未払消费税与仮受消费税、仮払消费税的关系。
- 区分申告确定和实际缴纳两个时点。
- 学会核对申告书、总账和银行记录。

## Mock 场景

假设消费税申告最终确定应缴440,000円。本数字仅为教学示意，不是实际税额。

```text
申告确定：按税込/税抜经理方式确认消费税振替
实际缴纳：借方 未払消費税 440,000 / 贷方 普通預金 440,000
```
