# 未払消費税完整案例

## 1. 公司与申告条件

| 日文项目 | 中文解释 | 案例条件 |
| --- | --- | --- |
| `課税事業者` | 消费税课税事业者 | 是 |
| `課税方式` | 计税方式 | 一般课税 |
| `経理方式` | 会计处理方式 | 税込経理 |
| `課税期間` | 课税期间 | 2025-05-01～2026-04-30 |
| `確定納付税額` | 确定应缴税额 | 500,000円 |
| `中間納付額` | 已中间缴纳 | 60,000円 |
| `差引納付額` | 决算后需缴纳 | 440,000円 |

## 2. 从决算到申告确认

1. 完成全年银行、信用卡和费用交易登记。
2. 打开 freee `決算申告` → `消費税申告書`。
3. 确认课税期间、一般课税和税込経理设置。
4. 检查 10% 课税销售、课税仕入、非课税和対象外税区分。
5. 确认中间纳付 60,000円已录入。
6. 运行申告书计算。
7. 确认全年确定纳付税额为 500,000円。
8. 确认差引纳付额为 440,000円。
9. 保存确定版申告书和计算依据。

## 3. 决算振替传票

税込経理下，本案例在期末计提尚未缴纳的 440,000円：

| 借方（日文） | 金额 | 贷方（日文） | 金额 | 中文解释 |
| --- | ---: | --- | ---: | --- |
| `租税公課` | 440,000円 | `未払消費税等` | 440,000円 | 确认期末消费税负债 |

### freee 操作步骤

1. 打开 `取引入力` → `振替伝票`。
2. 将発生日设为 2026-04-30。
3. 借方选择 `租税公課`，输入 440,000円。
4. 贷方选择 `未払消費税等`，输入 440,000円。
5. 税区分选择适用于税金结转的対象外区分。
6. 摘要填写 `2026年4月期 消費税確定分`。
7. 登记振替传票。
8. 打开 `会計帳簿` → `仕訳帳` 确认分录。
9. 打开 `貸借対照表`，确认未払消費税等为 440,000円。
10. 将该余额与消费税申告书差引纳付额核对。

## 4. 实际纳税

假设 2026-05-25 从公司普通预金完成电子纳税：

| 借方（日文） | 金额 | 贷方（日文） | 金额 |
| --- | ---: | --- | ---: |
| `未払消費税等` | 440,000円 | `普通預金` | 440,000円 |

1. 在 e-Tax 确认消费税受信通知无错误。
2. 通过电子纳税支付 440,000円。
3. 保存纳付完成画面和银行扣款记录。
4. 银行同步后打开 `自動で経理`。
5. 选择 440,000円出金明细。
6. 勘定科目选择 `未払消費税等`。
7. 登记交易。
8. 打开 `貸借対照表`，确认未払消費税等归零。
9. 确认没有再次计入租税公課。

## 5. 三方核对

1. 消费税申告书差引纳付额为 440,000円。
2. 决算日 BS 的未払消費税等为 440,000円。
3. 次期银行实际扣款为 440,000円。
4. 纳税后未払消費税等余额为 0円。
5. 申告书、振替传票、受信通知和银行记录完整保存。
