# 役員借入金 实例（占位版）

## 当前状态

占位文档：已建立学习结构，等待使用脱敏 Mock 数据和 freee 最新画面完成实测。

## 学习目标

- 理解代表者个人垫付公司费用时为什么使用役員借入金。
- 区分役員借入金、役員貸付金和現金。
- 学会登记垫付与公司返还两步仕訳。

## Mock 场景

代表者个人信用卡支付公司用云服务11,000円，之后公司银行账户返还代表者。

```text
垫付时：借方 通信費 11,000 / 贷方 役員借入金 11,000
返还时：借方 役員借入金 11,000 / 贷方 普通預金 11,000
```
