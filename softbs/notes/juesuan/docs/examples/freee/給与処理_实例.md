# 給与処理完整案例

## 1. 公司与处理条件

| 日文项目 | 中文解释 | 本案例数值 |
| --- | --- | ---: |
| `会社形態` | 公司形态 | 日本株式会社、一人法人 |
| `事業内容` | 业务 | IT・SES |
| `役員` | 董事 | 代表取締役 1 人 |
| `役員報酬` | 固定董事工资 | 480,000円 |
| `通勤手当` | 通勤补贴 | 0円（不支给） |
| `総支給額` | 工资总额 | 480,000円 |
| `社会保険料（本人負担）` | 个人负担社会保险 | 70,030円 |
| `源泉所得税` | 预扣所得税 | 15,000円 |
| `住民税` | 特别征收住民税 | 20,000円 |
| `差引支給額` | 银行实际发薪 | 374,970円 |
| `社会保険料（会社負担）` | 公司负担社会保险 | 71,722円 |

上述保险费来自本案例的决定通知与保险费资料；实际执行必须替换为公司收到的通知金额。

## 2. 工资计算

```text
役員報酬             480,000
通勤手当                   0
総支給額             480,000
社会保険料           -70,030
源泉所得税           -15,000
住民税               -20,000
差引支給額           374,970
```

### freee 人事劳务步骤

1. 登录 freee 人事劳务并选择正确事业所。
2. 打开 `給与` → 目标月份。
3. 选择代表董事，确认固定役员报酬为 480,000円。
4. 确认通勤手当为 0円，没有设置通勤补贴。
5. 核对社会保险本人负担 70,030円。
6. 核对源泉所得税 15,000円和住民税 20,000円。
7. 确认差引支给额为 374,970円。
8. 点击工资计算确定功能，锁定工资明细。
9. 输出工资明细并由代表董事确认。
10. 将工资交易联动至 freee 会计。

## 3. 工资确定时的会计分录

| 借方（日文） | 借方金额 | 贷方（日文） | 贷方金额 | 中文解释 |
| --- | ---: | --- | ---: | --- |
| `役員報酬` | 480,000円 | `未払費用（給与）` | 374,970円 | 确认董事工资净额负债 |
| `法定福利費` | 71,722円 | `預り金（社会保険）` | 70,030円 | 代扣个人社会保险 |
|  |  | `預り金（源泉所得税）` | 15,000円 | 代扣所得税 |
|  |  | `預り金（住民税）` | 20,000円 | 代扣住民税 |
|  |  | `未払費用（会社社会保険）` | 71,722円 | 公司负担社会保险 |

借贷合计均为 551,722円。

### freee 会计确认步骤

1. 打开 `取引入力` → `給与取引の登録`。
2. 确认目标月份工资交易已从 freee 人事劳务联动。
3. 核对役員報酬 480,000円、通勤手当 0円和法定福利费。
4. 核对社会保险、源泉所得税、住民税等负债。
5. 确认净工资处于未决済状态。
6. 打开 `会計帳簿` → `仕訳帳`，按日期和 374,970円搜索。
7. 确认借贷合计一致。

## 4. 银行支付与消込

1. 在发薪日从公司银行向代表董事账户汇款 374,970円。
2. 同步银行流水。
3. 打开 `取引入力` → `自動で経理`。
4. 选择 374,970円出金明细。
5. 选择对应的未决済工资交易。
6. 执行 `未決済取引の消込`。
7. 确认分录为借方未払費用 374,970円、贷方普通預金 374,970円。
8. 确认工资费用没有因银行流水再次登记。

## 5. 后续缴纳

| 支付项目 | 金额 | 支付时分录 |
| --- | ---: | --- |
| 社会保险本人及公司负担 | 141,752円 | 借方預り金 70,030、未払費用 71,722／贷方普通預金 141,752 |
| 源泉所得税 | 15,000円 | 借方預り金／贷方普通預金 |
| 住民税 | 20,000円 | 借方預り金／贷方普通預金 |

1. 社会保险扣款后，在 `自動で経理` 选择 141,752円明细并冲减两项负债。
2. 源泉所得税缴纳后选择 15,000円明细并冲减预り金。
3. 住民税缴纳后选择 20,000円明细并冲减预り金。
4. 打开 `貸借対照表`，确认已缴月份相关预り金余额归零。
5. 将工资台账、银行记录和会计仕訳交叉核对。

## 6. 验收结果

1. 工资明细净额与银行汇款 374,970円一致。
2. 工资确定和银行付款没有重复确认费用。
3. 社会保险总缴纳额 141,752円与本人、公司负担合计一致。
4. 源泉税和住民税预り金均有缴纳记录。
5. 所有操作保留工资明细、通知书和银行依据。
