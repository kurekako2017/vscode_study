# 住民税特別徴収完整案例

## 1. 实例目的

演示公司根据自治体发送的特别征收税额决定通知书，从工资控除住民税并缴纳。

本实例不是法人住民税申告实例，而是个人住民税的工资特别征收实例。

## 2. 特别征收通知

| 项目 | 案例内容 |
| --- | --- |
| 受给者 | 山田太郎 |
| 年税额 | 240,000円 |
| 适用期间 | 6月至次年5月 |
| 每月特别征收额 | 20,000円 |
| 提交自治体 | 代表者次年1月1日住所地市区町村 |

## 3. 工资处理

假设6月总支给额480,000円：

```text
总支给额                         480,000
－ 社会保险料                    70,030
－ 源泉所得税                   15,000
－ 住民税                        20,000
---------------------------------------
差引支给额                      374,970
```

## 4. 仕訳示例

工资确定时，把住民税20,000円计入预り金。缴纳时：

```text
借方 预り金（住民税） 20,000 / 贷方 普通预金 20,000
```

实际工资仕訳还包括工资、社会保险和源泉所得税，本例只突出住民税部分。

## 5. 操作步骤

1. 保存特别征收税额决定通知书。
2. 核对受给者、自治体和年税额。
3. 把6月至次年5月的月额输入工资系统。
4. 工资确定时控除当月住民税。
5. 在预り金中确认控除额。
6. 原则上在次月10日前向自治体缴纳。
7. 核对缴纳记录与预り金余额。
8. 退职或转勤时确认工资所得者异动届出书。

## 6. 验收清单

- [ ] 使用最新决定通知书
- [ ] 自治体和受给者正确
- [ ] 6月至次年5月金额已设置
- [ ] 工资控除与通知书一致
- [ ] 缴纳金额与预り金一致
- [ ] 异动时已确认届出手续

## 7. 注意

- 不能根据上一年度金额继续控除。
- 普通征收和特别征收不能混淆。
- 多个自治体时，应分别确认金额和接收结果。
- 纳期限和纳期特例应按自治体最新规定确认。
