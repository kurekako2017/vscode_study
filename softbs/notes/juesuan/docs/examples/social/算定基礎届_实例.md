# 算定基礎届完整案例

## 1. 实例目的

演示一人法人如何整理4月、5月、6月实际支付的报酬，并准备算定基础届。

> 标准报酬月额等级和保险料必须使用事业所所在地、加入制度及当年度最新保险料额表确认。本例不代替最终等级判定。

## 2. 公司与被保险者

| 日文项目 | 中文解释 | 案例内容 |
| --- | --- | --- |
| `会社形態` | 公司形态 | 日本株式会社、一人法人、IT・SES |
| `被保険者` | 被保险人 | 代表取締役 1 人 |
| `資格取得日` | 资格取得日 | 2024-05-01 |
| `給与支払日` | 工资支付日 | 每月 25 日 |
| `固定役員報酬` | 固定董事工资 | 480,000円/月 |
| `通勤手当` | 通勤补贴 | 0円（不支给） |
| `従前標準報酬月額` | 原标准报酬月额 | 470,000円 |

## 3. 4月至6月资料

| 支付月 | 役员报酬 | 通勤手当 | 报酬合计 | 支付基础日数 |
| --- | ---: | ---: | ---: | ---: |
| 4月 | 480,000円 | 0円 | 480,000円 | 30日 |
| 5月 | 480,000円 | 0円 | 480,000円 | 31日 |
| 6月 | 480,000円 | 0円 | 480,000円 | 30日 |

平均报酬月额：

```text
(480,000 + 480,000 + 480,000) ÷ 3 = 480,000円
```

## 4. 填写步骤

1. 确认对象在7月1日仍为被保险者。
2. 确认不是6月1日以后新取得资格等除外对象。
3. 从工资台账取得4月至6月实际支付额。
4. 确认本案例没有通勤手当，报酬只有役员报酬。
5. 确认支付基础日数。
6. 计算3个月平均报酬月额480,000円。
7. 对照当年度标准报酬月额表确定等级。
8. 填写算定基础届并提交。
9. 收到决定通知后，从适用月份更新工资控除。

## 5. 本例输出

| 项目 | 结果 |
| --- | --- |
| 平均报酬月额 | 480,000円 |
| 标准报酬月额 | 470,000円（按本案例适用等级表） |
| 提交期间 | 每年重新确认；2026年度为7月1日至7月10日 |
| 适用期间 | 原则为当年9月至次年8月 |

## 6. 验收清单

- [ ] 使用实际支付月份资料
- [ ] 已确认没有通勤手当或其他手当
- [ ] 支付基础日数正确
- [ ] 已确认除外对象
- [ ] 已使用当年度最新等级表
- [ ] 接收结果和决定通知已保存
- [ ] 新保险料已反映到工资系统

## 7. 届书填写案例

| 日文栏位 | 中文解释 | 填写值 |
| --- | --- | ---: |
| `被保険者整理番号` | 被保险者编号 | 公司实际编号 |
| `従前の標準報酬月額` | 原标准报酬月额 | 470,000円 |
| `4月 支払基礎日数` | 4 月支付基础日数 | 30日 |
| `4月 報酬月額` | 4 月报酬 | 480,000円 |
| `5月 支払基礎日数` | 5 月支付基础日数 | 31日 |
| `5月 報酬月額` | 5 月报酬 | 480,000円 |
| `6月 支払基礎日数` | 6 月支付基础日数 | 30日 |
| `6月 報酬月額` | 6 月报酬 | 480,000円 |
| `総計` | 三个月合计 | 1,440,000円 |
| `平均額` | 平均报酬月额 | 480,000円 |

1. 确认代表董事在 2026-07-01 仍为被保险者。
2. 确认资格取得日早于 2026-06-01。
3. 确认不属于 7～9 月月额变更预定者。
4. 逐月填写支付基础日数和货币报酬。
5. 现物报酬栏填写 0円。
6. 计算总计 1,440,000円和平均 480,000円。
7. 对照当年度等级表确认标准报酬月额 470,000円。

## 8. 电子申请完整流程

1. 从日本年金機構官网下载并更新 `届書作成プログラム`。
2. 使用在线事业所年金信息服务取得被保险者数据。
3. 登录程序并选择本公司事业所。
4. 选择 `被保険者報酬月額算定基礎届`。
5. 导入代表董事资料。
6. 输入 4～6 月各月日数和 480,000円报酬。
7. 执行程序检查并修正错误。
8. 使用 GビズIDプライム或事业主マイナンバーカード认证。
9. 在 2026-07-10 前发送。
10. 保存到达编号和发送数据。
11. 在 e-Gov 申请案件中确认受付和审查状态。
12. 取得决定通知书并确认标准报酬月额 470,000円。
13. 按公司保险费扣除规则更新工资系统。

## 9. 会计与工资联动

1. 将决定通知书保存到受限的社会保险资料区。
2. 在 freee 人事劳务更新健康保险、介护保险和厚生年金的标准报酬月额。
3. 生成新保险费适用月份工资预览。
4. 将个人负担额与保险料额表核对。
5. 将公司负担额与会计法定福利费核对。
6. 确认银行实际社会保险扣款与预り金、未払费用合计一致。
