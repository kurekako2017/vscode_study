# 算定基礎届 实例（Mock）

## 1. 实例目的

演示一人法人如何整理4月、5月、6月实际支付的报酬，并准备算定基础届。

> 标准报酬月额等级和保险料必须使用事业所所在地、加入制度及当年度最新保险料额表确认。本例不代替最终等级判定。

## 2. 虚构对象

| 项目 | Mock 内容 |
| --- | --- |
| 被保险者 | 代表取締役 山田太郎 |
| 资格取得日 | 2024-05-01 |
| 工资支付日 | 每月25日 |
| 固定役员报酬 | 480,000円/月 |
| 通勤手当 | 10,000円/月 |

## 3. 4月至6月资料

| 支付月 | 役员报酬 | 通勤手当 | 报酬合计 | 支付基础日数 |
| --- | ---: | ---: | ---: | ---: |
| 4月 | 480,000円 | 10,000円 | 490,000円 | 30日 |
| 5月 | 480,000円 | 10,000円 | 490,000円 | 31日 |
| 6月 | 480,000円 | 10,000円 | 490,000円 | 30日 |

Mock 平均报酬月额：

```text
(490,000 + 490,000 + 490,000) ÷ 3 = 490,000円
```

## 4. 填写步骤

1. 确认对象在7月1日仍为被保险者。
2. 确认不是6月1日以后新取得资格等除外对象。
3. 从工资台账取得4月至6月实际支付额。
4. 把属于报酬的通勤手当等计入。
5. 确认支付基础日数。
6. 计算3个月平均报酬月额490,000円。
7. 对照当年度标准报酬月额表确定等级。
8. 填写算定基础届并提交。
9. 收到决定通知后，从适用月份更新工资控除。

## 5. 本例输出

| 项目 | 结果 |
| --- | --- |
| 平均报酬月额 | 490,000円 |
| 标准报酬月额 | 根据最新等级表填写，不在本例固定 |
| 提交期间 | 每年重新确认；2026年度为7月1日至7月10日 |
| 适用期间 | 原则为当年9月至次年8月 |

## 6. 验收清单

- [ ] 使用实际支付月份资料
- [ ] 通勤手当等报酬项目没有遗漏
- [ ] 支付基础日数正确
- [ ] 已确认除外对象
- [ ] 已使用当年度最新等级表
- [ ] 接收结果和决定通知已保存
- [ ] 新保险料已反映到工资系统
