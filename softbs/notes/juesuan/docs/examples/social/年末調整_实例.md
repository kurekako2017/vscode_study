# 年末調整 实例（Mock）

## 1. 实例目的

演示一名役员完成申告书回收、全年工资确认、年税额计算和过不足额精算的流程。

> 所得控除和年税额会受年度法改正及个人情况影响。本例的最终税额是流程演示值，不能复制到真实源泉征收票。

## 2. 虚构对象

| 项目 | Mock 内容 |
| --- | --- |
| 受给者 | 代表取締役 山田太郎 |
| 年间工资收入 | 5,760,000円 |
| 社会保险料 | 850,000円 |
| 生命保险料申告额 | 60,000円 |
| 扶养亲族 | 0人 |
| 每月已源泉征收合计 | 180,000円 |

## 3. 回收资料

- 扶养控除等申告书
- 基础控除申告书
- 配偶者控除等申告书（符合时）
- 保険料控除申告书
- 生命保险料控除证明
- 社会保险料资料
- 住宅贷款控除资料（符合时）

## 4. Mock 处理流程

1. 确认全年12个月工资均已确定。
2. 合计年间工资收入5,760,000円。
3. 确认社会保险料850,000円。
4. 登记生命保险料申告资料60,000円。
5. 根据当年度规则计算工资所得控除和所得控除。
6. 使用当年度年末调整计算规则确定年税额。
7. 本例假设计算后的年税额为165,000円。
8. 与已源泉征收180,000円比较。
9. Mock 还付额为15,000円。
10. 将差额反映到最后工资并生成源泉征收票。

```text
已源泉征收额 180,000
－ Mock 年税额 165,000
= Mock 还付额   15,000
```

165,000円只是教学假设，真实年税额必须由当年度官方规则或已更新的工资软件计算。

## 5. 后续输出

- 源泉征收票
- 法定调书合计表
- 符合条件的其他支払调书
- 工资支付报告书
- 年末调整计算资料

## 6. 验收清单

- [ ] 使用当年度最新申告书
- [ ] 全年工资合计正确
- [ ] 社会保险料与工资台账一致
- [ ] 控除证明齐全
- [ ] 年税额使用当年度规则计算
- [ ] 过不足额已反映
- [ ] 源泉征收票已交付
- [ ] 法定调书和工资支付报告书已准备
