# 年末調整完整案例

## 1. 实例目的

演示一名役员完成申告书回收、全年工资确认、年税额计算和过不足额精算的流程。

> 所得控除和年税额会受年度法改正及个人情况影响。本例的最终税额是流程演示值，不能复制到真实源泉征收票。

## 2. 公司与受给者

| 日文项目 | 中文解释 | 案例内容 |
| --- | --- | --- |
| `会社形態` | 公司形态 | 日本株式会社、一人法人、IT・SES |
| `受給者` | 工资受领人 | 代表取締役 1 人 |
| `役員報酬` | 年度董事工资 | 5,760,000円 |
| `通勤手当` | 通勤补贴 | 0円（不支给） |
| `社会保険料` | 个人负担社会保险 | 840,360円 |
| `生命保険料` | 新合同一般生命保险缴费 | 60,000円 |
| `扶養親族` | 扶养亲属 | 0 人 |
| `源泉徴収済税額` | 全年已预扣税额 | 180,000円 |

## 3. 回收资料

- 扶养控除等申告书
- 基础控除申告书
- 配偶者控除等申告书（符合时）
- 保険料控除申告书
- 生命保险料控除证明
- 社会保险料资料
- 住宅贷款控除资料（符合时）

## 4. 年末调整计算

1. 确认全年12个月工资均已确定。
2. 合计年间工资收入5,760,000円。
3. 确认社会保险料840,360円。
4. 登记生命保险料申告资料60,000円。
5. 根据当年度规则计算工资所得控除和所得控除。
6. 使用当年度年末调整计算规则确定年税额。
7. 按本案例适用年度规则计算年调年税额 167,100円。
8. 与已源泉征收180,000円比较。
9. 计算还付额为12,900円。
10. 将差额反映到最后工资并生成源泉征收票。

```text
已源泉征收额 180,000
－ 年调年税额 167,100
= 还付额        12,900
```

本案例采用的主要计算为：工资所得控除后所得 4,168,000円、基础控除 680,000円、社会保险料控除 840,360円、生命保险料控除 35,000円。所得控除合计 1,555,360円，课税工资所得 2,612,000円。执行时必须使用目标年度国税庁规则和已更新的工资软件复算。

## 5. freee 人事劳务操作

1. 打开 freee 人事劳务 `年末調整`。
2. 选择目标年度和代表董事。
3. 确认其为年末调整对象。
4. 确认扶養控除等申告书和基础控除申告书已回收。
5. 登记社会保险料 840,360円。
6. 登记一般生命保险新合同支付额 60,000円。
7. 确认无配偶者控除、扶养控除和住宅贷款控除。
8. 确认应税工资 5,760,000円，并确认没有通勤手当。
9. 执行年末调整计算。
10. 核对年调年税额 167,100円和还付额 12,900円。
11. 确定年末调整结果。
12. 将 12,900円还付反映到 12 月工资。
13. 打开工资明细确认净发薪增加 12,900円。

## 6. 源泉征收票填写结果

| 日文栏位 | 中文解释 | 金额 |
| --- | --- | ---: |
| `支払金額` | 应税工资支付额 | 5,760,000円 |
| `給与所得控除後の金額` | 工资所得控除后金额 | 4,168,000円 |
| `所得控除の額の合計額` | 所得控除合计 | 1,555,360円 |
| `源泉徴収税額` | 年调后源泉税 | 167,100円 |
| `社会保険料等の金額` | 社会保险料 | 840,360円 |
| `生命保険料の控除額` | 生命保险料控除 | 35,000円 |

1. 生成代表董事源泉征收票。
2. 核对上述六项金额。
3. 确认受给者交付用不显示マイナンバー。
4. 向代表董事交付源泉征收票。
5. 按法人役员提交范围判断税务署提交义务。

## 7. 后续输出与提交

- 源泉征收票
- 法定调书合计表
- 符合条件的其他支払调书
- 工资支付报告书
- 年末调整计算资料

1. 通过 e-Tax 提交所需源泉征收票和法定调书合计表。
2. 通过 eLTAX 向次年 1 月 1 日住所地提交工资支付报告书。
3. 保存 e-Tax 和 eLTAX 受信通知。
4. 将还付后的源泉所得税金额反映到徴収高計算書。
5. 完成实际纳税并核对预り金。
