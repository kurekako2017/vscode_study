# 源泉所得税完整案例

## 1. 实例目的

演示一人法人每月从役员报酬控除源泉所得税，并在纳期特例获得批准时汇总缴纳。

> 令和8年源泉征收税额表已经更新。真实税额必须按支付年度、甲栏或乙栏、扶养人数和社会保险料控除后的金额查询当年度表格。

## 2. 工资处理条件

| 项目 | 案例内容 |
| --- | ---: |
| 月额役员报酬 | 480,000円 |
| 社会保险料 | 70,030円 |
| 社会保险料控除后金额 | 409,970円 |
| 适用栏 | 假设甲栏 |
| 扶养亲族 | 0人 |
| 源泉税额 | 15,000円 |

15,000円只是演示值，不能代替令和8年分或其他年度税额表查询结果。

## 3. 每月操作

1. 确认扶养控除等申告书是否已提交。
2. 确认甲栏、乙栏等适用区分。
3. 计算社会保险料控除后的工资金额409,970円。
4. 查询支付年度的源泉征收税额表。
5. 本案例根据工资台账使用15,000円控除额。
6. 工资确定时计入预り金。
7. 保存工资台账和税额查询依据。

## 4. 仕訳示例

```text
工资确定时：
借方 役员报酬 480,000
  贷方 预り金（社会保险等） 70,030
  贷方 预り金（源泉所得税） 15,000
  贷方 未払金（差引工资） 394,970

源泉税缴纳时：
借方 预り金（源泉所得税） / 贷方 普通预金
```

## 5. 纳期特例处理

假设1月至6月每月源泉税额均为15,000円：

```text
15,000 × 6个月 = 90,000円
```

如果已经获得纳期特例批准，1月至6月部分一般在7月10日前缴纳；7月至12月部分一般在次年1月20日前缴纳。

没有确认批准状态时，不能自行采用半年缴纳。

## 6. 验收清单

- [ ] 使用支付年度对应的税额表
- [ ] 甲栏、乙栏判断正确
- [ ] 扶养亲族人数正确
- [ ] 社会保险料控除后金额正确
- [ ] 工资台账与预り金一致
- [ ] 已确认纳期特例批准状态
- [ ] 缴纳记录已保存
