# freee 月次检查表

## 1. 使用方法

每月完成一轮后复制“月次记录”部分，填写月份、完成日期和未解决事项。不要把银行密码、API Key、个人编号写入本文件。

详细科目说明参见 [勘定科目辞典](../glossary/勘定科目辞典.md)。

## 2. 银行与信用卡

- [ ] 全部法人银行已同步
- [ ] 全部法人信用卡已同步
- [ ] 同步期间没有缺失
- [ ] `自動で経理` 待处理项目已确认
- [ ] 未登记明细已处理
- [ ] 重复登记已确认
- [ ] 普通预金月末余额与银行一致

## 3. 销售与收款

- [ ] 当月销售已与合同、请求书核对
- [ ] 売掛金已登记
- [ ] 入金已正确消込売掛金
- [ ] 未入金清单已更新
- [ ] 长期未收款已确认原因
- [ ] 销售没有跨期或重复

## 4. 费用与支付

- [ ] 请求书、收据和发票已保存
- [ ] 当月费用已登记
- [ ] 未払金已登记
- [ ] 支付时已正确消込未払金
- [ ] 个人垫付已登记为役員借入金
- [ ] 私人消费没有混入
- [ ] 外注费已有合同和请求书
- [ ] 固定资产候选已登记到确认清单

## 5. 现金与其他余额

- [ ] 实际现金与账面一致
- [ ] 现金没有负数
- [ ] 売掛金有明细依据
- [ ] 未払金有明细依据
- [ ] 役員借入金有垫付及精算记录
- [ ] 预り金与工资、纳税资料一致
- [ ] 不明入金和不明出金已调查

## 6. 工资、税金与社会保险

- [ ] 役员报酬和工资已确定
- [ ] 工资明细已交付
- [ ] 实际汇款与差引支给额一致
- [ ] 源泉所得税金额和期限已确认
- [ ] 住民税已控除并缴纳
- [ ] 社会保险料已确认
- [ ] 固定性工资变更时已确认月额变更届

## 7. 消费税与税区分

- [ ] 未设定税区分已确认
- [ ] 10%、8%轻减、旧8%没有混淆
- [ ] 海外服务税区分已单独确认
- [ ] インボイス登录编号和必要事项已确认
- [ ] 中间缴纳通知已登记

## 8. PL 与 BS

- [ ] PL 已与上月及去年同月比较
- [ ] 销售和主要费用的异常增减已解释
- [ ] BS 的普通预金、现金、売掛金、未払金已确认
- [ ] 役員借入金和预り金没有异常反对余额
- [ ] 未解决项目已记录到下月任务

## 9. 资料保存

- [ ] 证憑已按年月整理
- [ ] 工资台账和缴纳记录已保存
- [ ] 银行月末资料已保存
- [ ] 月次试算表已输出或确认
- [ ] 重要判断已有备注或专业确认记录
