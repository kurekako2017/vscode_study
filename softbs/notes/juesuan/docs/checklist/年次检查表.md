# 法人年次・決算检查表

## 1. 使用方法

本检查表覆盖1月至12月的年度作业、决算前检查、电子申报和资料保存。

期限会因决算月、节假日、特例和自治体而变化。每年必须从官方资料确认实际日期。

## 2. 1月至12月年度计划

| 月 | 主要工作 | 提交或缴纳 | 验收标准 |
| --- | --- | --- | --- |
| 1月 | 年末调整后处理 | 源泉所得税纳期特例、法定调书、工资支付报告书、符合时的償却资产申告 | 接收通知和缴纳记录齐全 |
| 2月 | 整理1月提交资料 | 定期缴纳 | 全部错误通知已处理 |
| 3月 | 合同、固定资产和年度末交易确认 | 需要时提交届出书 | 决算候选事项已列出 |
| 4月 | 4月决算法人的締切处理 | 收集决算资料 | 销售、费用期间归属正确 |
| 5月 | 决算整理和申告书制作 | 决算书、别表、内訳书 | BS、PL和税务调整完成 |
| 6月 | 4月决算法人申告缴纳 | 法人税、地方法人税、消费税、地方税 | e-Tax、eLTAX及缴纳全部完成 |
| 7月 | 算定基础届、源泉税纳期特例 | 算定基础届、1月至6月源泉税 | 官方期限内完成 |
| 8月 | 保存申告资料、确认翌期结转 | 定期缴纳 | 前期申告书和别表五已归档 |
| 9月 | 反映定时决定 | 需要时月额变更届 | 新社会保险料已反映 |
| 10月 | 上期经营和税额预估 | 需要时中间申告 | 中间申告义务已确认 |
| 11月 | 年末调整准备 | 回收各类申告书 | 资料缺失清单已完成 |
| 12月 | 年末调整和次年设置 | 准备源泉征收票等 | 差额已反映，1月任务已建立 |
