# 算定基礎届・月額変更届手册

## 1. 目的

本手册用于一人法人确认健康保险、厚生年金的算定基础届和月额变更届。

一人法人只有一名代表时，只要代表属于健康保险、厚生年金被保险者，也需要确认相关手续。

## 2. 算定基礎届是什么

事業主根据4月、5月、6月实际支付的报酬，提交算定基础届。日本年金机构据此重新决定标准报酬月额，通常从9月适用到次年8月。

2026年度官方公布的提交期间为2026年7月1日至7月10日。每年都必须重新确认期限和最新表格。

## 3. 准备资料

- 日本年金机构发送的対象者资料
- 4月工资台账
- 5月工资台账
- 6月工资台账
- 支付基础日数
- 通勤手当等属于报酬的项目
- 电子申报环境或邮寄资料

## 4. 算定基礎届操作步骤

1. 确认7月1日当前的被保险者。
2. 确认不需要提交的对象和月额变更预定者。
3. 从工资台账取得4月、5月、6月的报酬。
4. 确认各月支付基础日数。
5. 计算对象月份的平均报酬月额。
6. 填写算定基础届。
7. 使用电子申报、邮寄或窗口方式提交。
8. 保存提交控件和接收结果。
9. 收到标准报酬月额决定通知书后核对内容。
10. 从适用月份更新工资系统的社会保险控除额。

不能只看“工资所属月份”，应按实际支付月份和官方规则确认。

## 5. 月額変更届是什么

基本工资、役员报酬等固定性工资发生变化，并且之后连续3个月的平均报酬使标准报酬月额原则上变化2个等级以上，同时满足支付基础日数等条件时，需要确认随时改定。

## 6. 月額変更届操作步骤

1. 确认固定性工资变化的月份。
2. 收集变化月起连续3个月的工资台账。
3. 确认每月支付基础日数。
4. 计算3个月平均报酬。
5. 对照现行标准报酬月额，确认等级变化。
6. 使用日本年金机构最新要件判断是否符合随时改定。
7. 符合时及时提交月额变更届。
8. 收到改定通知后更新适用月份的保险费。

役员报酬的变更同时涉及法人税规则，不能只根据社会保险需要决定变更时点。

## 7. 算定基础届检查表

- [ ] 已确认本年度官方期限
- [ ] 已确认7月1日当前対象者
- [ ] 已准备4月至6月工资台账
- [ ] 已确认支付基础日数
- [ ] 已正确包含属于报酬的各种手当
- [ ] 已排除不属于对象的人员
- [ ] 已提交并保存接收结果
- [ ] 已保存决定通知书
- [ ] 已更新9月以后保险费

## 8. 月额变更检查表

- [ ] 固定性工资确实发生变化
- [ ] 已确认变化后3个月资料
- [ ] 已确认支付基础日数
- [ ] 已确认标准报酬月额等级差
- [ ] 已按最新规则判断
- [ ] 已提交并保存通知
- [ ] 已更新工资控除设置

## 9. 常见错误

- 只按基本工资计算，漏掉应计入报酬的手当。
- 把4月至6月“所属工资”和“实际支付工资”混淆。
- 收到决定通知后忘记更新工资控除。
- 固定性工资变化后没有检查月额变更届。
- 只改变社会保险设置，没有确认役员报酬的税务限制。
