# 年末調整・給与年次处理手册

## 1. 目的

本手册用于一人法人完成年末调整、源泉征收票、法定调书和工资支付报告书的年度工资作业。

即使只有一名役员，也应根据役员的工资、扶养申告书和年末调整对象条件逐项判断。

## 2. 年末调整整体流程

```text
申告书回收
  ↓
全年工资与控除资料确认
  ↓
年末调整计算
  ↓
差额反映到工资
  ↓
源泉征收票
  ↓
法定调书与工资支付报告书
```

## 3. 主要准备资料

- 給与所得者の扶養控除等（異動）申告書
- 基礎控除申告書
- 配偶者控除等申告書
- 所得金額調整控除申告書
- 保険料控除申告書
- 生命保险、地震保险等控除证明
- 社会保险料资料
- 住宅借入金等特别控除相关资料（符合时）
- 全年工资台账
- 已缴源泉所得税资料

使用表格名称和组合可能随年度更新，应从国税厅取得当年版本。

## 4. freee人事労務操作

1. 确认当年度所有工资已经确定。
2. 确认受给者基本信息、住所和扶养信息。
3. 回收并登记当年度年末调整申告书。
4. 登记保险料控除和其他证明资料。
5. 运行年末调整计算。
6. 检查全年支给额、所得控除和源泉征收税额。
7. 把过不足额反映到最后工资或指定处理月份。
8. 确定年末调整结果。
9. 生成并交付源泉征收票。
10. 准备法定调书和工资支付报告书。

## 5. 源泉所得税

工资中控除的源泉所得税属于公司代为保管的预り金，不是公司收入。

### 每月处理

1. 根据工资资料确认源泉税额。
2. 在工资仕訳中计入预り金。
3. 确认原有缴纳期限或纳期特例。
4. 通过 e-Tax 或缴纳书完成缴纳。
5. 核对预り金余额和缴纳记录。

纳期特例获得批准且工资受给人数常时少于10人时，一般期限为：

- 1月至6月支付部分：7月10日
- 7月至12月支付部分：次年1月20日

未确认批准状态时，不能擅自按半年汇总缴纳。

## 6. 源泉征收票与法定调书

### 操作步骤

1. 确认全年工资和年末调整结果。
2. 生成源泉征收票并交付受给者。
3. 确认向税务署提交源泉征收票的范围。
4. 制作給与所得の源泉徴収票等の法定調書合計表。
5. 制作其他符合条件的支払调书。
6. 通过 e-Tax 或其他认可方式提交。
7. 保存发送数据和接收通知。

一般提交期限为次年1月31日；遇节假日及年度特别规定时，以当年官方通知为准。

## 7. 工资支付报告书

工资支付报告书通常提交给受给者次年1月1日住所地的市区町村。

### 操作步骤

1. 确认受给者次年1月1日住所。
2. 制作个人别明细书和总括表。
3. 在 eLTAX 确认提交市区町村。
4. 发送并确认全部接收结果。
5. 保存提交控件。

地方税操作参见 [eLTAX手册](../tax/eLTAX手册.md)。

## 8. 住民税年度更新

1. 保存市区町村发送的特别征收税额决定通知书。
2. 确认6月至次年5月的每月金额。
3. 更新工资系统的住民税控除设置。
4. 每月从工资控除并按期限缴纳。
5. 离职或异动时确认异动届出书。

- [ ] 使用当年度最新申告书

- [ ] 扶养和配偶者信息已确认
- [ ] 控除证明资料齐全
- [ ] 社会保险料金额正确
- [ ] 年末调整差额已反映
- [ ] 源泉征收票已交付
- [ ] 法定调书已提交
