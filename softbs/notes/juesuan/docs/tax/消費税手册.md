# 消費税・freee税区分手册

## 1. 目的

本手册用于确认消费税课税状态、税率区分、freee登记、申告和缴纳流程。

> 消费税的课税事业者判定、简易课税、2割特例和仕入税额控除具有具体适用条件。实际选择前应参考当年国税厅资料或咨询税理士。

## 2. 开始前确认

- 当前是否为课税事业者
- 基准期间课税销售额
- 是否登记适格请求书发行事业者
- 原则课税或简易课税
- 是否存在中间申告和中间缴纳
- freee内的消费税设置
- 税込经理或税抜经理

不能只根据销售额或是否亏损单独判断申告义务。

## 3. 税率区分

### 10% 标准税率

2019年10月1日以后的一般课税交易。合计税率10%，其中消费税7.8%、地方消费税2.2%。

### 8% 旧税率

主要用于2019年9月30日以前的交易或依法适用经过措施的交易。

当前交易不能仅因为金额或内容相似就选择旧8%。

### 8% 轻减税率

一般适用于酒类、外食以外的饮食料品转让，以及符合条件的定期订阅报纸。

旧8%与轻减8%的国税、地方税构成不同，必须在 freee 中正确区分。

## 4. freee税区分操作

1. 打开公司消费税设置。
2. 确认课税事业者区分和课税方式。
3. 查看请求书和收据上的交易日期、税率和登录编号。
4. 判断课税销售、课税仕入、非课税、不课税或对象外。
5. 对当前交易选择10%或8%轻减。
6. 只有明确符合旧税率或经过措施时才使用旧8%。
7. 确认适格请求书所需记载事项。
8. 每月查看税区分错误和未设定交易。
9. 决算前核对税率别集计与申告资料。

税区分由交易实质决定，不能只根据勘定科目自动决定。

## 5. インボイス确认

- 发行者名称
- 适格请求书发行事业者登录编号
- 交易日期
- 交易内容
- 税率别合计金额
- 税率别消费税额
- 收件方名称等必要事项

无法满足要求时，应确认仕入税额控除和经过措施如何处理，不要自行补写对方登录编号。

## 6. 未払消费税

### 目的

管理决算时确定的消费税缴纳预定额和实际缴纳。

### 操作步骤

1. 完成全年税区分检查。
2. 确认中间缴纳和已缴税额。
3. 通过申告书确定消费税及地方消费税。
4. 根据税込、税抜经理方式确认决算仕訳。
5. 缴纳时从未払消费税消除。
6. 核对申告书、账簿和缴纳记录。

具体仕訳会受经理方式和中间缴纳影响，不能在申告税额确定前机械登记。

## 7. 申告步骤

1. 完成 freee 全年交易和税区分确认。
2. 确认课税方式与届出状态。
3. 确认课税销售、课税仕入和控除资料。
4. 反映中间缴纳。
5. 制作消费税及地方消费税申告书。
6. 核对税率别金额和缴纳税额。
7. 通过 e-Tax 或支持服务发送。
8. 确认接收通知。
9. 按期限完成缴纳。
10. 保存申告书、接收通知和缴纳记录。

法人课税事业者原则上应在课税期间结束后两个月内提交确定申告。延期等特殊处理必须查看官方规则。

## 8. 决算前检查表

- [ ] 已确认课税事业者状态
- [ ] 已确认原则课税或简易课税
- [ ] 已确认适用特例和届出状态
- [ ] 10%、旧8%、8%轻减区分正确
- [ ] インボイス资料完整
- [ ] 中间缴纳已反映
- [ ] 申告书与 freee 集计一致
- [ ] 未払消费税与申告税额一致
- [ ] e-Tax接收通知已保存
- [ ] 缴纳记录已保存

## 9. 常见错误

- 把旧8%和8%轻减混为同一税区分。
- 只按勘定科目判断税率。
- 忘记反映中间缴纳。
- 申告发送后忘记缴纳。
- 没确认 freee 方案是否支持当前申告类型。
