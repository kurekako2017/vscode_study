# eLTAX・地方税申告手册

## 1. 目的

本手册用于整理法人通过 eLTAX 处理地方税电子申告、电子提交和共同缴纳的基本流程。

## 2. 主要对象

- 法人都道府県民税
- 法人市町村民税
- 法人事業税
- 特別法人事業税
- 給与支払報告書
- 特別徴収相关手续

地方税的表格和处理可能因自治体而异，必须确认法人所在地及相关自治体的最新通知。

## 3. 事前准备

| 项目 | 确认内容 |
| --- | --- |
| eLTAX利用者ID | 是否属于当前法人 |
| PCdesk | 是否为当前支持版本 |
| 电子证明书 | 是否有效 |
| 提交目的地 | 都道府县和市区町村是否正确登记 |
| 法人税结果 | 法人税额和课税所得是否已确定 |
| 前期地方税资料 | 前期第六号样式等是否已准备 |

## 4. 法人地方税资料

一般需要确认：

- 法人税申告结果
- 法人税额
- 所得金额
- 资本金等金额
- 从业者人数和事务所信息
- 均等割区分
- 中间缴纳额
- 都道府县和市区町村的提交对象

即使法人税所得为负，也不能自动判断地方税为零。法人住民税均等割等可能仍然发生。

## 5. eLTAX申告步骤

1. 启动 PCdesk 或对应服务。
2. 选择当前法人和正确事业年度。
3. 确认都道府县及市区町村的提交目的地。
4. 根据法人税申告结果制作地方税数据。
5. 输入法人住民税、法人事业税和特别法人事业税资料。
6. 反映中间缴纳和已缴税额。
7. 运行形式检查并修正错误。
8. 使用电子证明书签名。
9. 向各提交目的地发送数据。
10. 确认全部接收通知，不只确认其中一个自治体。
11. 使用地方税共同缴纳等方法完成缴纳。
12. 保存申告数据、接收通知和缴纳记录。

## 6. 工资支付报告书

給与支払報告書一般提交给受给者次年1月1日住所地的市区町村。

### 操作步骤

1. 完成年末调整和源泉征收票。
2. 确认受给者次年1月1日的住所。
3. 制作給与支払報告書和总括表。
4. 在 eLTAX 登记正确市区町村。
5. 发送并确认接收结果。
6. 保存提交控件。

详细年末作业参见 [年末調整手册](../social/年末調整手册.md)。

## 7. 发送前检查表

- [ ] 法人所在地和事业年度正确
- [ ] 提交自治体没有遗漏
- [ ] 法人税额和所得金额与 e-Tax 一致
- [ ] 资本金、人数、事务所信息正确
- [ ] 中间缴纳额正确
- [ ] 都道府县和市区町村数据均已制作
- [ ] 电子签名有效
- [ ] 所有接收通知均已确认
- [ ] 申告和缴纳均已完成

## 8. 常见问题

### e-Tax发送后，地方税会自动完成吗？

不会。国税和地方税是不同流程，需要分别确认 e-Tax 和 eLTAX。

### 只向都道府县发送即可吗？

通常还需要确认市区町村分。应根据法人所在地和事务所情况确认全部提交目的地。

### 发送成功是否代表已经缴税？

不代表。发送后必须再次确认共同缴纳或其他缴纳流程。
