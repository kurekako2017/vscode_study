# 14. 本ガイドの利用上の注意 / 本指南使用注意

本ガイドは、一人法人が毎月の作業を漏れなく進めるための実務チェック用資料であり、
個別案件に対する税務・法律・社会保険上の最終判断を行うものではない。

本指南用于帮助一人法人按月完成会计、税务和社会保险工作，不能替代税理士、社会保险劳务士、税务署、自治体或日本年金机构对具体事项的最终判断。

特に次の場合は、処理を確定する前に公式窓口または専門家へ確認する。

- 初回の法人税申告
- 赤字・繰越欠損金・還付がある場合
- 消費税の課税方式、2割特例、簡易課税を判断する場合
- 役員報酬を変更する場合
- 固定資産、ソフトウェア、交際費の判断が難しい場合
- 税務調査、修正申告、更正の請求が必要な場合

税率、申告書様式、freeeの画面・料金・対応機能は変更される可能性があるため、毎年、作業開始前に公式情報を確認する。

# 15. freee会計：重要勘定科目の運用 / 重要科目操作

## 15.1 現金 / 现金

### 目的

実際に会社が保有・使用した現金だけを記録する。クレジットカード、銀行振込、役員個人の立替を「現金」にしない。

只登记公司实际持有和支付的现金。信用卡、银行转账和法人代表个人垫付款不能误记为现金。

### freee操作

1. `レポート` または `会計帳簿` から `総勘定元帳` を開く。
2. 勘定科目で `現金` を選択する。
3. 領収書、出金記録、期末の実際の手元現金を照合する。
4. 実際は個人立替なら決済口座を `役員借入金` に修正する。
5. 実際は銀行・カード支払なら正しい口座またはカードへ修正する。
6. マイナス残高や根拠不明残高を残したまま決算を確定しない。

注意：現金を使わない会社でも、帳簿残高を機械的にゼロへ変更してはいけない。必ず元取引を確認して修正する。

## 15.2 普通預金 / 普通存款

### 目的

freee上の銀行残高と銀行明細の実残高を一致させる。

### freee操作

1. `口座` から対象銀行を開く。
2. 同期停止、重複明細、未取得期間がないか確認する。
3. `自動で経理` の未処理明細を1件ずつ登録または無視判定する。
4. 月末日のfreee残高と銀行の月末残高を照合する。
5. 差額がある場合は、開始残高、重複登録、手入力、振替取引を確認する。
6. 照合日と確認者を月次チェックリストへ記録する。

## 15.3 売掛金 / 应收账款

### 目的

サービス提供・請求の時点と入金の時点が異なる場合に、売上を正しい期間へ計上する。

### freee操作

1. 請求書または契約書で売上計上日と金額を確認する。
2. 未入金の売上を `売上高 / 売掛金` として登録する。
3. 入金明細が同期されたら、該当する売掛金の消込として登録する。
4. 月末に `試算表` または `総勘定元帳` で売掛金残高を確認する。
5. 請求先別に、請求済・入金済・未入金を照合する。
6. 長期間残る売掛金は、入金漏れ、消込漏れ、貸倒れ可能性を確認する。

基本仕訳：

```text
売上計上時：借方 売掛金 / 貸方 売上高
入金時　　：借方 普通預金 / 貸方 売掛金
```

## 15.4 未払金 / 应付未付款

### 目的

当期に利用したサービスや購入物で、支払いが翌月以降になるものを当期費用へ計上する。

### freee操作

1. 請求書・カード明細から利用日、請求日、支払日を確認する。
2. 決算日までに発生した費用を `費用科目 / 未払金` として登録する。
3. 後日の銀行・カード支払を未払金の決済として消し込む。
4. 月末と決算前に未払金元帳を開く。
5. 支払済みなのに残っているもの、二重計上、翌期費用を確認する。

基本仕訳：

```text
費用発生時：借方 通信費等 / 貸方 未払金
支払時　　：借方 未払金 / 貸方 普通預金等
```

## 15.5 役員借入金 / 董事借款

### 目的

代表者が会社経費を個人資金で立て替えた場合、会社が代表者へ返すべき金額を管理する。

### freee操作

1. 領収書・請求書が会社業務に必要な支出か確認する。
2. 取引の勘定科目を `通信費`、`旅費交通費` など実態に合わせる。
3. 決済口座を `役員借入金` として登録する。
4. 会社口座から代表者へ精算したとき、`役員借入金 / 普通預金` で消し込む。
5. 毎月、残高が領収書と精算記録の合計に一致するか確認する。
6. 長期・多額の残高や反対残高は税理士へ確認する。

注意：役員借入金と役員貸付金は意味が逆である。会社が役員へ貸した状態を誤って役員借入金にしない。

## 15.6 未払消費税 / 应付消费税

### 目的

課税事業者が決算で確定した消費税等の納付予定額と、実際の納付を管理する。

### freee操作

1. `設定` で課税事業者区分、課税方式、経理方式を確認する。
2. 各取引の税区分が正しいか確認する。
3. freee申告または申告書で納付税額を確定する。
4. 税込経理の場合の決算振替は、freeeの設定と申告結果に基づき確認する。
5. 納付時は、確定済みの未払消費税を普通預金から消し込む。
6. 申告書、納付情報、仕訳金額が一致するか確認する。

例示仕訳は会社の経理方式や中間納付の有無で変わるため、申告書確定前に自動で決めない。

# 16. 給与・源泉所得税・住民税 / 工资与代扣税

## 16.1 毎月の給与処理 / 每月工资处理

### freee操作

1. 役員報酬・給与の固定額、勤怠、手当、控除項目を確認する。
2. freee人事労務で給与計算を実行する。
3. 健康保険、厚生年金、雇用保険、源泉所得税、住民税を確認する。
4. 給与明細を確定し、支給対象者へ交付する。
5. 差引支給額を法人銀行口座から振り込む。
6. freee会計へ連携された給与仕訳を確認する。
7. 給与台帳、振込記録、納付記録を保存する。

基本確認式：

```text
総支給額
－ 社会保険料
－ 源泉所得税
－ 住民税
＝ 差引支給額
```

## 16.2 源泉所得税 / 预扣所得税

給与や対象となる報酬から預かった源泉所得税は会社の収益ではなく、国へ納付する預り金である。

### 運用手順

1. 給与確定時に源泉徴収税額表と申告書区分を確認する。
2. 給与仕訳で源泉所得税を `預り金` として計上する。
3. 原則の納期限、または承認済みの納期の特例を確認する。
4. e-Taxまたは納付書で納付する。
5. 納付後、預り金残高と納付額を照合する。

納期の特例が承認され、給与の支給人員が常時10人未満の場合の一般的な納期限：

- 1月～6月支給分：7月10日
- 7月～12月支給分：翌年1月20日

特例の承認状況が不明な場合は、半年分をまとめて納付せず、所轄税務署へ確認する。

## 16.3 住民税 / 居民税

特別徴収の場合、会社は給与から住民税を控除し、原則として翌月10日までに各自治体へ納入する。

### 運用手順

1. 毎年届く特別徴収税額決定通知書を確認する。
2. 6月から翌年5月までの月額を給与システムへ設定する。
3. 給与から住民税を控除し `預り金` として計上する。
4. 指定された方法で自治体へ納入する。
5. 納入額、通知書、給与台帳の控除額を照合する。
6. 退職・異動があれば給与所得者異動届出書の要否を確認する。

# 17. 社会保険 / 社会保险

## 17.1 算定基礎届 / 定时决定

目的：4月・5月・6月に支払った報酬をもとに、原則として9月から翌年8月までの標準報酬月額を決定する。

2026年度の提出期間は、日本年金機構の案内では2026年7月1日から7月10日までである。年度ごとに期限を再確認する。

### 操作手順

1. 日本年金機構から届く対象者情報を確認する。
2. 4月、5月、6月の給与台帳を準備する。
3. 支払基礎日数と報酬に含める項目を確認する。
4. 算定基礎届を作成する。
5. 電子申請、郵送または窓口で期限内に提出する。
6. 標準報酬月額決定通知書を保存する。
7. 9月適用分から給与控除額を更新する。

一人法人でも、代表取締役が健康保険・厚生年金保険の被保険者なら対象確認が必要である。

## 17.2 月額変更届 / 随时改定

基本給や役員報酬など固定的賃金が変わり、その後3か月の平均報酬により標準報酬月額が原則2等級以上変動するなど、随時改定の要件を満たした場合に提出する。

### 操作手順

1. 固定的賃金が変動した月を特定する。
2. 変動月から3か月間の給与台帳と支払基礎日数を確認する。
3. 従前と新しい標準報酬月額の等級差を確認する。
4. 日本年金機構の最新要件で対象か判定する。
5. 対象なら月額変更届を速やかに提出する。
6. 改定通知を受領後、適用月の社会保険料を給与へ反映する。

役員報酬の変更には税務上の制限もあるため、社会保険だけを見て変更日や金額を決めない。

# 18. 決算書・PL・BS / 财务报表

## 18.1 決算書 / 决算文件

一般的な決算書には次の資料が含まれる。

- 貸借対照表（BS）
- 損益計算書（PL）
- 株主資本等変動計算書
- 個別注記表

### freee操作

1. すべての銀行・カード・現金取引を登録する。
2. 売掛金、未払金、固定資産、役員借入金を確認する。
3. `レポート` で月次推移と残高を確認する。
4. `決算申告` から決算書作成画面を開く。
5. 会計期間、会社情報、注記内容を確認する。
6. BS・PL・株主資本等変動計算書・個別注記表を出力する。
7. 株主総会等の会社法上の承認手続を確認し、確定版を保存する。

## 18.2 損益計算書（PL）/ 利润表

PLは一定期間の売上、費用、利益を示す。

決算前の確認項目：

- 売上の計上漏れ・期間ずれ
- 通信費、外注費、旅費交通費、交際費の異常増減
- 私的支出の混入
- 減価償却費の計上
- 法人税等計上前後の利益

## 18.3 貸借対照表（BS）/ 资产负债表

BSは決算日時点の資産、負債、純資産を示す。

決算前の確認項目：

- 普通預金が銀行残高と一致している
- 現金が実在し、マイナスではない
- 売掛金が請求・入金記録と一致している
- 未払金が請求・支払記録と一致している
- 役員借入金の根拠資料がある
- 固定資産台帳と固定資産残高が一致している
- 未払消費税、預り金の残高に根拠がある

# 19. 法人税申告書の主要別表 / 法人税主要附表

別表は互いに連動する。次の4表だけで全法人の申告が完結するとは限らず、別表二、別表十五、適用額明細書、勘定科目内訳明細書、法人事業概況説明書などが必要になる場合がある。

## 19.1 別表一

役割：法人税の課税所得と税額を集約し、最終的な申告税額を示す中心表。

確認事項：

- 事業年度、法人番号、所在地
- 所得金額
- 法人税額、控除税額、中間納付額
- 翌期へ持ち越す金額

## 19.2 別表四

役割：会計上の当期利益から、税務上の所得金額へ調整する。

中文理解：把会计利润通过“加算、减算”调整为税法上的所得。

主な確認例：

- 損金不算入の法人税・住民税
- 交際費等の損金不算入
- 減価償却超過額
- 前期以前の留保項目の認容

## 19.3 別表五（一）

役割：利益積立金額と資本金等の額の税務上の増減を管理する。

別表四の「留保」項目と連動するため、前期申告書からの繰越を確認する。

## 19.4 別表五（二）

役割：法人税、地方法人税、住民税、事業税などの発生・納付・未納状況を管理する。

納税充当金、未払法人税等、中間納付、確定納付の関係を確認する。

### 作成手順

1. 前期申告書一式を準備する。
2. 当期の確定決算書と総勘定元帳を準備する。
3. 税務調整項目を一覧化する。
4. 別表四を作成し、所得金額を確定する。
5. 別表五（一）へ留保項目を連動する。
6. 別表五（二）で税金の発生・納付状況を照合する。
7. 別表一で税額を集約する。
8. 必要なその他別表・添付資料を確認する。
9. e-Tax送信前に決算書と全別表の整合性を確認する。

初回申告や前期繰越がある場合は、前期資料なしで別表五を新規推測しない。

# 20. 消費税の税率区分 / 消费税税率分类

## 20.1 10%（標準税率）

2019年10月1日以後の一般的な課税取引に適用される標準税率。国税7.8%と地方消費税2.2%の合計が10%である。

## 20.2 8%（旧税率・経過措置）

2019年9月30日以前の取引や、法令上の経過措置が適用される取引で使用される旧税率。

現在の取引について、単に税率が低いという理由で選択してはいけない。

## 20.3 8%（軽減税率）

2019年10月1日以後の軽減税率。一般に、酒類・外食を除く飲食料品の譲渡と、一定の定期購読新聞が対象となる。

旧8%と軽減8%は合計税率が同じでも国税・地方税の内訳が異なるため、freee上で区別する。

### freee操作

1. 証憑の日付、取引内容、請求書の税率表示を確認する。
2. 課税売上・課税仕入・対象外・非課税・不課税を判定する。
3. 税率を `10%`、`8%（軽）`、必要な場合のみ旧 `8%` から選ぶ。
4. 適格請求書発行事業者番号と必要記載事項を確認する。
5. 月次で税区分別の取引を確認する。
6. 決算前に消費税集計表と申告書の税率別金額を照合する。

税区分は取引実態によって決まり、勘定科目だけでは決まらない。

# 21. 電子申告：e-Tax・eLTAX / 电子申报

## 21.1 e-Tax

主な対象：

- 法人税
- 地方法人税
- 消費税
- 源泉所得税
- 法定調書

### 操作手順

1. 利用者識別番号、電子証明書、暗証番号の有効性を確認する。
2. 最新のe-Taxソフトまたは対応サービスを準備する。
3. 申告データと添付資料を作成する。
4. 決算書、別表、勘定科目内訳明細書等の整合性を確認する。
5. 電子署名して送信する。
6. 受信通知とメッセージボックスを確認する。
7. 納付情報を確認し、期限内に納付する。
8. 送信データ、受信通知、納付記録を年度別に保存する。

## 21.2 eLTAX

主な対象：

- 法人都道府県民税
- 法人市町村民税
- 法人事業税
- 特別法人事業税
- 給与支払報告書

### 操作手順

1. eLTAX利用者ID、提出先自治体、電子証明書を確認する。
2. PCdeskまたは対応サービスで申告先を設定する。
3. 法人税申告結果をもとに地方税申告データを作成する。
4. 都道府県分と市区町村分の提出先・様式を確認する。
5. 電子署名して送信する。
6. 受付完了通知とエラーの有無を確認する。
7. 地方税共通納税等で期限内に納付する。
8. 申告データ、受付通知、納付記録を保存する。

電子送信完了と納税完了は別作業である。送信後は必ず納付状況も確認する。

# 22. freee申告のプランと代替方法 / freee申报方案

## 22.1 用語整理

- `freee会計`：日常記帳、帳簿、決算書作成の基盤。
- `freee申告スタータープラン`：法人事業所向けfreee申告の契約プランの一つ。
- `freee申告スタンダードプラン`：より広い申告関連機能を提供する契約プラン。
- `申告プラン未契約`：freee会計だけを契約し、freee申告法人税の有料機能を利用できない状態を含む。

プラン名称・料金・対応帳票は変更されるため、購入前にfreee公式の最新比較表と画面上の利用要件を確認する。

## 22.2 申告スターター・スタンダードで確認すべきこと

2026年6月確認時点のfreee公式情報では、法人向けスターター・スタンダードは法人税の確定申告を対象とし、修正申告、予定申告、仮決算に基づく中間申告には対応しないと案内されている。

購入・作業開始前の確認事項：

1. 現在のfreee会計プランが購入条件を満たすか。
2. 当期の事業年度が対応年度か。
3. 必要な別表・地方税・内訳書・概況書に対応するか。
4. 確定申告以外の申告ではないか。
5. 電子申告に必要なOS、アプリ、電子証明書が利用可能か。
6. 消費税申告が別契約・別機能になっていないか。

## 22.3 利用不可・未対応時の代替方法

### 方法A：公式電子申告ソフトを利用

1. freee会計で記帳と決算整理を完了する。
2. BS、PL、株主資本等変動計算書、個別注記表、総勘定元帳を出力する。
3. 国税はe-Tax、地方税はeLTAXで申告データを作成する。
4. 必要別表と添付書類を手動で確認する。
5. 送信後、受付通知と納付記録を保存する。

### 方法B：税理士へ申告部分だけ依頼

1. freee会計の記帳と証憑整理を自社で完了する。
2. 試算表、決算書、元帳、前期申告書、届出書控えを準備する。
3. 税理士へ法人税・地方税・消費税申告を依頼する。
4. 完成した申告書一式と翌期繰越情報を受領する。

### 方法C：対応プランへ変更

1. 必要機能と利用回数を整理する。
2. freee公式サポートで対応可否を確認する。
3. 料金だけでなく、対応帳票と申告種別を比較する。
4. バックアップを取得してから申告作業を開始する。

「freee会計から決算書を出力できる」ことと「法人税申告書を正しく完成できる」ことは別である。

# 23. 年間スケジュール / 1月至12月年度计划

以下は一般的な一人法人の管理例である。決算月、給与支給日、自治体、特例適用の有無に応じて調整する。4月決算法人の場合、法人税・消費税の原則的な申告期限は6月末となる。

| 月 | 主な作業 / 主要工作 | 提出・納付資料 / 提交资料 | 確認ポイント |
|---|---|---|---|
| 1月 | 年末調整後処理、前年給与の集計 | 源泉所得税（納期特例7～12月分）、法定調書、給与支払報告書、該当時は償却資産申告 | 原則期限は1月20日または1月31日前後。休日調整と当年案内を確認 |
| 2月 | 1月提出資料の控え整理、月次決算 | 通常は定例納付中心 | 自治体・税務署の受付結果、未処理エラー確認 |
| 3月 | 年度末取引がある取引先の請求確認 | 必要に応じ届出書 | 契約更新、固定資産購入、未収未払の把握 |
| 4月 | 4月決算の場合は締め処理 | 決算資料の収集 | 売上・費用の期間帰属、銀行、売掛金、未払金、固定資産を確定 |
| 5月 | 決算整理、申告書作成 | BS、PL、株主資本等変動計算書、個別注記表、別表、内訳書 | 前期申告書との連動、税務調整、消費税区分を確認 |
| 6月 | 4月決算法人の申告・納付、住民税更新 | 法人税、地方法人税、消費税、法人住民税、法人事業税等 | 原則は事業年度終了翌日から2か月以内。e-Tax/eLTAX送信と納付を別々に確認 |
| 7月 | 算定基礎届、源泉所得税納期特例 | 算定基礎届、源泉所得税（1～6月分） | 原則7月10日。年度の休日・公式案内を確認 |
| 8月 | 決算・申告控えの保管、月次見直し | 通常は定例納付中心 | 翌期へ繰り越す別表五、固定資産、未払税金を整理 |
| 9月 | 定時決定後の社会保険料更新 | 必要に応じ月額変更届 | 新しい標準報酬月額の適用月と給与控除を確認 |
| 10月 | 上期業績レビュー、税金見込更新 | 必要に応じ中間申告・納付 | 中間申告義務、消費税中間申告の通知を確認 |
| 11月 | 年末調整準備 | 扶養控除等申告書、保険料控除申告書等を回収 | 最新様式、マイナンバー管理、証明書不足を確認 |
| 12月 | 年末調整、翌年給与設定 | 源泉徴収票、法定調書、給与支払報告書の準備 | 年末調整結果を最終給与へ反映し、1月提出物を一覧化 |

毎月共通：給与、源泉所得税・住民税・社会保険の納付、銀行同期、証憑保存、売掛金・未払金確認を行う。
