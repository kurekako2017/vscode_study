# freee会计・法人决算知识库 Backlog

本文件是当前工作的第一入口，用于统一管理已完成任务、待办任务和长期维护事项。

## 使用规则

1. 每次开始工作前，先读取 `backlog.md`，再读取 `task.md`。
2. 开始任务时，不删除既有记录；未完成事项保持 `[ ]`。
3. 完成任务后，将对应事项改为 `[x]`。
4. 每次完成任务后，同步更新 `backlog.md`、`task.md` 和 `updates/CHANGELOG.md`。
5. 涉及真实公司数据时，必须遵守 `AGENTS.md` 的 Private Data Rules。
6. `private/` 中的内容不得提交、推送或复制到公开文档。
