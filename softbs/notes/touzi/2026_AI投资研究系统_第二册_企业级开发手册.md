# 2026 AI投资研究系统（第二册）

------------------------------------------------------------------------

# 第1章 项目总览

目标不是开发一个自动买卖机器人，而是打造一个 AI 投资研究平台。

核心能力：

-   全球股票扫描
-   ETF 分析
-   基金分析
-   财报分析
-   新闻分析
-   AI 打分
-   每日投资报告

# 第4章 AI Agent 分工

## Research Agent

负责： - 股票基本面 - 行业 - 竞争优势

## News Agent

负责： - 新闻 - 财报 - SEC - Reddit - X

## ETF Agent

负责： - ETF资金流 - 成分股变化

## Portfolio Agent

负责： - 持仓分析 - 风险暴露 - 再平衡建议

## Tax Agent

负责： - 日本股票税务 - 特定口座 - 损益通算 - 三年亏损结转
