# WSL2 无损迁移到 D 盘完整教程（2026版）

适用环境：
- Windows 11
- WSL2
- Ubuntu
- VSCode Remote WSL
- Ollama
- Codex CLI
- Continue
- LangChain / LangGraph / MCP

## 为什么要迁移

当前你的 WSL 虚拟磁盘位于：

C:\Users\victorkure\AppData\Local\wsl

大小约 66.5GB。

随着 Ollama、LangGraph、MCP 项目增加，未来可能超过 100GB。

---

## 第一步：关闭 WSL

```powershell
wsl --shutdown
```

## 第二步：导出 Ubuntu

```powershell
wsl --export Ubuntu D:\WSLBackup\ubuntu.tar
```

## 第三步：确认备份存在

确认：

D:\WSLBackup\ubuntu.tar

## 第四步：注销旧 Ubuntu

⚠️ 确认备份成功后执行

```powershell
wsl --unregister Ubuntu
```

## 第五步：导入到 D 盘

```powershell
wsl --import Ubuntu D:\WSL\Ubuntu D:\WSLBackup\ubuntu.tar --version 2
```

## 第六步：验证

```bash
python3 --version
ollama list
```

确认 Python、Ollama、VSCode Remote WSL、Codex Skills 均正常。

---
