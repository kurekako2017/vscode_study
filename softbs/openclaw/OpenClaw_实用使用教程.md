# OpenClaw 实用使用教程

> 适用人群：已经在 Win11 + WSL 里把 OpenClaw 跑起来，并且已经通过 Telegram 等渠道成功连上机器人。

---

## 1. 先明确一件事：OpenClaw 现在能做什么

OpenClaw 本质上不是单纯聊天机器人，它更像是一个“接在聊天软件里的 AI 助手入口”。

你现在最适合用它做这几类事：

- 日常问答
- 文案改写
- 翻译
- 总结长文本
- 帮你整理待办
- 帮你写日报、周报、邮件
- 帮你生成表格草稿
- 如果后面再接入更多工具，还可以查网页、读邮件、调接口、执行自动化流程

---

## 2. 最推荐的 5 个立即可用场景

### 2.1 查股票和做简单行情解读

如果当前 OpenClaw 没有接实时金融数据工具，它不能保证给你实时股价；但非常适合帮你分析你贴进去的数据。

可直接发送：

```text
请用中文帮我分析这只股票：
代码：AAPL
现价：213.50
今日涨跌：+1.82%
我的关注点：
1. 这更像短期反弹还是趋势延续
2. 我应该重点看哪些风险
3. 请给我一个 3 行总结
```

### 2.2 总结邮件内容

可直接发送：

```text
下面是一封邮件，请帮我：
1. 用中文总结核心内容
2. 提取所有待办事项
3. 标记哪些事项需要我今天处理
4. 最后给我一版可以直接回复对方的英文邮件

邮件内容：
（把邮件正文粘贴进来）
```

### 2.3 写日报 / 周报

可直接发送：

```text
请把下面这些内容整理成一版正式的中文日报：
1. 今天处理了 OpenClaw 的 Telegram 接入
2. 修复了 token 配置问题
3. 测试了消息收发
4. 还在研究后续工具接入

要求：
1. 语气专业一点
2. 分成“今日完成 / 遇到问题 / 明日计划”
3. 不要太口语化
```

### 2.4 翻译和润色

```text
把下面内容翻译成自然商务英文，不要直译，要像真实邮件表达：
（粘贴中文内容）
```

```text
请帮我把下面这段英文改得更专业、更简洁，保留原意：
（粘贴英文内容）
```

### 2.5 整理待办事项

```text
请把下面这些碎片信息整理成待办清单，按优先级排序，并拆成：
1. 今天必须做
2. 本周要做
3. 可以延后

内容：
1. 要回复客户邮件
2. 要看 OpenClaw Telegram 是否稳定
3. 想研究 Gemini 免费方案
4. 要整理本周周报
5. 要给服务器做一次备份检查
```

## 3. 最适合一开始就用起来的方式

先把它当成“文本处理助理”：

- 总结助手
- 翻译助手
- 写作助手
- 整理助手

这类任务现在立刻能用，不依赖额外插件和接口。

## 4. 可以直接复制测试的指令

### 4.1 会议纪要整理

```text
下面是会议记录，请帮我整理成会议纪要：
1. 先总结会议目标
2. 再列关键结论
3. 再列待办项和负责人
4. 最后给一个简短风险提示

会议内容：
（粘贴内容）
```

### 4.2 帮我回客户

```text
下面是客户发来的消息，请帮我写一个礼貌、专业、简洁的回复。
语气要求：商务正式，但不要太生硬。

客户消息：
（粘贴内容）
```

### 4.3 帮我压缩成长话短说版

```text
请把下面内容压缩成 5 句话以内，保留核心信息，不要丢重点：
（粘贴内容）
```

### 4.4 帮我列行动计划

```text
基于下面目标，请给我一个可执行的行动计划。
要求分成：
1. 今天
2. 这周
3. 本月

目标：
（粘贴你的目标）
```

### 4.5 帮我做对比

```text
请帮我对比下面两个方案，输出成表格：
1. 优点
2. 缺点
3. 成本
4. 风险
5. 适合什么场景

方案 A：
（内容）

方案 B：
（内容）
```

## 5. 股票、邮箱这类“工具型需求”怎么理解

### 5.1 只做内容分析

如果你把数据贴给它，它现在就能做。

### 5.2 自动去外部系统取数据

如果你想让它自动查实时股价、自动看未读邮件，那就需要后续再接工具或 API。

也就是说：

- 现在已经能用的是“你提供内容，它来分析”
- 以后还能扩展成“它自己去系统里拿数据再分析”

## 6. 下一步最值得加的扩展能力

- 网页信息获取

- 表格和文档处理
- 个人自动化

---

## 7. 最实在的建议

不要一开始就把 OpenClaw 当成“全自动助手”，先把它当成“在 Telegram 里的办公副驾驶”。

先用它做：

- 总结
- 翻译
- 改写
- 回复草稿
- 待办整理

这 5 件事一旦用顺了，它就已经很值了。
