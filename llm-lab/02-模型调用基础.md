# 模型调用基础

## 1. 这一阶段要解决什么

第一阶段只做一件事：

- 稳定调用模型

你至少要会：

- 发送请求
- 控制模型参数
- 读取结果
- 处理异常

## 2. 为什么这一步重要

日本现场很多生成 AI PoC，第一版其实就是：

- Python 调模型
- 输入文本
- 输出结果

如果这一步都不稳，后面的：

- 结构化输出
- `RAG`
- API 化

都会很虚。

## 3. 必学内容

- API Key 管理
- `system / user` 消息
- 模型参数
- 超时与错误处理
- CLI 运行方式
