# Java Lab

面向对日开发项目的 Java 系统学习资料库。

本资料库的目标不是只讲 Java 语法，而是把日本企业项目里常见的开发环境、工具链、框架、分层方式、文档习惯、测试与交付方式一起整理出来，方便你按项目视角学习。

## 学习目标

- 掌握 Java 基础语法、面向对象、集合、异常、IO、多线程等核心能力。
- 理解对日项目中常见的 Java Web 开发结构和分层设计。
- 熟悉日本项目里常见的 IDE、应用服务器、构建工具、数据库连接工具、版本管理工具。
- 建立从环境搭建到编码、调试、测试、部署、维护的完整认知。
- 能看懂日文项目资料中的常见术语，例如 `設計書`、`単体テスト`、`結合テスト`、`障害票`、`保守運用`。

## 适合人群

- 想系统学习 Java 的初学者
- 准备进入对日外包或驻场开发项目的工程师
- 已经会一点 Java，但对企业项目工具链不熟悉的人
- 想从传统 Java Web 项目过渡到 Spring Boot 项目的人
- 想系统了解 Spring Boot 以及它和 React / Vue / MyBatis / JPA / Redis / 消息队列如何组合的人
- 想继续拆分 Spring Boot 到具体专题实战的人

## 使用建议

- 先把环境搭起来，再学语法，再学 Web，再学数据库，再看项目结构。
- 学每个主题时，不要只看概念，建议同步做一个小项目。
- 对日项目通常更强调规范、文档、流程和沟通，学习时要把这些内容当成正式技能，而不是附加项。
