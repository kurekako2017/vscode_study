# EC2 学习笔记（中日对照）

EC2 是 AWS 的虚拟服务器服务，适合学习“云上主机、网络、安全组、磁盘、启动流程”这些基础概念。

## 1. 这个服务是什么 / このサービスは何か

- 中文：EC2 提供可配置的云服务器实例，你可以像管理普通主机一样管理它。
- 日本語：EC2 は設定可能なクラウドサーバーインスタンスを提供し、通常のマシンのように扱えます。

## 2. 核心概念 / 基本概念

| 英文 | 中文说明 | 日本語説明 |
|---|---|---|
| Instance | 实例，一台运行中的虚拟机 | インスタンス。稼働中の仮想マシン |
| AMI | 镜像，实例启动模板 | AMI。起動テンプレートとなるイメージ |
| Instance Type | 实例规格 | インスタンスタイプ。CPU/メモリ構成 |
| Security Group | 安全组，实例级防火墙 | セキュリティグループ。インスタンス単位の firewall |
| Key Pair | 登录密钥对 | キーペア |
| EBS | 块存储 | EBS。ブロックストレージ |

## 3. 学习重点 / 学習ポイント

- 中文：理解 AMI、实例类型、安全组、密钥对、EBS 之间的关系。
- 日本語：AMI、インスタンスタイプ、セキュリティグループ、キーペア、EBS の関係を理解する。
- 中文：理解启动、停止、重启、终止的生命周期差异。
- 日本語：起動、停止、再起動、終了のライフサイクルの違いを理解する。
- 中文：理解“云服务器”和“容器”不是一回事。
- 日本語：クラウドサーバーとコンテナは別物であることを理解する。

## 4. LocalStack 里怎么理解 / LocalStack でどう捉えるか

- 中文：LocalStack 对 EC2 的支持更偏概念验证，不一定等同真实 EC2。
- 日本語：LocalStack の EC2 は概念理解寄りで、実際の EC2 と完全一致するとは限りません。
- 中文：建议把 EC2 放在架构层面学习，真正的实例操作可以结合真实 AWS 文档。
- 日本語：EC2 はアーキテクチャの理解を中心に学び、実操作は AWS 公式ドキュメントも併用するのがおすすめです。

## 5. 常见场景 / よくある場面

- 中文：部署 Web 应用、API 服务、批处理任务、SSH 运维。
- 日本語：Web アプリ、API サービス、バッチ処理、SSH 運用。

## 6. 常见坑 / よくある落とし穴

- 中文：忘记安全组放行端口。
- 日本語：セキュリティグループでポート開放を忘れる。
- 中文：把实例规格和网络问题混在一起排查。
- 日本語：インスタンススペックとネットワーク問題を混同する。
