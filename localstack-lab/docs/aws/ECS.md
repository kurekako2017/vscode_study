# ECS 学习笔记（中日对照）

ECS 是 AWS 的容器编排服务，适合学习服务调度、任务定义、服务发现这些概念。

## 1. 这个服务是什么 / このサービスは何か

- 中文：ECS 用来管理容器任务和服务，帮助你在 AWS 上运行容器化应用。
- 日本語：ECS はコンテナタスクとサービスを管理し、AWS 上でコンテナアプリを運用するためのサービスです。

## 2. 核心概念 / 基本概念

| 英文 | 中文说明 | 日本語説明 |
|---|---|---|
| Cluster | 集群 | クラスター |
| Task | 任务 | タスク |
| Task Definition | 任务定义 | タスク定義 |
| Service | 服务 | サービス |
| Container | 容器 | コンテナ |
| Scheduling | 调度 | スケジューリング |

## 3. 学习重点 / 学習ポイント

- 中文：理解 Cluster、Task、Service、Task Definition 的关系。
- 日本語：Cluster、Task、Service、Task Definition の関係を理解する。
- 中文：理解 ECS 和 ECR 的组合方式。
- 日本語：ECS と ECR の連携を理解する。
- 中文：理解 ECS 管的是“运行和调度”，不是“构建镜像”。
- 日本語：ECS は「実行と調度」を管理し、イメージ作成は担当しないことを理解する。

## 4. LocalStack 里怎么练 / LocalStack での練習方法

- 中文：先练任务定义，再练服务部署。
- 日本語：まずタスク定義を作り、その後サービスとして配置する練習をする。
- 中文：把 ECS 和 ECR、Fargate 一起理解成容器平台。
- 日本語：ECS、ECR、Fargate をセットでコンテナ基盤として理解する。

## 5. 常见坑 / よくある落とし穴

- 中文：把 ECS 当成镜像仓库。
- 日本語：ECS をイメージレジストリだと誤解する。
- 中文：把任务定义和服务配置混淆。
- 日本語：タスク定義とサービス設定を混同する。
