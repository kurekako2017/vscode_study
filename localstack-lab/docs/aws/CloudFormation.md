# CloudFormation 学习笔记（中日对照）

CloudFormation 是 AWS 的模板化基础设施管理服务，适合学习 IaC 的基础思路。

## 1. 这个服务是什么 / このサービスは何か

- 中文：CloudFormation 用模板文件描述 AWS 资源，然后一键创建、更新、删除。
- 日本語：CloudFormation はテンプレートで AWS リソースを定義し、一括で作成・更新・削除するサービスです。

## 2. 核心概念 / 基本概念

| 英文 | 中文说明 | 日本語説明 |
|---|---|---|
| Template | 模板 | テンプレート |
| Stack | 堆栈，资源集合 | スタック |
| Change Set | 变更集 | 変更セット |
| Parameter | 参数 | パラメータ |
| Output | 输出值 | 出力 |
| Resource | 资源定义 | リソース定義 |

## 3. 学习重点 / 学習ポイント

- 中文：理解模板、堆栈、参数、输出之间的关系。
- 日本語：テンプレート、スタック、パラメータ、出力の関係を理解する。
- 中文：理解“用代码管理基础设施”的思路。
- 日本語：「コードでインフラを管理する」考え方を理解する。
- 中文：理解 CloudFormation 和 CDK、Terraform 的区别。
- 日本語：CloudFormation と CDK、Terraform の違いを理解する。

## 4. LocalStack 里怎么练 / LocalStack での練習方法

- 中文：先写最小模板，再逐步增加资源。
- 日本語：まず最小テンプレートを書き、徐々にリソースを追加する。
- 中文：把 S3、Lambda、IAM 等资源写进模板里观察变化。
- 日本語：S3、Lambda、IAM などをテンプレートに書いて変化を確認する。

## 5. 常见坑 / よくある落とし穴

- 中文：模板过大，难以维护。
- 日本語：テンプレートが大きくなりすぎて保守しづらい。
- 中文：参数和输出设计混乱。
- 日本語：パラメータと出力の設計が混乱する。
