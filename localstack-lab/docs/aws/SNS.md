# SNS 学习笔记（中日对照）

SNS 是 AWS 的发布/订阅通知服务，适合学习广播通知、主题订阅和事件联动。

## 1. 这个服务是什么 / このサービスは何か

- 中文：SNS 用于把一条消息广播给多个订阅者。
- 日本語：SNS は 1 つのメッセージを複数の購読者へ配信するためのサービスです。

## 2. 核心概念 / 基本概念

| 英文 | 中文说明 | 日本語説明 |
|---|---|---|
| Topic | 主题 | トピック |
| Subscription | 订阅 | サブスクリプション |
| Publisher | 发布者 | 発行元 |
| Endpoint | 通知接收端 | 通知の受信先 |

## 3. 学习重点 / 学習ポイント

- 中文：理解一个 Topic 对多个订阅端的广播模型。
- 日本語：1 つの Topic から複数の購読先へ配信するモデルを理解する。
- 中文：理解 SNS 和 SQS 的组合方式。
- 日本語：SNS と SQS の組み合わせを理解する。
- 中文：理解 SNS 适合“通知”，SQS 适合“排队处理”。
- 日本語：SNS は「通知」、SQS は「順番待ち処理」に向いていることを理解する。

## 4. LocalStack 里怎么练 / LocalStack での練習方法

- 中文：先练 Topic 创建、订阅绑定和消息发布。
- 日本語：まず Topic の作成、購読の設定、メッセージ配信を練習する。
- 中文：再把 SNS 和 SQS 或 Lambda 串起来练事件链路。
- 日本語：その後 SNS と SQS / Lambda をつないでイベントの流れを確認する。

## 5. 常见坑 / よくある落とし穴

- 中文：把 SNS 当成任务队列。
- 日本語：SNS をタスクキューとして扱う。
- 中文：忽略订阅端过滤和投递失败。
- 日本語：購読フィルタや配信失敗を見落とす。
