# 静态示例站（VS Code 快速入门）

这是一个最小可运行的静态网站示例，用于在 VS Code 中演示开发流程。

快速开始：

1. 在 VS Code 中打开此文件夹（`File → Open Folder`），选择 `static-site-example`。
2. 推荐安装扩展：`Live Server`, `Prettier`, `Path Intellisense`, `Auto Close Tag`。
3. 右键 `index.html` → `Open with Live Server`（或使用你喜欢的静态服务器）即可预览。

说明：该示例使用 Tailwind CDN 做样式演示，不需要 Node 环境。
