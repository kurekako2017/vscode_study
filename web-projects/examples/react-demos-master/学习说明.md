# React Demos 学习说明

这个目录是一组很适合入门 React 的小例子。它的好处是把每个知识点拆得很细，适合按顺序边看边改，而不是一次性把全部源码看完。

## 这套示例能学到什么

- `demo01` 到 `demo03`: JSX 的基本写法、在 JSX 中嵌入 JavaScript、数组渲染
- `demo04` 到 `demo05`: 组件定义、`props`、`children`
- `demo06`: `PropTypes` 类型检查，`defaultProps` 在说明文字里有补充
- `demo07`: `ref` 和直接操作 DOM
- `demo08` 到 `demo10`: `state`、受控表单、生命周期
- `demo11` 到 `demo12`: Ajax、Promise、异步数据渲染
- `demo13`: 服务端渲染、首屏 HTML、客户端接管

## 这套示例没有系统覆盖的内容

这份仓库重点是 React 基础入门，不是完整的现代 React 教程，所以这些主题没有展开：

- 函数组件与 Hooks
- `useState`、`useEffect`、`useMemo`、`useCallback`、`useRef`
- Context
- Portals
- Error Boundary
- Suspense 和并发特性
- `React.memo`、`forwardRef`
- 自定义 Hook
- 状态管理方案，比如 Redux、Zustand、MobX
- 路由方案，比如 React Router
- TypeScript + React
- 单元测试、组件测试、E2E 测试
- 现代工程化工具链，比如 Vite、Next.js、Remix
- `defaultProps` 没有单独做成一个演示文件，只在说明文字里补充了概念

## 怎么学更合适

1. 先按 `demo01` 到 `demo13` 的顺序看一遍。
2. 每个 demo 只改一处，比如字符串、数组、状态值、事件处理函数。
3. 打开浏览器控制台，观察 `PropTypes`、生命周期和异步请求的效果。
4. 看完这套以后，再把它当成“React 早期知识地图”，继续补现代 React。

## 版本提醒

这套示例使用的是 React 16.x 时代的写法。很多概念现在仍然成立，但一部分实践已经被更现代的 API 替代。学习时建议把“原理”和“现行最佳实践”分开看。
