# Next.js Hello

这是一个更完整的 Next.js 示例，包含页面、全局样式与基本配置文件。

## 运行方式

```bash
npm install
npm run dev
```

## 可用脚本

- `npm run dev`：启动开发服务器
- `npm run build`：构建生产产物
- `npm run start`：启动生产服务器

## 学习重点

- `pages/index.js`：页面入口
- `pages/_app.js`：全局样式和应用包装
- `styles/globals.css`：全局样式
- `next.config.js`：Next.js 配置
