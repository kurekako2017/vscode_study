# React Hello

这是一个完整但极简的 React + Vite 示例，包含入口文件、组件、样式和项目配置。

## 运行方式

```bash
npm install
npm run dev
```

## 可用脚本

- `npm run dev`：启动开发服务器
- `npm run build`：构建生产产物
- `npm run preview`：预览构建结果

## 学习重点

- `src/main.jsx`：应用入口与挂载逻辑
- `src/App.jsx`：根组件
- `src/style.css`：基础样式
- `vite.config.js`：Vite 配置
