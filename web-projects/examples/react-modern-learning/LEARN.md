# React Modern Learning 说明

这个项目的目标不是“一次性做成完整业务应用”，而是把现代 React 常见主题拆开，形成可以单独读、单独改、单独跑的章节。

## 为什么这么拆

- `Hooks` 是现代 React 的入口
- `Router` 负责页面组织
- `Context` 负责跨层共享
- `API` 负责异步数据流
- `Test` 负责验证组件行为

如果把这些内容全部塞进一个 `App.jsx`，读起来会很快失焦。拆章节后，每次只处理一个主题，学习成本低很多。

## 结构约定

- 每个章节一个目录
- 每个章节尽量有一个可运行的主示例
- 必要时再配一个测试文件或说明文件
- 首页只做导航，不承载复杂逻辑

## 这一版还没有纳入的内容

- Next.js / Remix
- 更完整的状态管理方案
- 真实后端接口封装层
- E2E 测试框架
- TypeScript 版本

这些都可以在后续继续加，但建议先把章节骨架跑顺。
