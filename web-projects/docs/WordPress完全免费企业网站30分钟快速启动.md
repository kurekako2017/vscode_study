## 📝 Step 3: 联系表单创建 (5分钟)

### 3-1: 创建表单

```
Contact Form 7 → 添加新表单
标题: 联系表单
```

### 3-2: 复制粘贴表单代码

```html
<label> 姓名 (必填)
    [text* your-name autocomplete:name placeholder "山田太郎"] </label>

<label> 公司名称
    [text your-company autocomplete:organization placeholder "株式会社Sample"] </label>

<label> 邮箱地址 (必填)
    [email* your-email autocomplete:email placeholder "example@company.co.jp"] </label>

<label> 电话号码
    [tel your-phone autocomplete:tel placeholder "03-1234-5678"] </label>

<label> 咨询内容 (必填)
    [textarea* your-message placeholder "请输入您的咨询内容"] </label>

<label>
    [acceptance acceptance-privacy] 同意<a href="/privacy-policy" target="_blank">隐私政策</a>
</label>

[submit "提交"]
```

### 3-3: 邮件设置

```
点击"邮件"标签

收件人: info@yourcompany.co.jp  ← 改为您的公司邮箱
发件人: wordpress@yourcompany.co.jp
題名: [お問い合わせ] [your-name]様より

メッセージ本文:
差出人: [your-name] 様
会社名: [your-company]
メールアドレス: [your-email]
電話番号: [your-phone]

お問い合わせ内容:
[your-message]
```

### 3-4: 自動返信メール設定 (オプション)

```
「邮件 (2) を使用」にチェック

送信先: [your-email]
送信元: info@yourcompany.co.jp
題名: お問い合わせを受け付けました

メッセージ本文:
[your-name] 様

この度はお問い合わせいただき、誠にありがとうございます。
以下の内容で受け付けました。

-----------------------------------------
【お問い合わせ内容】
[your-message]
-----------------------------------------

担当者より2営業日以内にご連絡いたします。

株式会社〇〇
担当:〇〇
Email: info@yourcompany.co.jp
```

### 3-5: 保存してショートコードコピー

```
保存 → ショートコードをコピー
例: [contact-form-7 id="123" title="お問い合わせフォーム"]
```

## 📧 Step 4: メール送信設定 (5分)


### 4-1: 基本設定

```

送信者名: 株式会社〇〇

```

### 4-2: SMTP設定 (onamae.com)

```
SMTPホスト: mail.onamae.com
暗号化: SSL

自動TLS: OFF

認証: ON
SMTPユーザー名: info@yourcompany.co.jp
SMTPパスワード: [メールパスワード入力]
```

### 4-3: テスト送信


## 🆘 トラブルシューティング

### メールが届かない

```
【確認1】迷惑メールフォルダ
→ 迷惑メール設定を確認

【確認2】SMTP設定
→ WP Mail SMTP → Email Test で再テスト
→ ポート 587 / TLS も試す

【確認3】onamae.com メールサーバー
→ onamae.com管理画面でメール設定確認
→ メールアカウントが有効か確認

【最終手段】
Contact Form 7 → インテグレーション
→ Sendinblue 連携 (無料: 300通/日)
```

### フォームが表示されない

```
【確認1】ショートコード
→ 正しくコピーできているか確認
→ [ ] の記号が全角になっていないか

【確認2】Contact Form 7 有効化
→ プラグイン → インストール済みで確認

【確認3】キャッシュクリア
→ ブラウザのキャッシュをクリア
→ Ctrl+Shift+Delete
```

### ページが遅い

```
【対策1】画像最適化
→ EWWW Image Optimizer が有効化されているか確認
→ メディア → 一括最適化

【対策2】不要プラグイン削除
→ 使っていないプラグインを無効化・削除

【対策3】キャッシュプラグイン
→ WP Super Cache インストール
→ 設定 → キャッシング ON
```
