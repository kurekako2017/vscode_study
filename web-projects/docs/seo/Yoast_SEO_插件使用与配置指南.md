# Yoast SEO 插件使用与配置指南

## 一、Yoast SEO 简介
Yoast SEO 是 WordPress 中最流行的 SEO 插件之一，主要用于：
- 优化页面与文章的搜索引擎可读性
- 管理站点 SEO 基础设置
- 生成 XML 站点地图
- 控制面包屑、Schema 结构化数据等

适合对象：
- 内容型网站 / 博客
- 企业官网
- WooCommerce 商城

---

## 二、安装与启用

1. 进入 **WordPress 后台 → 插件 → 安装插件**
2. 搜索 **Yoast SEO**
3. 点击 **安装** → **启用**

建议版本：官方最新版（免费版已足够 80% 使用场景）

---

## 三、首次配置向导（必做）

路径：**SEO → 常规 → 首次配置**

### 1. 站点类型
- 博客 / 企业网站 / 电商 / 作品集
- 根据实际情况选择

### 2. 组织或个人
- 企业：填写公司名称、Logo
- 个人：填写姓名

### 3. 搜索引擎可见性
- 允许搜索引擎索引（通常选择“是”）

### 4. 社交媒体
- 填写 Facebook、Twitter、微博等（可选）

### 5. 完成
配置完成后可在后续随时修改

---

## 四、核心功能配置详解

### 4.1 常规设置（SEO → 常规）

#### 仪表盘
- SEO 问题提示（红 / 橙 / 绿）
- 建议全部修复为绿色

#### 功能
建议开启：
- SEO 分析
- 可读性分析
- 基石内容
- Schema 结构化数据

---

### 4.2 搜索外观（SEO → 搜索外观）

#### ① 常规
- 分隔符：`-` 或 `|`
- 首页标题模板示例：
  ```
  %%sitename%% - %%sitedesc%%
  ```

#### ② 内容类型（重点）

**文章（Posts）**
- 在搜索结果中显示：是
- 标题模板：
  ```
  %%title%% %%sep%% %%sitename%%
  ```

**页面（Pages）**
- 同文章设置

**媒体**
- 建议：重定向附件 URL 到媒体文件（是）

#### ③ 分类法（Categories / Tags）
- 分类：显示（是）
- 标签：一般建议 **不显示**（避免重复内容）

#### ④ 面包屑
- 若主题支持，建议开启
- 可用于 SEO + 用户体验

---

### 4.3 站点地图（SEO → 常规 → 功能）

- 启用 XML 站点地图
- 访问地址：
  ```
  https://你的域名/sitemap_index.xml
  ```

建议提交到：
- Google Search Console
- Bing Webmaster Tools

---

## 五、文章 / 页面 SEO 实战设置（最重要）

每篇文章编辑页底部都会看到 **Yoast SEO 面板**

### 5.1 焦点关键词（Focus Keyphrase）
- 每篇文章 1 个核心关键词
- 示例：
  ```
  WordPress SEO 教程
  ```

### 5.2 SEO 标题（Title）
建议结构：
```
核心关键词 + 修饰词 | 品牌名
```

示例：
```
WordPress SEO 教程：新手完整优化指南 | XXX博客
```

### 5.3 元描述（Meta Description）
- 120–156 字
- 包含关键词
- 有行动引导

示例：
```
本教程详细讲解 WordPress SEO 优化方法，适合新手快速提升搜索排名。
```

### 5.4 可读性分析
- 多用短句
- 使用小标题（H2/H3）
- 避免大段文字

---

## 六、基石内容（Cornerstone Content）

适合：
- 核心教程
- 流量主入口文章

用法：
- 编辑文章 → 勾选“基石内容”
- Yoast 会提高 SEO 要求标准

建议数量：5–20 篇

---

## 七、高级设置（慎用）

路径：**SEO → 工具 / 高级**

### 7.1 noindex 设置
可用于：
- 登录页
- 感谢页
- 筛选结果页

### 7.2 面包屑 Schema
- 自动生成结构化数据
- 有助于富摘要展示

---

## 八、Yoast SEO + WooCommerce（如适用）

建议安装：
- Yoast WooCommerce SEO（付费）

可优化：
- 产品 Schema
- 分类 SEO
- 面包屑结构

---

## 九、常见错误与优化建议

❌ 每篇文章都用相同关键词  
❌ Meta Description 留空  
❌ 标签页被索引过多  
❌ 只看绿灯，不看内容质量  

✅ SEO = 技术 + 内容 + 用户体验

---

## 十、推荐搭配工具

- Google Search Console
- Google Analytics / GA4
- Rank Math（可替代方案）
- Ahrefs / Semrush

---

## 十一、结语

Yoast SEO 是**基础必装插件**，但 SEO 成效取决于：
- 内容质量
- 站内结构
- 外链与时间积累

> 插件只是工具，排名靠的是长期优化。
