## 🎨 Kadence 主题详解

### Kadence 免费版功能

Kadence Theme 免费版已经包含了大量企业网站所需的功能：

#### 核心功能
- ✅ 响应式设计（移动端完美适配）
- ✅ 内置页面构建器（Kadence Blocks）
- ✅ 丰富的自定义选项
- ✅ 多布局选项（全宽、盒装等）
- ✅ 内置 SEO 优化
- ✅ 高性能（轻量级代码）
- ✅ WooCommerce 集成
- ✅ 多语言支持

#### Kadence Blocks 内置组件
- 按钮（多种样式）
- 图标
- 分隔线
- 文本块
- 图片
- 图标列表
- 表单集成
- 布局容器

### Kadence Blocks Pro 高级功能（付费版）

如果免费版无法满足需求，Kadence Blocks Pro 提供：

- 高级标题设计
- 图标框
- 轮播图
- 手风琴（Accordion）
- 标签页（Tabs）
- 表单样式增强
- 条件显示
- 动画效果
- 更多布局选项

### 安装 Kadence 主题

#### 方法一：WordPress 管理后台安装（推荐）

```
1. 登录 WordPress 管理后台
2. 外观 → 主题 → 添加新主题
3. 搜索: "Kadence"
4. 点击"安装" → "启用"
```

#### 方法二：官方网站下载

```
1. 访问: https://www.kadencewp.com/kadence-theme/
2. 下载主题文件
3. 外观 → 主题 → 添加新主题 → 上传主题
4. 上传 zip 文件 → 安装 → 启用
```

### Kadence 基本配置

#### 1. 站点身份设置

```
外观 → 自定义 → 站点身份
- 上传 Logo
- 设置网站标题和副标题
- 设置网站图标（Favicon）
```

#### 2. 布局设置

```
外观 → 自定义 → 布局
- 选择容器宽度（默认 1200px）
- 设置内容宽度
- 配置侧边栏布局
```

#### 3. 颜色设置

```
外观 → 自定义 → 颜色
- 设置主色调
- 设置链接颜色
- 配置标题颜色
```

#### 4. 标题设置

```
外观 → 自定义 → 标题
- 选择标题布局（中心、左侧、右侧）
- 配置导航菜单
- 设置粘性标题
```

### 使用 Kadence Blocks 创建页面

1. **创建新页面**
   ```
   页面 → 添加新页面
   标题: 关于我们（示例）
   ```

2. **使用 Kadence Blocks**
   ```
   在编辑器中点击 "+" 按钮
   搜索 "Kadence" 查看可用块
   选择需要的块（如按钮、图标、分隔线等）
   ```

3. **常用块示例**
   - **按钮块**: 添加 CTA 按钮
   - **图标块**: 展示服务图标
   - **分隔线**: 分隔内容区域
   - **文本块**: 自定义样式文本

## 🛠️ Elementor 页面构建器详解

### Elementor 免费版功能

Elementor 免费版提供了强大的可视化页面构建功能：

- ✅ 拖拽式页面构建
- ✅ 响应式编辑
- ✅ 丰富的 Widget（小部件）
- ✅ 模板库（部分免费）
- ✅ 实时预览
- ✅ 主题兼容性好

### Elementor Pro 高级功能（付费版）

- 主题构建器
- 全局 Widget
- 表单构建器
- 弹出窗口构建器
- WooCommerce Widget
- 动画效果
- 更多模板

### 安装 Elementor

#### 安装 Elementor 插件

```
1. 插件 → 添加新插件
2. 搜索: "Elementor"
3. 点击"立即安装" → "启用"
```

#### 安装 Hello Elementor 主题（可选）

```
1. 外观 → 主题 → 添加新主题
2. 搜索: "Hello Elementor"
3. 点击"安装" → "启用"
```

**注意**: Hello Elementor 是 Elementor 官方提供的免费主题，专为 Elementor 优化，轻量级且快速。

### 使用 Elementor 创建页面

#### 1. 使用 Elementor 编辑器

```
页面 → 添加新页面
标题: 首页（示例）
点击"使用 Elementor 编辑"
```

#### 2. Elementor 界面介绍

- **左侧面板**: Widget 库、设置选项
- **中间区域**: 页面编辑区域（实时预览）
- **右侧面板**: 页面设置、响应式预览

#### 3. 常用 Widget（免费版）

**基础 Widget**:
- 标题（Heading）
- 文本编辑器（Text Editor）
- 图片（Image）
- 按钮（Button）
- 图标（Icon）
- 分隔线（Divider）
- 间距（Spacer）

**内容 Widget**:
- 图片框（Image Box）
- 图标框（Icon Box）
- 图像轮播（Image Carousel）
- 文本路径（Text Path）

**表单 Widget**:
- 表单（Form）- 需要 Elementor Pro
- 联系表单 7（Contact Form 7）- 需要插件

**布局 Widget**:
- 内部部分（Inner Section）
- 列（Columns）

#### 4. 创建首页示例

```
1. 添加部分（Section）
2. 选择列布局（如 3 列）
3. 在第一列添加"图标框" Widget
   - 选择图标
   - 输入标题和描述
4. 在第二列添加"图标框" Widget
5. 在第三列添加"图标框" Widget
6. 添加"按钮" Widget（全宽）
7. 点击"发布"
```

### Elementor 模板库

#### 使用免费模板

```
1. 在 Elementor 编辑器中
2. 点击左侧的文件夹图标（模板库）
3. 浏览免费模板
4. 点击模板 → "插入"
5. 自定义内容
```

#### 模板分类

- **页面模板**: 完整的页面设计
- **块模板**: 特定区域的模板（如页眉、页脚）
- **弹出窗口**: 需要 Elementor Pro
