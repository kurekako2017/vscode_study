# 邮件服务快速部署指南

## 🚀 5分钟快速上手

### 步骤 1：创建项目

```bash
mkdir email-service
cd email-service
npm init -y
npm install express nodemailer dotenv body-parser cors
```

### 步骤 2：创建 .env 配置

```env
# Gmail 配置示例
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password

# 接收邮件
ADMIN_EMAIL=admin@company.com

# 服务器
PORT=3000
```

### 步骤 3：创建 server.js

```javascript
const express = require('express');
const nodemailer = require('nodemailer');
const bodyParser = require('body-parser');
const cors = require('cors');
require('dotenv').config();

const app = express();
app.use(cors());
app.use(bodyParser.json());
app.use(express.static('public'));

// 邮件配置
const transporter = nodemailer.createTransport({
    host: process.env.SMTP_HOST,
    port: process.env.SMTP_PORT,
    secure: false,
    auth: {
        user: process.env.SMTP_USER,
        pass: process.env.SMTP_PASS
    }
});

// 联系表单接口
app.post('/api/contact', async (req, res) => {
    try {
        const { name, email, message } = req.body;

        // 发送给管理员
        await transporter.sendMail({
            from: process.env.SMTP_USER,
            to: process.env.ADMIN_EMAIL,
            replyTo: email,
            subject: `网站留言 - ${name}`,
            html: `
                <h2>新的网站留言</h2>
                <p><strong>姓名：</strong>${name}</p>
                <p><strong>邮箱：</strong>${email}</p>
                <p><strong>留言：</strong></p>
                <p>${message}</p>
            `
        });

        // 自动回复用户
        await transporter.sendMail({
            from: process.env.SMTP_USER,
            to: email,
            subject: '感谢您的留言',
            html: `
                <h2>感谢您的留言，${name}！</h2>
                <p>我们已经收到您的消息，会在24小时内回复。</p>
            `
        });

        res.json({ success: true, message: '发送成功' });
    } catch (error) {
        console.error(error);
        res.status(500).json({ success: false, message: '发送失败' });
    }
});

app.listen(process.env.PORT, () => {
    console.log(`✅ 服务器运行在 http://localhost:${process.env.PORT}`);
});
```

## 📧 Gmail 应用密码设置

1. 访问 https://myaccount.google.com/security
2. 启用"两步验证"
3. 搜索"应用专用密码"
4. 生成新密码用于 SMTP
5. 将密码填入 `.env` 的 `SMTP_PASS`

## 🔧 常见问题

### Q1: 发送失败
- 检查 Gmail 是否启用了"允许不够安全的应用"
- 使用应用专用密码，不是 Gmail 登录密码
- 检查网络是否能访问 Gmail SMTP 服务器

### Q2: 端口被占用
```bash
# 更改 .env 中的 PORT
PORT=8080
```

### Q3: 测试邮件配置
```bash
# 添加测试路由
app.get('/test-email', async (req, res) => {
    try {
        await transporter.verify();
        res.send('✅ 邮件配置正确');
    } catch (error) {
        res.send('❌ 邮件配置错误: ' + error.message);
    }
});
```

访问：http://localhost:3000/test-email
