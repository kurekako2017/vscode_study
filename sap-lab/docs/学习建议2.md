# 在 GitHub Codespaces 学习现代 SAP 开发 - 学生版优化路径（2025年底）

**特别适合有 GitHub Student Developer Pack 的同学！**  
你将获得**每月 180 core hours**（核心小时） + **20 GB Codespaces 存储**（相当于 Pro 级别），完全够用来练习 RAP、CAP、CDS 等现代 ABAP Cloud 开发，几乎不会超出免费额度。

## 前置准备（10–30分钟）
1. 确保已激活 GitHub Student Developer Pack（https://education.github.com/pack）  
   → 已自动获得 GitHub Pro + Codespaces 每月180核心小时 + Copilot Pro
2. 注册免费的 **SAP BTP Trial**（强烈推荐！90天有效）  
   https://account.hanatrial.ondemand.com  
   → 试用期内可创建 ABAP Environment（注意：90天后会自动终止，记得提前备份）
3. （可选但推荐）在 SAP BTP Trial 中启用 **ABAP Environment** 服务  
   → 搜索 "Prepare an Account for ABAP Trial" Booster，一键创建（有效期同样90天）

## 阶段1 - 基础准备与环境（1–3天）
1. 创建一个新的 GitHub 仓库（建议名字：`sap-rap-student-learning`）
2. 在仓库中添加 `.devcontainer/devcontainer.json` 文件  
   （推荐使用 SAP 官方或社区的 ABAP Cloud 模板，包含 ADT/VS Code 扩展）
3. 点击仓库的绿色 **Code** 按钮 → **Open with Codespaces** → **New codespace**  
   → 学生权益下可选择较高配置机器（4核/8核/16核，根据剩余 core hours 灵活选）
4. 安装关键 VS Code 扩展（Codespaces 会自动建议）：
   - ABAP Language Support
   - ABAP CDS
   - abaplint
   - SAP Fiori Tools / UI5（可选，前端开发用）

**小贴士**：每月180 core hours ≈ 每天可用6小时（2核机器）或3小时（4核机器），学习时可随时停止 codespace 节省额度。

## 阶段2 - 核心学习内容（最推荐的顺序，Codespaces 友好）

| 周次     | 学习主题                          | 主要资源（developers.sap.com）                                      | 实践方式（在 Codespaces 中）                  | 预计 core hours 消耗 |
|----------|-----------------------------------|---------------------------------------------------------------------|-----------------------------------------------|----------------------|
| 第1周    | ABAP Cloud 语法 & Clean Core 原则 | https://developers.sap.com/topics/abap-cloud.html                  | 写纯 ABAP 代码 + 用 abaplint 检查             | 很低                |
| 第2周    | Core Data Services (CDS) 基础     | https://developers.sap.com/tutorials/abap-environment-cds-view.html| 写 CDS 视图、注解、消费视图                   | 低                  |
| 第2–3周  | RAP 入门 - Managed Scenario       | https://developers.sap.com/mission.sap-fiori-abap-rap100.html      | 生成 RAP BO → 用 Mock Server 本地测试         | 中等                |
| 第3–4周  | RAP + Fiori Elements UI           | RAP100 后续教程                                                    | 跑 Fiori preview（浏览器内预览）              | 中等                |
| 第4–5周  | RAP - Unmanaged + 自定义逻辑      | RAP200 / RAP300 系列                                               | 写 determination / validation / action        | 中等–较高           |
| 第5–6周  | RAP 消费外部服务（HTTP/BTP 服务） | https://developers.sap.com/tutorials/abap-environment-outbound-http.html | 练习 outbound HTTP client                     | 较高（网络调用）    |
| 第6+周   | CAP 模型快速体验（可选）          | https://developers.sap.com/topics/cloud-application-programming.html| 在 Codespaces 跑 Node.js / Java CAP 项目      | 中等                |

## 阶段3 - 进阶挑战项目（做出能展示的成果！）
1. 做一个小型「**物料主数据管理**」应用（Material + Supplier + 简单审批）
2. 包含：创建/修改/删除 + 业务校验 + Fiori List Report + Object Page
3. 在 Codespaces 中开发 + 测试（Mock 数据）
4. **有 BTP Trial 时** → 连接真实 ABAP Environment 部署运行（90天内完成最理想）
5. 把项目推到 GitHub + 写 README（附上截图/视频），这就是很好的简历亮点！

- **Codespaces 学生权益**：每月180 core hours + 20GB存储，**非常够用**，但记得及时停止不用的 codespace（右上角管理）
- **SAP BTP ABAP Environment Trial**：只有**90天**，到期后实例会终止（可备份代码到 GitHub）
- **学不到的技术**（Codespaces + Trial 基本无法练习）：

  5. S/4HANA 核心配置（IMG/SPRO）
  6. Basis 运维、系统监控、内核升级
  7. 老 Gateway OData V2 开发
