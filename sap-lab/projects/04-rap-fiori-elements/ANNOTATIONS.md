## Fiori Elements 重要注解

### UI 注解
```cds
@UI.facet - 定义页面分区
@UI.lineItem - 列表显示
@UI.identification - 对象页显示
@UI.selectionField - 搜索/过滤字段
@UI.dataPoint - KPI 显示
@UI.chart - 图表配置
```

### Presentation 注解
```cds
@UI.presentationVariant - 排序、分组
@UI.dataFieldForAction - 操作按钮
@UI.dataFieldForIntentBasedNavigation - 导航
```

### Search 注解
```cds
@Search.searchable: true
@Search.defaultSearchElement: true
@Search.fuzzinessThreshold: 0.8
```

### 字段级注解
```cds
@UI.hidden: true
@UI.readOnly: true
@UI.multiLineText: true
```
