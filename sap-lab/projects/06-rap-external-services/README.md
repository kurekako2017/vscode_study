# RAP 消费外部服务 (HTTP/BTP 服务)

## 学习目标
- 学习如何从 RAP 调用外部 HTTP 服务
- 理解 BTP（Business Technology Platform）服务集成
- 处理异步通信和错误处理
- 实现服务认证和授权

## 学习资源
- 官方教程：https://developers.sap.com/tutorials/abap-environment-outbound-http-client.html

## 核心内容
1. HTTP 客户端基础
2. REST API 调用
3. 请求和响应处理
4. 错误处理和日志
5. 服务认证（OAuth2、Basic Auth）
6. BTP 服务接着目录

## 预计学习时间
较高 (约 8-10 core hours)
