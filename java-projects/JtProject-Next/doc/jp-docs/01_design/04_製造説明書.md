# 製造説明書

## 1. 目的

本書は `JtProject-Next` の実装内容、主要改修箇所、製造時の確認ポイントを整理するための文書である。

## 2. 製造対象

| No | 対象 | ファイル |
|---|---|---|
| 1 | ユーザーログイン制御 | `src/main/java/com/jtspringproject/JtSpringProject/controller/UserController.java` |
| 2 | 管理者制御 | `src/main/java/com/jtspringproject/JtSpringProject/controller/AdminController.java` |
| 3 | ユーザーログイン画面 | `src/main/webapp/views/userLogin.jsp` |
| 4 | 商品トップ画面 | `src/main/webapp/views/index.jsp` |

| 7 | 商品更新画面 | `src/main/webapp/views/productsUpdate.jsp` |

## 3. 実装方針

- Controller でリクエストを受ける
- Service へ業務処理を委譲する
- DAO で DB アクセスを行う
- JSP で画面を表示する

## 4. 実装上のポイント

### 4.1 ユーザーログイン

- `userloginvalidate` と `userloginvalidate1` の両方に対応
- `null` ガードを入れた実装
- ログイン成功時に Cookie と Session に値を保存

### 4.2 カート機能

- 未ログイン時のガードあり
- 管理者はカート利用不可
- カート未作成時は新規作成

### 4.3 管理者商品更新

- 画像が未入力でも既存画像を保持する実装
- 更新後は一覧画面へリダイレクト

## 5. 製造時チェック観点

- URL と JSP form action の整合
- `@RequestParam` 名と画面項目名の整合
- Session キー名の統一
- リダイレクト先の妥当性
- 空データ、未ログイン、権限違いの扱い

## 6. 今後の改善候補

- 例外処理とログ出力の統一
- 画面メッセージの言語統一
- バリデーション強化
- 管理者系の認可共通化
