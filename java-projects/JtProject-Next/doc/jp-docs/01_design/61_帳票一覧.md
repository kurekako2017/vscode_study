# 61. 帳票一覧

## 1. 文書情報

| 項目 | 内容 |
|---|---|
| 文書名 | 帳票一覧 |
| 対象システム | JtProject |
| 文書区分 | 画面・出力設計補足 |
| 版数 | v1.0 |

## 2. 目的

本書は、`JtProject` における帳票、一覧出力、画面上の参照系出力を整理し、出力対象、利用者、出力タイミングを明確化することを目的とする。

## 3. 前提

- 現行システムには PDF や CSV の専用帳票出力機能は実装されていない。
- ただし、日本プロジェクトの資料では、画面一覧表や管理一覧画面も帳票に準じて整理することがあるため、本書では画面出力を含めて整理する。

## 3.1 現行の帳票関連技術

| 項目 | 現行実装 |
|---|---|
| 画面出力方式 | `Spring Boot` + `JSP` + `JSTL` |
| テンプレート実行基盤 | `tomcat-embed-jasper` |
| データ取得方式 | `Controller` + `Service` + `DAO` + `JPA/Hibernate` |
| PDF 帳票ツール | 未導入 |
| 専用帳票エンジン | 未導入 |
| CSV 出力機能 | 未実装 |

補足:

- `pom.xml` に含まれる `tomcat-embed-jasper` は JSP 描画用であり、`JasperReports` のような PDF 帳票エンジンではない。
- 現在の帳票相当機能は、JSP 画面として一覧を表示する方式で実装されている。
- 実案件で PDF 帳票を扱う場合は、`SVF`、`JasperReports`、`ActiveReports`、`iText` などの導入を別途検討する。

## 4. 帳票一覧

| 帳票ID | 帳票名 | 区分 | 利用者 | 出力内容 | 出力契機 | 備考 |
|---|---|---|---|---|---|---|
| RPT-01 | 商品一覧表示 | 画面一覧 | 一般利用者 | 商品名、価格、分類、詳細導線 | `/index` 表示時 | 画面帳票扱い |
| RPT-02 | カート一覧表示 | 画面一覧 | 一般利用者 | カート内商品、数量、価格情報 | `/user/cart` 表示時 | 明細性あり |
| RPT-03 | 管理者商品一覧 | 管理一覧 | 管理者 | 商品 ID、名称、価格、分類、更新対象 | `/admin/products` 表示時 | 管理用主要帳票 |
| RPT-04 | 管理者分類一覧 | 管理一覧 | 管理者 | 分類 ID、分類名 | `/admin/categories` 表示時 | 主数据确认用 |
| RPT-05 | 顧客一覧表示 | 管理一覧 | 管理者 | 顧客 ID、名称、連絡先等 | 顧客管理画面表示時 | 個人情報取扱注意 |
| RPT-06 | 試験結果報告書 | 文書帳票 | 開発者、レビュー担当 | 試験区分、結果、残課題 | 試験完了時 | 文書納品対象 |
| RPT-07 | 障害一覧 | 管理台帳 | 開発者、運用担当 | 障害番号、状態、優先度、影響範囲 | 障害管理時 | 台帳形式 |

## 5. 帳票設計観点

- 利用者に必要な項目だけを表示すること。
- 管理者向け一覧には識別用キー項目を含めること。
- 個人情報は過剰表示しないこと。
- 将来 CSV/PDF 出力を追加する場合、本一覧を起点に帳票 ID を採番すること。
- PDF 帳票を追加する場合は、出力レイアウト、改ページ、フォント、文字コード、印刷要件を別途定義すること。
