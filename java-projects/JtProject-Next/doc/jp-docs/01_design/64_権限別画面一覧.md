# 64. 権限別画面一覧

## 1. 文書情報

| 項目 | 内容 |
|---|---|
| 文書名 | 権限別画面一覧 |
| 対象システム | JtProject |
| 文書区分 | 権限・画面設計資料 |
| 版数 | v1.0 |

## 2. 目的

本書は、利用者権限ごとにアクセス可能な画面を整理し、画面制御と権限設計の整合を明確化することを目的とする。

## 3. 権限区分

| 権限 | 内容 |
|---|---|
| `ROLE_NORMAL` | 一般利用者 |
| `ROLE_ADMIN` | 管理者 |
| `未ログイン` | 認証前利用者 |

## 4. 権限別画面一覧

| 画面ID | 画面名 | URL | 未ログイン | `ROLE_NORMAL` | `ROLE_ADMIN` | 備考 |
|---|---|---|---|---|---|---|
| SCR-01 | 一般ログイン画面 | `/` | 可 | 可 | 可 | 入口画面 |
| SCR-02 | ユーザ登録画面 | `/register` | 可 | 可 | 可 | 新規登録用 |
| SCR-03 | 商品一覧画面 | `/index` | 不可/要方針確認 | 可 | 可 | 現状はログイン後利用前提 |
| SCR-04 | カート画面 | `/user/cart` | 不可 | 可 | 不可 | 一般利用者専用 |
| SCR-05 | 管理者ログイン画面 | `/admin/login` | 可 | 可 | 可 | 管理入口 |
| SCR-06 | 商品管理画面 | `/admin/products` | 不可 | 不可 | 可 | 管理者専用 |
| SCR-07 | 分類管理画面 | `/admin/categories` | 不可 | 不可 | 可 | 管理者専用 |
| SCR-08 | 顧客一覧画面 | `/admin/customers` 想定 | 不可 | 不可 | 可 | 実装状況に応じて調整 |
| SCR-09 | 管理者プロフィール画面 | `/admin/profile` 想定 | 不可 | 不可 | 可 | 実装状況に応じて調整 |

## 5. 権限制御観点

- 未ログイン利用者が管理画面へ直接アクセスできないこと。
- `ROLE_NORMAL` 利用者が管理機能を実行できないこと。
- 管理者は必要な管理画面にアクセス可能であること。
- URL 直打ち時の制御は Controller 判定だけでなく、将来は Spring Security 等で補強すること。

## 6. 運用上の留意点

- 権限変更時は `権限一覧`、`URL一覧`、`権限申請・アカウント管理表` をあわせて更新すること。
- 画面追加時は本一覧に画面 ID 単位で追記すること。
