# 画面一覧

## 1. 目的

本書は `JtProject-Next` の画面一覧を整理し、画面 ID、JSP、画面用途、主な入力項目、遷移先、利用者を明示する。
あわせて、各画面がどのレイアウト定義、画面項目定義に対応するかをナビゲーションできるようにする。

## 4. 代表的な画面遷移

### ユーザー導線

```text
SCR-01 ユーザーログイン
  -> SCR-03 商品トップ
  -> SCR-04 商品一覧
  -> SCR-05 カート
```

### 管理者導線

```text
SCR-06 管理者ログイン
  -> SCR-07 管理者ホーム
  -> SCR-08 カテゴリ管理
  -> SCR-09 商品管理一覧
  -> SCR-10 商品追加 / SCR-11 商品更新
  -> SCR-12 顧客一覧
  -> SCR-13 プロフィール更新
```

## 6. 実装との差分メモ

- `buy.jsp` は現時点で存在しないため、`/buy` ルートは画面一覧に含めていない。
- 管理画面・利用者画面ともに共通ヘッダは JSP include 化されておらず、各画面にナビゲーションが個別記述されている。
- `cartproduct.jsp` は主要業務画面ではなく、補助的な遷移案内ページとして扱う。
