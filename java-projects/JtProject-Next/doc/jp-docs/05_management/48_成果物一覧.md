## 2. 目的

本書は、`JtProject-Next` の日本案件向け成果物を一覧化し、成果物区分、用途、状態、主な参照先を管理するための台帳である。

## 3. 管理方針

- 設計、試験、DB、管理台帳、運用資料を成果物区分ごとに整理する。
- 成果物の有無、整備状況、承認対象かどうかを把握できるようにする。
- 追加・改訂時は本書も更新する。

## 5. 状態定義

| 状態 | 内容 |
|---|---|
| 作成済み | 文書が作成され、利用可能 |
| 継続整備中 | 文書は存在するが、内容拡張を継続中 |
| 一部未入力 | ひな形はあるが実績データ未記録 |
| 未作成 | まだ作成していない |

## 6. 利用ルール

- 成果物追加時は本書へ登録する。
- 状態更新時は承認履歴や変更履歴と整合を取る。
- 新規参画者には本書と `47_ドキュメント相関図.md` を優先案内する。

## 7. 備考

- 実案件では納品日、版数、保管場所、提出先、レビュー状態も追加する。
- Excel 化する場合は区分ごとのフィルタ利用を想定する。
