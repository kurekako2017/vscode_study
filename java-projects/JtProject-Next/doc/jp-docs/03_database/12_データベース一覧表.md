## 2. テーブル一覧

| No | テーブル名 | 論理名 | 主用途 | 備考 |
|---|---|---|---|---|
| 1 | `CUSTOMER` | 顧客情報 | ユーザー認証、顧客管理、管理者判定 | `username` は一意 |
| 2 | `CATEGORY` | 商品カテゴリ | 商品分類管理 | 管理者画面で CRUD |
| 3 | `PRODUCT` | 商品情報 | 商品一覧、商品管理、カート追加対象 | カテゴリ FK 保持 |
| 4 | `CART` | カート | 顧客別カート管理 | 顧客 FK 保持 |
| 5 | `CART_PRODUCT` | カート商品明細 | カートと商品の関連管理 | 中間テーブル |

表名と用途の概要：

- `CUSTOMER`: 顧客情報管理、ログイン認証、権限判定に使用
- `CATEGORY`: 商品カテゴリ管理に使用
- `PRODUCT`: 商品情報管理、商品一覧表示、商品管理に使用
- `CART`: 顧客ごとのカートヘッダ管理に使用
- `CART_PRODUCT`: カート内商品明細管理に使用
