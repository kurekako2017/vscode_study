## 2. 目的

本書は、利用者アカウント、ロール、申請状況、発行状況、停止状況を管理するための台帳である。

## 3. 管理方針

- アカウント発行、変更、停止は記録を残す。
- 権限付与理由と承認状況を明確にする。
- 管理者権限は一般ユーザー権限と区別して管理する。

## 4. ロール定義

| ロール | 内容 |
|---|---|
| `ROLE_NORMAL` | 一般ユーザー権限 |
| `ROLE_ADMIN` | 管理者権限 |

## 8. アカウント管理上の注意

- パスワードは現行実装では平文前提であるため、本番用途には適さない。
- 管理者権限付与時は、権限一覧と整合していることを確認する。
- 利用不要となったアカウントは停止管理対象とする。

## 9. 備考
