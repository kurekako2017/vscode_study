# IntelliJ IDEA ↔ Eclipse 常用快捷键功能对照一覧（Windows / Linux）

> 适用于 Java / Spring / 通用开发  
> 重点涵盖：代码导航、呼び出し出处（调用关系）、搜索、重构、编辑

---

## 一、代码导航（Navigation）

| 功能 | Eclipse | IntelliJ IDEA | 说明 |
|---|---|---|---|
| 跳到定义 / 声明 | F3 | Ctrl + B | 方法 / 类 / 变量 |
| 鼠标跳转 | Ctrl + Click | Ctrl + Click | 同上 |
| 跳到实现类 | Ctrl + T | Ctrl + Alt + B | 接口 → 实现 |
| 查看类型层级 | Ctrl + T | Ctrl + H | 继承结构 |
| 返回上一次位置 | Alt + ← | Ctrl + Alt + ← | 光标跳转历史 |
| 前进 | Alt + → | Ctrl + Alt + → | |
