# JtProject-SpringBoot-TypeScript 文档入口

这个目录保存 Spring Boot + 纯 TypeScript 版本的学习文档。

## 推荐阅读顺序

1. [springboot-typescript-flow.md](./springboot-typescript-flow.md)：先理解数据如何从 Java 后端流到 TypeScript 前端
2. [project-code-map.md](./project-code-map.md)：再按文件定位源码
3. [../frontend/README.md](../frontend/README.md)：最后看前端工程入口

## 这个版本重点解决什么

- 在不使用 React、Vue、Next 的情况下理解前后端分离
- 用 TypeScript 类型约束接口返回数据
- 用 `fetch`、状态对象和 DOM API 实现页面交互
- 为学习 React / Vue / Next 之前补上“框架之前”的基础层
