## 2. 目的

本書は、後任担当者、学習者、保守担当者へ `JtProject` の構成、重要文書、既知課題、運用上の注意点を引き継ぐための資料である。

## 3. システム概要

- 種別: Java / Spring Boot 学習用 EC サンプル
- 画面: JSP / JSTL
- 主機能:
  - 一般ユーザーログイン
  - ユーザー登録
  - 商品一覧表示
  - カート追加、表示、削除
  - 管理者ログイン
  - カテゴリ管理
  - 商品管理
  - 顧客一覧
  - プロフィール更新
