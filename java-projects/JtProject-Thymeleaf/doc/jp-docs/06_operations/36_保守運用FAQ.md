## 3. FAQ 一覧

### Q1. プロジェクトの起動方法は？

**A.** `JtProject` 直下で以下を実行する。

```powershell
.\mvnw.cmd spring-boot:run
```

または

```powershell
.\scripts\start\run.ps1
```

詳細は [33_運用手順書.md](../../../../JtProject/doc/jp-docs/06_operations/33_運用手順書.md) を参照。

### Q2. ログインできない場合は何を確認する？

**A.**

- `user01 / pass01` または `admin01 / admin01` の初期データ有無
- DB 初期化状態
- `CUSTOMER` テーブルに対象ユーザーが存在するか
- パスワードが平文比較前提であること

### Q3. 管理者画面に入れない場合は？

**A.**

- ログインユーザーの `role` が `ROLE_ADMIN` か確認する
- Session の `adminLoggedIn`、`adminUsername` が設定されているか確認する
- 直接 URL アクセス時の権限制御結果を確認する

### Q4. 商品が画面に表示されない場合は？

**A.**

- `PRODUCT` テーブルにデータがあるか確認する
- `ProductService.getProducts()` が正常に返却しているか確認する
- トップ画面または商品一覧画面で `msg` にエラーが出ていないか確認する

### Q5. カートに追加できない場合は？

**A.**

- 一般ユーザーでログインしているか
- 管理者アカウントで操作していないか
- 商品 ID が存在するか
- `CART` が作成されているか
- `CART_PRODUCT` に明細が追加されているか

### Q6. 初期データを入れ直したい場合は？

**A.**

- `27_DDL一覧.md` のサンプル DML を参照する
- 必要に応じて既存データをクリアする
- `25_テストデータ一覧.md` の内容と一致するよう投入する
