# 79. 画面テスト観点一覧

## 1. 文書情報

| 項目 | 内容 |
|---|---|
| 文書名 | 画面テスト観点一覧 |
| 対象システム | JtProject |
| 文書区分 | 画面試験補足資料 |
| 版数 | v1.0 |

## 2. 目的

本書は、主要画面に対する試験観点を整理し、単体、結合、総合試験の観点抽出を容易にすることを目的とする。

## 3. 観点一覧

| 画面ID | 画面名 | 正常系観点 | 異常系観点 | 権限観点 |
|---|---|---|---|---|
| SCR-01 | ユーザーログイン画面 | 正常ログインで商品トップへ遷移 | 必須未入力、認証失敗 | 管理者認証との混同なし |
| SCR-02 | ユーザ登録画面 | 正常登録でログイン画面へ戻る | 重複、入力不正 | 未ログインで利用可能 |
| SCR-03 | 商品トップ画面 | 商品一覧表示、カート追加成功 | 商品なし、追加失敗 | 未ログイン利用制御 |
| SCR-05 | カート画面 | 商品表示、削除成功 | 対象なし、削除失敗 | 他ユーザのカート参照不可 |
| SCR-06 | 管理者ログイン画面 | 正常ログインで管理ホームへ遷移 | 認証失敗 | 一般利用者との分離 |
| SCR-08 | 分類管理画面 | 追加、更新、削除成功 | 未入力、重複、削除失敗 | 管理者専用 |
| SCR-09 | 商品管理画面 | 一覧表示、追加/更新導線 | 商品なし、遷移失敗 | 管理者専用 |
| SCR-10 | 商品追加画面 | 商品登録成功 | 入力不正、登録失敗 | 管理者専用 |
| SCR-11 | 商品更新画面 | 商品更新成功 | 対象なし、更新失敗 | 管理者専用 |
| SCR-12 | 顧客一覧画面 | 顧客一覧表示成功 | データなし | 管理者専用 |
| SCR-13 | プロフィール更新画面 | 更新成功 | 入力不正、更新失敗 | 管理者専用 |
