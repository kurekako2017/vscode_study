# 76. 画面メッセージ一覧

## 1. 文書情報

| 項目 | 内容 |
|---|---|
| 文書名 | 画面メッセージ一覧 |
| 対象システム | JtProject |
| 文書区分 | 画面表示設計資料 |
| 版数 | v1.0 |

## 2. 目的

本書は、画面上に表示する主要メッセージを整理し、どの画面で、どの条件で、どの文言を表示するかを明確化することを目的とする。

## 4. 設計上の留意点

- 利用者向け文言は簡潔にし、内部エラー詳細を直接出さないこと。
- 同一意味のメッセージは画面間で統一すること。
- 実装時のメッセージキー化を考慮する場合は ID 体系を別途定義すること。
