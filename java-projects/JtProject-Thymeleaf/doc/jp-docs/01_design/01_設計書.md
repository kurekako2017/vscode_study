## 2. システム概要

`JtProject` は、ユーザー向け画面と管理者向け画面を持つ簡易 EC サンプルシステムである。

主な機能：

- ユーザーログイン
- ユーザー登録
- 商品一覧表示
- カート追加、カート表示、カート削除
- 管理者ログイン
- 商品カテゴリ管理
- 商品管理
- 顧客一覧表示
- 管理者プロフィール更新

## 3. アーキテクチャ概要

採用している構成：

- View: `src/main/webapp/views/*.jsp`
- Controller: `controller/UserController.java`, `controller/AdminController.java`
- Service: `services/*`
- DAO: `dao/*`
- Model: `models/*`
- Config: `application*.properties`, `WebMvcConfig.java`, `TransactionConfig.java`

## 4. 対象機能一覧

| 機能ID | 機能名 | 利用者 | 主なURL |
|---|---|---|---|
| F-01 | ユーザーログイン | 一般ユーザー | `/`, `/userloginvalidate` |
| F-02 | ユーザー登録 | 一般ユーザー | `/register`, `/newuserregister` |
| F-03 | 商品一覧 | 一般ユーザー | `/index`, `/user/products` |
| F-04 | カート管理 | 一般ユーザー | `/products/addtocart`, `/user/cart` |
| F-05 | 管理者ログイン | 管理者 | `/admin/login`, `/admin/loginvalidate` |
| F-06 | カテゴリ管理 | 管理者 | `/admin/categories` |
| F-07 | 商品管理 | 管理者 | `/admin/products` |
| F-08 | 顧客一覧 | 管理者 | `/admin/customers` |
| F-09 | プロフィール更新 | 管理者 | `/admin/profileDisplay`, `/admin/updateuser` |

## 5. 非機能観点

| 項目 | 内容 |
|---|---|
| セッション管理 | 一般ユーザーは Cookie + Session、管理者は Session 管理 |
| 例外処理 | 一部 Controller に try-catch 実装あり |
| ログ | `System.out.println` とログファイル出力資料あり |
| 文字コード | JSP は UTF-8 指定 |
| テスト | `src/test/java` に一部 Controller テストあり |
