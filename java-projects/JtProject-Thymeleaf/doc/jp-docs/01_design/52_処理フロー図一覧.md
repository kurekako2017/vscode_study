# 処理フロー図一覧

## 1. 目的

本書は `JtProject` に存在する `処理フロー図` の配置先を一覧化し、業務単位、機能単位、クラス単位のどこに何を記載しているかを明確にする。

## 2. 記載レベル別の配置方針

| レベル | 主配置先 | 記載対象 |
|---|---|---|
| 業務 / 画面機能全体 | `03_詳細設計書` | 画面入力から画面返却までの主要業務フロー |
| 機能単位 | `13_機能設計書` | 機能観点の処理手順、判定条件、入出力 |
| クラス / メソッド単位 | `15a` / `15b` / `15c` 配下 | コード実装に直結する処理順、戻り値、分岐 |
| 総覧 | `22_クラス図・処理フロー図` | 全体俯瞰用の代表フロー |

## 4. 運用ルール

- 新しい業務フローを追加する場合は、まず `03_詳細設計書` に載せるべきかを判断する。
- コード実装順やメソッド分岐を説明したい場合は、対象クラス文書へ追加する。
- `22_クラス図・処理フロー図` は代表図のみを維持し、全メソッドの置き場にはしない。
- 追加時は本一覧も更新し、重複記載を避ける。
