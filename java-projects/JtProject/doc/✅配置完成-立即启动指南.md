## 🚀 现在立即启动项目（3个简单步骤）

### 步骤1：在IDEA中刷新项目（必做）

**方式A：使用菜单（推荐）**
```
1. 在IDEA中点击菜单：File → Invalidate Caches...
2. 在弹出的对话框中，勾选 "Invalidate and Restart"
3. 点击 "Invalidate and Restart" 按钮
4. 等待IDEA重启（约30秒）
```

**方式B：关闭重开（简单）**
```
1. 关闭IDEA
2. 重新打开IDEA
3. 打开此项目
```

### 步骤2：等待Maven同步完成

- 重启后，IDEA会自动识别Maven项目
- 观察右下角的进度条：`Resolving Maven dependencies...`
- 等待完成（首次可能需要2-5分钟下载依赖）
- 完成后会显示：`All files are up-to-date`

### 步骤3：运行项目

现在运行按钮应该可以使用了！

**3种运行方式任选其一：**

#### 方式1：使用右上角运行按钮 ▶（最简单）
- 右上角会显示：`JtSpringProjectApplication`
- 点击旁边的绿色三角形按钮 ▶
- 完成！

#### 方式2：使用快捷键（最快）
```
1. 按 Shift + F10
2. 完成！
```

#### 方式3：从主类文件运行
```
1. 打开文件：src/main/java/com/jtspringproject/JtSpringProject/JtSpringProjectApplication.java
2. 在main方法左侧（行号旁边）会出现绿色三角形 ▶
3. 点击绿色三角形
4. 选择 "Run 'JtSpringProjectApplication.main()'"
5. 完成！
```

## 🛠️ 如果运行按钮还是看不到

### 检查1：Maven是否同步完成
```
1. 点击右侧边栏的 "Maven" 标签
2. 点击刷新图标（圆形箭头）🔄
3. 或按快捷键：Ctrl + Shift + O
```

### 检查2：JDK是否配置
```
1. 按 Ctrl + Alt + Shift + S 打开项目结构
2. 检查 "Project" → "SDK" 是否选择了 Java 11+
3. 如果显示 "No SDK"，点击 "Edit" 添加JDK
```

### 检查3：查看运行配置
```
1. 按 Alt + Shift + F10
2. 应该会看到 "JtSpringProjectApplication"
3. 选择它并按回车运行
```

### 检查4：查看事件日志
```
1. 按 Ctrl + E 然后输入 "Event Log"
2. 查看是否有错误信息
```
