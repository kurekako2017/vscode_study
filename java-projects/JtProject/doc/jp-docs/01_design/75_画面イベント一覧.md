# 75. 画面イベント一覧

## 1. 文書情報

| 項目 | 内容 |
|---|---|
| 文書名 | 画面イベント一覧 |
| 対象システム | JtProject |
| 文書区分 | 画面動作設計資料 |
| 版数 | v1.0 |

## 2. 目的

本書は、各画面で発生する主なイベントと、その結果呼び出される処理、遷移先、メッセージの有無を整理することを目的とする。

## 4. 設計上の留意点

- イベント名はボタンやリンクのラベルと対応が取れるようにすること。
- 画面遷移先は画面遷移図と一致させること。
- イベントに伴う入力チェック、メッセージ表示は別資料と整合させること。
