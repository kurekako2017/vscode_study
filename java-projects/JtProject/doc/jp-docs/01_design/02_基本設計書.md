## 4. 主要機能設計

### 4.1 ユーザーログイン

| 項目 | 内容 |
|---|---|
| 機能ID | F-01 |
| 機能名 | ユーザーログイン |
| 画面 | `userLogin.jsp` |
| 入力 | username, password |
| 出力 | ログイン成功時 `index.jsp`、失敗時エラーメッセージ |
| 主処理 | `UserController.userlogin()` |
| 備考 | Cookie と Session にユーザー名を保持 |

### 4.2 カート追加

| 項目 | 内容 |
|---|---|
| 機能ID | F-04 |
| 機能名 | カート追加 |
| 入力 | 商品 ID |
| 前提条件 | ユーザーログイン済み |
| 出力 | カート画面へ遷移 |
| 主処理 | `UserController.addToCart()` |
| 異常系 | 未ログイン、ユーザーなし、商品なし、管理者操作 |

### 4.3 管理者商品管理

| 項目 | 内容 |
|---|---|
| 機能ID | F-07 |
| 機能名 | 商品管理 |
| 利用者 | 管理者 |
| 機能内容 | 一覧、追加、更新、削除 |
| 主処理 | `AdminController` 商品系メソッド |
| 使用画面 | `products.jsp`, `productsAdd.jsp`, `productsUpdate.jsp` |

## 5. 画面遷移概要

```text
ユーザーログイン画面
  -> ログイン成功
  -> 商品トップ画面
  -> 商品一覧
  -> カート追加
  -> カート画面

管理者ログイン画面
  -> 管理者ホーム
  -> カテゴリ管理
  -> 商品管理
  -> 顧客一覧
  -> プロフィール更新
```

## 6. DB 利用概要

主要エンティティ：

- User
- Product
- Category
- Cart
- CartProduct

主な関係：

- Product は Category に属する
- Cart は User に紐づく
- CartProduct は Cart と Product の中間関係を表現する
