# 70. 画面レイアウト定義書

## 1. 文書情報

| 項目 | 内容 |
|---|---|
| 文書名 | 画面レイアウト定義書 |
| 対象システム | JtProject |
| 文書区分 | 基本設計補足 |
| 版数 | v1.0 |

## 2. 目的

本書は、主要画面のレイアウト構成、表示ブロック、入力エリア、ボタン配置、一覧表示領域を整理し、画面設計の基準資料とすることを目的とする。

## 3. レイアウト定義方針

- 基本設計では画面の役割、レイアウトの大枠、主要部品配置を定義する。
- 項目単位の詳細は `画面項目定義書` を参照する。
- 入力チェックやメッセージ詳細は `入力チェック一覧`、`メッセージ一覧` と整合させる。

## 4. 画面レイアウト一覧

### 4.1 SCR-01 一般ログイン画面

画面用途:
一般利用者が認証を行い、商品閲覧機能へ遷移するための入口画面。

```text
+--------------------------------------------------+
| JtProject / ユーザログイン                       |
+--------------------------------------------------+
| [メッセージ表示エリア]                           |
|  ユーザ名  : [                    ]              |
|  パスワード: [                    ]              |
|                                                  |
|  [ログイン]   [新規登録へ]                        |
+--------------------------------------------------+
| フッタ / 案内表示                                |
+--------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | システム名、画面タイトル |
| 入力エリア | `username`、`password` |
| 操作エリア | ログインボタン、登録画面リンク |
| メッセージエリア | 認証失敗メッセージ表示 |
| フッタ | 著作権または簡易案内 |

### 4.2 SCR-02 ユーザ登録画面

画面用途:
新規利用者が登録情報を入力し、利用開始前のアカウント作成を行う画面。

```text
+--------------------------------------------------+
| JtProject / ユーザ登録                           |
+--------------------------------------------------+
| [メッセージ表示エリア]                           |
|  ユーザ名  : [                    ]              |
|  パスワード: [                    ]              |
|  メール    : [                    ]              |
|                                                  |
|  [登録]     [ログイン画面へ戻る]                 |
+--------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | 画面タイトル |
| 入力エリア | ユーザ名、パスワード、補助情報 |
| 操作エリア | 登録ボタン、戻るリンク |
| メッセージエリア | 登録結果、重複エラー表示 |

### 4.3 SCR-03 商品トップ / 一覧画面

画面用途:
ログイン済利用者が商品一覧を確認し、カート追加や商品参照を行う主画面。

```text
+--------------------------------------------------------------+
| JtProject / 商品一覧         ログインユーザ: xxxx  [ログアウト] |
+--------------------------------------------------------------+
| [案内メッセージ / お知らせ]                                   |
|--------------------------------------------------------------|
| 商品名      | 価格      | 分類      | 操作                   |
|--------------------------------------------------------------|
| 商品A       | 1000      | 食品      | [カート追加]           |
| 商品B       | 2000      | 雑貨      | [カート追加]           |
| ...                                                        |
|--------------------------------------------------------------|
| [カートを見る]                                               |
+--------------------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | ログインユーザ表示、画面タイトル |
| 一覧エリア | 商品一覧、価格、分類情報 |
| 操作エリア | カート追加ボタン、詳細導線 |
| ナビゲーション | カート画面、ログアウト等 |

### 4.4 SCR-05 カート画面

画面用途:
利用者がカート投入済商品を確認し、削除や戻り遷移を行う画面。

```text
+--------------------------------------------------------------+
| JtProject / カート画面                     ユーザ: xxxx       |
+--------------------------------------------------------------+
| [メッセージ表示エリア]                                       |
|--------------------------------------------------------------|
| 商品名      | 価格      | 数量/件数 | 操作                   |
|--------------------------------------------------------------|
| 商品A       | 1000      | 1         | [削除]                |
| 商品B       | 2000      | 1         | [削除]                |
|--------------------------------------------------------------|
| [商品一覧へ戻る]   [購入確定(将来拡張)]                      |
+--------------------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | 画面タイトル、ユーザ情報 |
| 一覧エリア | カート内商品一覧 |
| 操作エリア | 削除、戻る、将来の購入確定導線 |
| メッセージエリア | 商品追加結果、エラー表示 |

### 4.5 SCR-06 管理者ログイン画面

画面用途:

```text

+--------------------------------------------------+
| [メッセージ表示エリア]                           |
|  管理者名  : [                    ]              |
|  パスワード: [                    ]              |
|                                                  |
|  [ログイン]                                      |
+--------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | 管理者画面タイトル |
| 入力エリア | 管理者名、パスワード |
| 操作エリア | ログインボタン |
| メッセージエリア | 認証失敗表示 |

### 4.6 SCR-08 分類管理画面

画面用途:
管理者が分類マスタを追加、更新、削除するための管理画面。

```text
+--------------------------------------------------------------+
| 管理メニュー / 分類管理                                      |
+--------------------------------------------------------------+
| 分類名: [                          ]   [追加]                |
|--------------------------------------------------------------|
| 分類ID     | 分類名                 | 操作                   |
|--------------------------------------------------------------|
| 1          | 食品                   | [更新] [削除]         |
| 2          | 雑貨                   | [更新] [削除]         |
|--------------------------------------------------------------|
| [管理ホームへ戻る]                                           |
+--------------------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | 管理メニュー、画面タイトル |
| 入力エリア | 分類名入力 |
| 一覧エリア | 既存分類一覧 |
| 操作エリア | 追加、更新、削除ボタン |

### 4.7 SCR-09 商品管理画面

画面用途:
管理者が商品マスタを一覧確認し、追加、更新、削除を実施する管理画面。

```text
+--------------------------------------------------------------------------------+
| 管理メニュー / 商品管理                                                        |
+--------------------------------------------------------------------------------+
| [商品追加へ]   [検索欄(将来拡張)]                                              |
|--------------------------------------------------------------------------------|
| 商品ID | 商品名         | 価格      | 分類      | 操作                         |
|--------------------------------------------------------------------------------|
| 101    | 商品A          | 1000      | 食品      | [更新] [削除]               |
| 102    | 商品B          | 2000      | 雑貨      | [更新] [削除]               |
|--------------------------------------------------------------------------------|
| [管理ホームへ戻る]                                                             |
+--------------------------------------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | 管理メニュー、画面タイトル |
| 検索/入力エリア | 将来検索欄、入力欄 |
| 一覧エリア | 商品 ID、名称、価格、分類 |
| 操作エリア | 追加、更新、削除、詳細導線 |

### 4.8 SCR-07 管理者ホーム画面

画面用途:
管理者が各管理機能へ遷移するためのメニュー画面。

```text
+--------------------------------------------------------------+
| 管理メニュー / 管理者ホーム                                  |
+--------------------------------------------------------------+
| [お知らせ / 管理者メッセージ]                                |
|                                                              |
|  [分類管理]   [商品管理]   [顧客一覧]                        |
|                                                              |
|  [プロフィール更新]   [ログアウト]                           |
|                                                              |
+--------------------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | 管理画面タイトル、管理者識別情報 |
| メニューエリア | 各管理機能への遷移ボタン |
| メッセージエリア | お知らせ、処理完了案内 |
| 操作エリア | ログアウト、補助遷移 |

### 4.9 SCR-12 顧客一覧画面

画面用途:
管理者が登録済顧客情報を参照し、管理対象ユーザを確認する画面。

```text
+--------------------------------------------------------------------------------+
| 管理メニュー / 顧客一覧                                                        |
+--------------------------------------------------------------------------------+
| [検索欄(将来拡張)]                                                             |
|--------------------------------------------------------------------------------|
| 顧客ID | ユーザ名       | メール/連絡先     | 権限      | 操作/参照             |
|--------------------------------------------------------------------------------|
| 1      | user01         | user01@example    | NORMAL    | [詳細]                |
| 2      | admin01        | admin@example     | ADMIN     | [詳細]                |
|--------------------------------------------------------------------------------|
| [管理ホームへ戻る]                                                             |
+--------------------------------------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | 管理メニュー、画面タイトル |
| 検索エリア | 将来の顧客検索条件入力欄 |
| 一覧エリア | 顧客 ID、ユーザ名、連絡先、権限表示 |
| 操作エリア | 詳細参照、戻る導線 |

画面用途:

```text
+--------------------------------------------------------------+

+--------------------------------------------------------------+
| [メッセージ表示エリア]                                       |
|  ユーザ名      : [                    ]                      |
|  パスワード    : [                    ]                      |
|  メール/連絡先 : [                    ]                      |
|                                                              |
|  [更新]     [管理ホームへ戻る]                               |
+--------------------------------------------------------------+
```

| 領域 | 内容 |
|---|---|
| ヘッダ | 管理メニュー、画面タイトル |
| 入力エリア | ユーザ名、パスワード、連絡先等 |
| メッセージエリア | 更新結果、入力エラー表示 |
| 操作エリア | 更新、戻るボタン |

## 5. レイアウト共通ルール

- 画面タイトルは各画面上部に配置する。
- エラーメッセージ、完了メッセージは入力欄近傍または画面上部に統一表示する。
- 一覧系画面は表形式を基本とし、識別キーを含める。
- 管理画面では更新、削除操作を一覧近傍に配置する。
