## 2. 目的

本書は、画面入力項目と DB 項目の対応関係を整理し、画面設計、クラス設計、テーブル設計の追跡性を明確化することを目的とする。

## 4. 設計上の留意点

- パラメータ名の大小文字差異、`categoryId` / `categoryid` の揺れは統一対象とすること。
- DB 項目名はテーブル定義書と一致させること。
- 画面項目追加時は本一覧、画面項目定義書、テーブル定義書を同時更新すること。
