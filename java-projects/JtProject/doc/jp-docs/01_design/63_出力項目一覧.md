# 63. 出力項目一覧

## 2. 目的

本書は、主要画面および一覧表示で利用者に出力される項目を整理し、表示目的、出力元、利用者区分を明確にすることを目的とする。

## 3. 出力項目一覧

| 項目ID | 画面/帳票 | 出力項目 | 出力元 | 利用者 | 用途 | 備考 |
|---|---|---|---|---|---|---|
| OUT-01 | 商品一覧画面 | 商品名 | `PRODUCT.name` | 一般利用者 | 商品識別 | 主表示項目 |
| OUT-02 | 商品一覧画面 | 価格 | `PRODUCT.price` | 一般利用者 | 商品比較 | 通貨表示形式は要統一 |
| OUT-03 | 商品一覧画面 | 分類名 | `CATEGORY.name` | 一般利用者 | 商品分類把握 | 参照項目 |

## 4. 出力設計観点

- 利用者に不要な内部 ID は原則非表示とし、管理画面のみ表示を許容する。
- 個人情報は最小限表示とし、用途がない項目は画面出力しない。
- 画面表示文言は `メッセージ一覧` と整合させること。
- 出力項目追加時は `帳票一覧`、`画面一覧` とあわせて更新すること。
