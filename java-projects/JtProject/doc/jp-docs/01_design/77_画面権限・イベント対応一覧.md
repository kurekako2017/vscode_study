# 77. 画面権限・イベント対応一覧

## 1. 文書情報

| 項目 | 内容 |
|---|---|
| 文書名 | 画面権限・イベント対応一覧 |
| 対象システム | JtProject |
| 文書区分 | 画面権限設計資料 |
| 版数 | v1.0 |

## 2. 目的

本書は、各画面で実行可能なイベントと利用可能権限の対応関係を整理し、画面設計、権限設計、イベント設計の整合を明確化することを目的とする。

## 3. 対応一覧

| 画面ID | 画面名 | イベント | 未ログイン | `ROLE_NORMAL` | `ROLE_ADMIN` | 備考 |
|---|---|---|---|---|---|---|
| SCR-01 | ユーザーログイン画面 | ログイン実行 | 可 | 可 | 可 | 認証入口 |
| SCR-01 | ユーザ登録画面遷移 | 可 | 可 | 可 | 共通導線 |
| SCR-02 | ユーザ登録実行 | 可 | 可 | 可 | 新規登録 |
| SCR-03 | カート追加 | 不可 | 可 | 原則不可 | 一般利用者向け |
| SCR-03 | カート画面遷移 | 不可 | 可 | 原則不可 | 一般利用者向け |
| SCR-05 | カート商品削除 | 不可 | 可 | 不可 | 一般利用者向け |
| SCR-06 | 管理者ログイン | 可 | 可 | 可 | 管理認証 |
| SCR-07 | 分類管理遷移 | 不可 | 不可 | 可 | 管理者のみ |
| SCR-07 | 商品管理遷移 | 不可 | 不可 | 可 | 管理者のみ |
| SCR-07 | 顧客一覧遷移 | 不可 | 不可 | 可 | 管理者のみ |
| SCR-08 | 分類追加/更新/削除 | 不可 | 不可 | 可 | 管理者のみ |
| SCR-09 | 商品追加/更新/削除 | 不可 | 不可 | 可 | 管理者のみ |
| SCR-12 | 顧客詳細参照 | 不可 | 不可 | 可 | 管理者のみ |
| SCR-13 | プロフィール更新 | 不可 | 不可 | 可 | 管理者のみ |

## 4. 設計上の留意点

- 画面表示可否とイベント実行可否を分けて考えること。
- URL 直打ち時も同等の権限制御が必要である。
- 権限変更時は画面一覧、権限一覧、イベント一覧も更新すること。
