## 目录结构

- `README.md` / `00_文档一覧.md`
  入口与总索引
- `01_design/`
  设计书、画面、时序、例外、流程相关文档
- `02_class-design/`
  类设计总览与 Controller / Service / DAO / Model 分层类文档
- `03_database/`
  ER、表定义、URL、DDL、批处理与 DB 资料
- `04_testing/`
  测试书、测试数据、测试结果、实施记录
- `05_management/`
  障害、改修、调查、承认、风险、成果物、版数、纳品管理
- `06_operations/`
  运用、发布、FAQ、监视、交接、文档相关图

## 项目对象

本套文档基于以下实际功能整理：

- 用户登录 `GET /`, `POST /userloginvalidate`
- 用户注册 `GET /register`, `POST /newuserregister`
- 商品首页 `GET /index`
- 用户购物车 `GET /user/cart`, `GET|POST /products/addtocart`
- 管理员登录 `GET /admin/login`, `POST /admin/loginvalidate`
- 管理员商品管理 `GET /admin/products`, `POST /admin/products/add`, `POST /admin/products/update/{id}`
- 管理员分类管理 `GET|POST /admin/categories`
