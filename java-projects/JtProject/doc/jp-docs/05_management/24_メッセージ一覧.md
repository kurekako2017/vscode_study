# メッセージ一覧

## 1. 目的

本書は `JtProject` で画面表示や処理結果として利用されている主なメッセージを整理する。

## 2. メッセージ一覧

| No | メッセージ | 使用箇所 | 条件 | 備考 |
|---|---|---|---|---|
| MSG-01 | `Please enter correct email and password` | ユーザーログイン | 認証失敗 | username 画面に対し email 文言、改善余地あり |
| MSG-02 | `系统错误：无法验证用户，请联系管理员（user=null）` | ユーザーログイン | `userService` が null 返却 | システム異常系 |
| MSG-03 | `系统异常：...` | ユーザーログイン | 予期しない例外 | 例外内容を表示 |
| MSG-04 | `No products are available` | 商品一覧 / 商品トップ | 商品一覧空 | 画面表示用 |
| MSG-05 | `Error loading products: ...` | 商品トップ | 商品一覧取得失敗 | 画面表示用 |
| MSG-06 | `Please login first` | カート追加 / カート表示 | 未ログイン | セッションメッセージにも使用 |
| MSG-07 | `User not found, please login again` | カート追加 | ユーザー取得失敗 | 再ログイン要求 |
| MSG-08 | `管理员请使用后台管理商品。` | カート追加 / カート表示 | 管理者が一般カート操作 | 日本語未統一 |
| MSG-09 | `Product not found` | カート追加 | 商品不存在 | 追加中止 |

| MSG-12 | `User not found` | カート表示 | 顧客不存在 | 画面表示用 |
| MSG-13 | `Cart not found` | カート削除 | カート不存在 | セッションメッセージ |
| MSG-14 | `Product not found in cart` | カート削除 | 対象明細不存在 | セッションメッセージ |

| MSG-18 | `<username> is taken. Please choose a different username.` | ユーザー登録 | ユーザー名重複 | 登録画面表示用 |

## 3. 課題

- 英語、中国語、日本語が混在している
- メッセージの外部化がされていない
- コード内ハードコードが多い

## 4. 改善方針

- `messages.properties` 導入
- 画面向け文言の日本語統一
- システムエラー文言と業務エラー文言の分離
