# 59. 設定値一覧

## 1. 文書情報

| 項目 | 内容 |
|---|---|
| 文書名 | 設定値一覧 |
| 対象システム | JtProject |
| 文書区分 | 環境設定資料 |
| 版数 | v1.0 |

## 2. 目的

本書は、`JtProject` の起動、DB 接続、ログ出力、アプリ動作に必要な設定値を整理し、環境差異による不具合を防止することを目的とする。

## 3. 設定値管理方針

- 環境依存値は `application.properties` 等の設定ファイルに外出しすること。
- 開発、試験、本番で差異が出る値は識別可能にすること。
- 認証情報、接続パスワードは文書上でマスクし、実値は別管理とすること。

## 4. 設定値一覧

| 設定ID | 設定名 | 用途 | 設定例 | 設定位置 | 備考 |
|---|---|---|---|---|---|
| CFG-01 | `server.port` | アプリ待受ポート | `8080` | `application.properties` | 開発標準 |
| CFG-02 | `spring.datasource.url` | DB 接続先 URL | `jdbc:mysql://localhost:3306/jtproject` | `application.properties` | 環境依存 |
| CFG-03 | `spring.datasource.username` | DB 接続ユーザ | `root` | `application.properties` | 実値管理注意 |
| CFG-04 | `spring.datasource.password` | DB 接続パスワード | `******` | `application.properties` | 文書ではマスク |
| CFG-05 | `spring.jpa.hibernate.ddl-auto` | DDL 自動生成制御 | `update` | `application.properties` | 本番要注意 |
| CFG-06 | `spring.jpa.show-sql` | SQL 表示有無 | `true/false` | `application.properties` | 開発向け |
| CFG-07 | `logging.level.root` | ルートログレベル | `INFO` | `application.properties` | 障害調査時変更候補 |
| CFG-08 | `logging.file.name` | ログファイル名 | `jtproject.log` | `application.properties` | 任意 |
| CFG-09 | Session Timeout | セッション有効期間 | `30 min` | サーバ設定 | 未明確時は要定義 |
| CFG-10 | JVM Option | メモリ、起動パラメータ | `-Xms256m -Xmx512m` | 実行設定 | 環境依存 |

## 5. 環境別差異管理

| 環境 | 主な差異項目 | 管理方針 |
|---|---|---|
| 開発環境 | DB 名、ログレベル、SQL 表示 | 開発者向けに可視性優先 |
| 試験環境 | テストデータ、ログ保持、接続先 | 試験証跡優先 |
| 本番想定 | DB 接続先、パスワード、ログレベル | 安全性と運用性優先 |

## 6. 変更時の留意点

- 設定値変更時は `版数管理表` と `変更履歴一覧` の更新要否を確認すること。
- DB 接続値変更時は `外部接続一覧` と整合を取ること。
- ログ関連設定変更時は `ログ設計一覧` も見直すこと。
