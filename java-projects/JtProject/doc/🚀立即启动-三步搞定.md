## 🚀 立即启动（只需3步）

### 第1步：刷新IDEA项目（必须做）

**在IDEA中执行以下操作：**

#### 方法A：使用菜单（推荐）⭐
1. 点击顶部菜单：`File` → `Invalidate Caches...`
2. 在弹出的对话框中勾选：`Invalidate and Restart`
3. 点击：`Invalidate and Restart` 按钮
4. 等待IDEA重启（约30秒）

#### 方法B：关闭重开（简单）
1. 关闭IDEA（File → Exit）
2. 重新打开IDEA
3. 打开此项目

---

### 第2步：等待Maven同步（自动）

IDEA重启后会自动同步Maven：
- 右下角会显示：`Resolving Maven dependencies...`
- 等待进度条完成（首次约2-5分钟）
- 完成后显示：`All files are up-to-date`

💡 **提示：** 如果没有自动同步，按 `Ctrl + Shift + O` 手动刷新

---

### 第3步：运行项目

#### 🎯 运行方式1：点击运行按钮（最直观）
- 查看IDEA右上角
- 应该显示：`JtSpringProjectApplication` ▼
- 点击旁边的绿色三角形按钮 ▶
- 完成！🎉

#### ⚡ 运行方式2：快捷键（最快）
```
按键：Shift + F10
```

#### 📝 运行方式3：从代码运行
1. 打开文件：`JtSpringProjectApplication.java`
2. 在 `main` 方法左侧点击绿色三角形 ▶
3. 选择：`Run 'JtSpringProjectApplication.main()'`

## 🔍 如果运行按钮没出现

### 快速解决方案：

#### 1️⃣ 手动刷新Maven
```
按 Ctrl + Shift + O
或者：右侧 "Maven" 标签 → 点击刷新图标 🔄
```

#### 2️⃣ 检查JDK配置
```
按 Ctrl + Alt + Shift + S
→ Project → SDK 
→ 确保选择了 Java 11 或更高版本
```

#### 3️⃣ 使用运行配置快捷键
```
按 Alt + Shift + F10
→ 选择 "JtSpringProjectApplication"
→ 按回车
```

#### 4️⃣ 查看配置是否存在
```
右上角运行按钮旁边的下拉菜单
→ Edit Configurations...
→ 应该能看到 "JtSpringProjectApplication"
```
