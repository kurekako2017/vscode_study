## 项目概述

### 基本信息
- **项目名称**：JT Spring E-commerce System（JT电商系统）
- **项目类型**：电商购物平台
- **开发框架**：Spring Boot 2.7.18
- **架构模式**：MVC（Model-View-Controller）
- **构建工具**：Maven
- **Java版本**：Java 11

### 主要功能模块
1. **用户模块**：用户注册、登录、个人信息管理
2. **商品模块**：商品浏览、搜索、详情查看
3. **购物车模块**：添加商品到购物车、管理购物车
4. **管理员模块**：商品管理、分类管理、用户管理
5. **分类模块**：商品分类管理

### 分层架构说明

#### 1. **Controller层（控制层）** - `com.jtspringproject.JtSpringProject.controller`
- **职责**：接收HTTP请求，处理参数，调用Service层，返回视图
- **主要类**：
  - `UserController`：处理用户相关请求
  - `AdminController`：处理管理员相关请求
- **特点**：
  - 使用 `@Controller` 注解
  - 使用 `@RequestMapping`、`@GetMapping`、`@PostMapping` 映射URL
  - 通过 `Model` 或 `ModelAndView` 向视图传递数据

#### 2. **Service层（业务层）** - `com.jtspringproject.JtSpringProject.services`
- **职责**：实现业务逻辑，处理事务，调用DAO层
- **主要接口和实现**：
  - `UserService` / `UserServiceImpl`：用户业务逻辑
  - `ProductService` / `ProductServiceImpl`：商品业务逻辑
  - `CartService` / `CartServiceImpl`：购物车业务逻辑
  - `CategoryService` / `CategoryServiceImpl`：分类业务逻辑
- **特点**：
  - 接口定义业务方法
  - 实现类使用 `@Service` 注解
  - 通过 `@Autowired` 注入DAO

#### 3. **DAO层（数据访问层）** - `com.jtspringproject.JtSpringProject.dao`
- **职责**：直接操作数据库，执行CRUD操作
- **主要接口和实现**：
  - `UserDao` / `UserDaoImpl`
  - `ProductDao` / `ProductDaoImpl`
  - `CartDao` / `CartDaoImpl`
  - `CategoryDao` / `CategoryDaoImpl`
  - `CartProductDao` / `CartProductDaoImpl`
- **特点**：
  - 接口定义数据访问方法
  - 实现类使用 `@Repository` 注解
  - 使用Hibernate的 `SessionFactory` 操作数据库

#### 4. **Model层（实体层）** - `com.jtspringproject.JtSpringProject.models`
- **职责**：定义数据库实体类，映射数据库表
- **主要实体**：
  - `User`：用户实体
  - `Product`：商品实体
  - `Cart`：购物车实体
  - `CartProduct`：购物车商品关联实体
  - `Category`：分类实体
- **特点**：
  - 使用JPA注解（`@Entity`、`@Table`、`@Id`等）
  - 定义属性和getter/setter方法

#### 5. **View层（视图层）** - `src/main/webapp/views/`
- **职责**：展示数据，用户交互界面
- **主要页面**：
  - `userLogin.jsp`：用户登录页
  - `register.jsp`：用户注册页
  - `index.jsp`：首页
  - `products.jsp`：商品列表页
  - `adminHome.jsp`：管理员首页
  - 等等...

## 启动流程详解

### 🚀 从启动到准备接收请求的完整过程

```
步骤1: 启动JVM
    │
    ├─> 执行 JtSpringProjectApplication.main() 方法
    │
    ▼
步骤2: Spring Boot初始化
    │
    ├─> SpringApplication.run(JtSpringProjectApplication.class, args)
    │   │
    │   ├─> 创建Spring应用上下文（ApplicationContext）
    │   ├─> 扫描 @SpringBootApplication 注解
    │   ├─> 组件扫描（com.jtspringproject.JtSpringProject包及子包）
    │   └─> 加载 application.properties 配置文件
    │
    ▼
步骤3: 自动配置
    │
    ├─> 排除 HibernateJpaAutoConfiguration（因为使用手动配置）
    ├─> 启用 Spring MVC 自动配置
    ├─> 配置嵌入式Tomcat服务器
    └─> 配置JSP视图解析器
    │
    ▼
步骤4: 加载Hibernate配置
    │
    ├─> 执行 HibernateConfiguration 类
    │   │
    │   ├─> 读取数据库配置（db.driver, db.url, db.username, db.password）
    │   ├─> 创建 DataSource（数据源）
    │   ├─> 创建 SessionFactory
    │   └─> 配置Hibernate属性（方言、显示SQL等）
    │
    ▼
步骤5: 初始化数据库
    │
    ├─> Hibernate根据实体类自动创建/更新数据库表
    │   （hibernate.hbm2ddl.auto=update）
    │
    ├─> 执行 data.sql（如果存在）
    │   └─> 插入初始数据（用户、商品、分类等）
    │
    ▼
步骤6: 注册Bean（依赖注入）
    │
    ├─> 扫描并注册所有带 @Component、@Service、@Repository、@Controller 的类
    │   │
    │   ├─> Controller层：
    │   │   ├─> UserController
    │   │   └─> AdminController
    │   │
    │   ├─> Service层：
    │   │   ├─> UserServiceImpl
    │   │   ├─> ProductServiceImpl
    │   │   ├─> CartServiceImpl
    │   │   └─> CategoryServiceImpl
    │   │
    │   └─> DAO层：
    │       ├─> UserDaoImpl
    │       ├─> ProductDaoImpl
    │       ├─> CartDaoImpl
    │       ├─> CategoryDaoImpl
    │       └─> CartProductDaoImpl
    │
    ├─> 解析 @Autowired 依赖关系
    │   └─> 注入依赖（Controller → Service → DAO → SessionFactory）
    │

步骤7: 映射URL路由
    │
    ├─> 扫描所有 @RequestMapping、@GetMapping、@PostMapping

### 示例2：用户提交登录表单

```
用户填写用户名和密码，点击"登录"按钮
    │
    ▼
【HTTP Request】POST /userloginvalidate
    │
    │ 携带参数:
    │ - username: "admin"
    │ - password: "123456"
    │
    ▼
┌──────────────────────────────────────────────┐
│ DispatcherServlet接收请求                     │
└──────────┬───────────────────────────────────┘
           │
           ▼
┌──────────────────────────────────────────────┐
│ HandlerMapping查找处理器                      │
│ 找到: UserController.userlogin(...)          │
└──────────┬───────────────────────────────────┘
           │
           ▼
┌──────────────────────────────────────────────┐
│ UserController.userlogin(                    │
│     @RequestParam String username,           │
│     @RequestParam String password,           │
│     HttpServletResponse response,            │
│     Model model                              │
│ )                                            │
│                                              │
│ ▼ 步骤1: 接收参数                            │
│   username = "admin"                         │
│   password = "123456"                        │
│                                              │
│ ▼ 步骤2: 调用Service层验证用户                │
│   User user = userService.checkLogin(        │
│       username, password                     │
│   );                                         │
└──────────┬───────────────────────────────────┘
           │
           ▼
┌──────────────────────────────────────────────┐
│ UserServiceImpl.checkLogin(...)              │
│                                              │
│ ▼ 步骤3: 调用DAO层查询数据库                  │
│   User user = userDao.getUser(username);     │

           │
           ▼
┌──────────────────────────────────────────────┐
│ UserDaoImpl.getUser(String username)         │
│                                              │
│ ▼ 步骤4: 使用Hibernate查询数据库              │
│   Session session = sessionFactory           │
│       .getCurrentSession();                  │

│   Query query = session.createQuery(         │
│       "FROM User WHERE username = :username" │
│   );                                         │
│   query.setParameter("username", username);  │
│                                              │
│   User user = query.uniqueResult();          │
│                                              │

## 各模块详细交互流程 (Detailed Interaction Flows by Module)

> **本节详细列出了每个功能模块从前端JSP页面到后端数据库的完整调用链路。**

### 1. 用户模块 (User Module)

#### 1.1 用户登录 (User Login)
*   **前端页面**: `userLogin.jsp` (用户输入用户名和密码)
*   **请求动作**: 点击"Login"按钮
*   **请求URL**: `POST /userloginvalidate`
*   **Controller**: `UserController.userlogin(@RequestParam username, @RequestParam password, ...)`
    *   接收参数: `username`, `password`
*   **Service**: `UserService.checkLogin(username, password)`
*   **DAO**: `UserDao.getUser(username)`
*   **数据库操作**: `SELECT * FROM users WHERE username = ?`
*   **逻辑判断**:
    *   **成功**: 密码匹配 -> 设置Cookie -> 返回 `index` (首页)
    *   **失败**: 密码不匹配 -> 返回 `userLogin` (带错误信息)

#### 1.2 用户注册 (User Register)
*   **前端页面**: `register.jsp` (用户填写注册信息)
*   **请求动作**: 点击"Register"按钮
*   **请求URL**: `POST /newuserregister`
*   **Controller**: `UserController.newUseRegister(@ModelAttribute User user)`
    *   接收参数: `User` 对象 (自动绑定)
*   **Service**: `UserService.checkUserExists(username)`
    *   **存在**: 返回 `register` (带错误信息 "username is taken")
    *   **不存在**:
        *   `user.setRole("ROLE_NORMAL")`
        *   `UserService.addUser(user)`
*   **DAO**: `UserDao.addUser(user)`
*   **数据库操作**: `INSERT INTO users (username, email, password, ...) VALUES (?, ?, ?, ...)
*   **结果**: 返回 `userLogin` (注册成功，去登录)

#### 1.3 用户查看商品 (User View Products)
*   **前端页面**: `index.jsp` 或 直接访问链接
*   **请求动作**: 点击"Products"链接
*   **请求URL**: `GET /user/products`
*   **Controller**: `UserController.getproduct()`
*   **Service**: `ProductService.getProducts()`
*   **DAO**: `ProductDao.getProducts()`
*   **数据库操作**: `FROM Product` (HQL)
*   **结果**: 返回 `uproduct` (展示商品列表 `uproduct.jsp`)

### 2. 管理员模块 (Admin Module)

#### 2.1 管理员登录 (Admin Login)
*   **前端页面**: `adminlogin.jsp`
*   **请求动作**: 点击"Login"按钮
*   **请求URL**: `POST /admin/loginvalidate`
*   **Controller**: `AdminController.adminlogin(@RequestParam username, @RequestParam password)`
*   **Service**: `UserService.checkLogin(username, password)`
*   **逻辑判断**:
    *   **成功**: 用户存在且 `role == "ROLE_ADMIN"` -> 设置 `adminlogcheck=1` -> 返回 `adminHome`
    *   **失败**: 返回 `adminlogin` (带错误信息)

**A. 查看所有分类**
*   **前端页面**: `adminHome.jsp` (点击"Categories")
*   **请求URL**: `GET /admin/categories`
*   **Controller**: `AdminController.getcategory()`
*   **Service**: `CategoryService.getCategories()`
*   **DAO**: `CategoryDao.getCategories()`
*   **数据库操作**: `FROM Category`
*   **结果**: 返回 `categories` (`categories.jsp`)

**B. 添加分类**
*   **前端页面**: `categories.jsp` (输入分类名，点击"Add")
*   **请求URL**: `POST /admin/categories`
*   **Controller**: `AdminController.addCategory(@RequestParam categoryname)`
*   **Service**: `CategoryService.addCategory(name)`
*   **DAO**: `CategoryDao.addCategory(category)`
*   **数据库操作**: `INSERT INTO categories (name) VALUES (?)`
*   **结果**: 重定向 `redirect:categories`

**C. 删除分类**
*   **前端页面**: `categories.jsp` (点击"Delete")
*   **请求URL**: `GET /admin/categories/delete?id={id}`
*   **Controller**: `AdminController.removeCategoryDb(@RequestParam id)`
*   **Service**: `CategoryService.deleteCategory(id)`
*   **DAO**: `CategoryDao.deleteCategory(id)`
*   **数据库操作**: `DELETE FROM categories WHERE id = ?`
*   **结果**: 转发 `forward:/categories`

**D. 更新分类**
*   **前端页面**: `categories.jsp` (点击"Update" -> 弹出模态框或跳转)
*   **请求URL**: `GET /admin/categories/update?categoryid={id}&categoryname={name}`
*   **Controller**: `AdminController.updateCategory(...)`
*   **Service**: `CategoryService.updateCategory(id, name)`
*   **DAO**: `CategoryDao.updateCategory(category)`
*   **数据库操作**: `UPDATE categories SET name = ? WHERE id = ?`
*   **结果**: 重定向 `redirect:/admin/categories`
