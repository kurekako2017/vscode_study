## 13. 最短记忆版

```text
LLM = 生成能力核心
Prompt = 给模型的任务说明
Structured Output = 让模型返回程序能解析的 JSON
Pydantic = Python 侧校验 JSON 和数据结构
RAG = 先检索资料，再基于资料回答
Embedding = 把文本变成向量
Chunk = 文档切分后的检索单位
Retriever = 根据问题找相关资料
FastAPI = 把 Python Agent 包成 HTTP API
React / Vue = 用户画面
CORS = 让前端可以调用后端 API
Tool Calling = 模型决定调用哪个工具
Agent = 围绕目标自动推进任务的系统
Workflow = 固定步骤控制 Agent 处理
State = 保存中间过程的数据
Eval = 评价回答质量
Guardrail = 控制安全风险
Docker / ECS = 部署 Python API 服务
Managed Model = 通过 API 使用托管大模型
Private LLM = 自社环境部署模型
```

日语面试一句话：

```text
生成AIアプリでは、FastAPIでPythonのAgentやRAG機能をバックエンドAPI化し、ReactやVueなどの画面、または既存業務システムからAPI連携で利用する構成が多いと理解しております。
```
