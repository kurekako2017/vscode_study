# Stock Agent 风险说明

## 重要声明

本项目及其生成的全部报告仅用于编程学习、系统设计演示和一般研究辅助，不构成投资建议、证券推荐、收益承诺或交易指令。

## 数据风险

- 第一版的公司信息、价格、PER、PBR 和股息率均为项目内置 Mock 数据。
- Mock 数据不是真实行情，不保证准确、完整、及时或适用于任何实际决策。
- 数据不会自动更新，不能用于判断当前市场状态。

## 分析风险

- “AI 总结”由简单规则生成，并非专业证券分析。
- 报告没有覆盖财务报表、公司治理、市场流动性、汇率、政策或突发事件等全部风险。
- 单一估值指标不能说明一家公司的真实价值或未来表现。

## 功能边界

- 不读取 `.env` 或任何 API Key。
- 不登录、连接或操作证券账户。
- 不提供买入、卖出、强烈推荐等投资指令。
- 不执行模拟交易或真实交易。

## 使用者责任

任何投资决定都应由使用者独立判断，并在需要时咨询具备相应资质的专业人士。请勿依据本项目的 Mock 报告投入真实资金。
