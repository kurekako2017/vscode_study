# Stock Agent 使用教程

## 1. 准备条件

- 已安装 Python 3.10 或更高版本。
- 当前终端位于 `vscode_study` 工作区。
- 不需要 API Key，也不需要创建 `.env`。

## 2. 运行第一个报告

```bash
cd ai-lab/stock-agent
python3 src/main.py 7203
```

终端应显示“报告生成成功（Mock 模式）”，项目会生成：

```text
reports/7203_report.md
```

## 3. 尝试其他样例

```bash
python3 src/main.py 8306
python3 src/main.py 9432
```

每个股票代码会生成独立报告。

## 4. 验证报告

打开生成的 Markdown 文件，确认包含：

- 股票代码、公司名称和行业。
- 标有“模拟”的价格、PER、PBR 和股息率。
- AI 风格总结、风险提示和数据来源说明。
- “不构成投资建议”的明显声明。

## 5. 常见问题

### 为什么不需要安装依赖？

第一版只使用 Python 标准库，`requirements.txt` 不包含第三方包。

### 为什么只有三个股票代码？

第一版使用固定 Mock 数据，目的是先理解程序结构。输入其他代码时，程序会列出当前支持的代码。

### 为什么数据和市场行情不同？

所有数字都是模拟值，不来自交易所或行情服务，也不会自动更新。

### 可以用它交易吗？

不可以。项目没有账户登录、下单或自动交易能力，报告也不是投资建议。
