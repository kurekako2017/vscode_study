# 日本现场应用与案件关键词

这一章解决的问题是：把技术学习和日本现场常见表达对齐，知道哪些能力更容易落地、哪些关键词更常出现在案件里。

## 1. 现场更常见的应用形态

日本企业里常见的生成 AI 落地方向通常是：

- 社内検索
- ナレッジ検索
- FAQ / 問い合わせ対応
- 文档摘要
- 翻译和邮件草拟
- 资料整理和流程辅助
- RAG API / 生成 AI 后端 PoC

`AI Agent` 热度在上升，但很多现场仍然先从可控的 RAG、分类、摘要、API 集成开始。

## 4. Agent 在现场里的位置

现场说的 `Agent` 通常不是单纯聊天，而是：

- 能调用工具
- 能根据状态推进任务
- 能连接内部 API 或资料
- 能在固定流程里辅助执行

所以学习 Agent 前，最好先具备：

- 结构化输出
- RAG
- FastAPI / API 集成
- 基础评估意识
