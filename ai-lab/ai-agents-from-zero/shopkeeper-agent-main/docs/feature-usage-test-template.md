## 1. 功能一览

| 功能 | 测试输入 | 预期结果 |
| --- | --- | --- |
| Mock 问数链路 | `统计华北地区销售额` | 返回模拟 SSE，包含 progress 和 result |
| 真实问数链路 | `统计 2025 年第一季度各大区的 GMV` | 返回真实 SSE，最终带 SQL 和查询结果 |

## 3. 最短测试例子

### 3.1 Mock 问数链路

- 输入：
  ```text
  统计华北地区销售额
  ```
- 预期结果：
  - 先返回多段 `progress`
  - 最后返回 `result`
  - `result` 里有示例 SQL 和示例数据

### 3.2 真实问数链路

- 输入：
  ```text
  按会员等级统计 2025 年第一季度的订单数和销售额
  ```
- 预期结果：
  - 返回真实 SSE
  - SQL 和结果和语义匹配

### 3.3 元数据知识库构建

- 输入：
  ```bash
  python3 -m app.scripts.build_meta_knowledge -c conf/meta_config.yaml
  ```
- 预期结果：
  - 终端显示构建完成
  - `meta` 库里有表、字段、指标
  - Qdrant 和 ES 中有索引数据

### 3.4 前端问数页面

- 输入：
  - 打开前端页面
  - 输入 `统计华北地区销售额`
- 预期结果：
  - 页面能显示执行步骤
  - 页面能显示 SQL 摘要
  - 页面能显示结果表格

### 3.5 NAS MySQL 连接

- 输入：
  ```bash
  mysql -h 192.168.10.2 -P 3306 -u root -p
  ```
- 预期结果：
  - 能登录
  - 能看到 `meta` 和 `dw`

## 4. 快速排错

1. 先看 `.env`
2. 再看后端启动日志
3. 再看基础服务是否在线
4. 再看 NAS MySQL 是否连通
5. 再看前端是否连到正确地址
