# 案例快速运行与概要

这份文档把本仓库里可直接运行的案例集中到一处，方便你按“案例简介 + 启动命令”快速定位。

## 表格说明

- 所有案例表格都新增了 `运行条件` 列。
- `纯本地可跑` 表示不依赖云端模型接口，直接在本机就能执行。
- `OpenRouter free 可用` 表示这类云端模型脚本已经整理成当前环境可直接跑的方式；如果代码里做了回退，通常会先试 OpenRouter，失败后再退到本地 Ollama。
- `需要专用 Key（OpenRouter free 不适用）` 表示这类脚本仍然绑定特定厂商接口，不能直接用 OpenRouter free 代替。
- `必须本地服务（Ollama）`、`必须本地服务（Redis / 外部服务）`、`需要专用向量化 Key` 表示还要额外服务或专用接口。

## 启动说明

### 根目录运行版

适合直接按仓库里的相对路径启动案例。

```bash
cd /home/victorkure/workspace/vscode_study/ai-lab/ai-agents-from-zero
python3 -m pip install -r requirements.txt
```

### 绝对路径运行版

适合不想切目录，或者当前终端里 `python3` 容易被别的虚拟环境抢走的时候使用。

```bash
/usr/bin/python3 -m pip install -r /home/victorkure/workspace/vscode_study/ai-lab/ai-agents-from-zero/requirements.txt
```

如果你已经有自己的 `.env`，就直接运行对应案例即可。没有的话，先复制 `.env-example` 为 `.env` 再补环境变量。

> 下面的案例表格主要用于浏览，不建议直接拖选复制；如果你要复制到终端，请优先用下面的“复制版命令”或末尾附录。
