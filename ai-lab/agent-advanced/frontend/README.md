# frontend

这个目录是 `agent-advanced` 的前端专题区。

这里主要放两类内容：

- 前端学习说明
- 前端可运行 demo

## 业务场景说明

- 适用场景：当 AI 能力需要给真实用户使用，而不是只停留在后端接口或命令行时。
- 如果不用这种方式：后端能力虽有了，但业务人员看不到交互效果，落地和验收都会受限。
- 解决的问题：说明浏览器端如何接入 AI 服务，让“模型能力”变成可操作的产品界面。

## 目录角色

- `frontend/` 本身：讲 Agent / RAG 的前端接入方式
- `frontend/chat_ui_demo/`：最小可运行的 React 聊天界面示例

## 你应该怎么用

先看前端专题文档，再跑 `chat_ui_demo`，理解：

- 输入框怎么接用户问题
- 对话历史怎么显示
- 引用来源怎么展示
- 结果怎么从“纯文本”变成“有结构的 UI”
