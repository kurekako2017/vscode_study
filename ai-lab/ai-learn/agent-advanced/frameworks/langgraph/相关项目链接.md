## 学习建议

1. 先画出 State、Node、Edge 和条件路由。
2. 跑基础 workflow，确认状态如何更新和循环如何终止。
3. 再看 graph team，理解多个角色如何共享状态。
4. 最后学习 checkpoint、interrupt、恢复和完整项目。

## 注意事项

- 多 Agent 不等于必须使用 LangGraph；请区分概念协作与真实图编排。
- 企业示例涉及 checkpoint 和 interrupt 时，要验证线程标识、恢复点和人工审批。
- 不要移动大型项目；它们含测试、Docker、前端、SQLite runtime 和内部模块路径。
