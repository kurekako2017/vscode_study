# 项目简介

doc_qa_agent 是一个本地文档问答示例项目，目标是验证“先检索本地文档，再基于命中文档片段回答问题”的完整流程。

这个项目的核心流程很简单：

1. 读取当前目录下的 `.md` 和 `.txt` 文档。
2. 将长文本按固定长度切成多个 chunk。
3. 用问题关键词和 chunk 内容做简单匹配。
4. 把命中的片段交给模型生成回答。
5. 在输出中显示来源标签，便于追溯。

这个文档适合测试以下问题：

- 项目简介是什么
- 这个项目做什么
- RAG 的基本流程是什么
- 文档如何被读取和切分

如果程序能正确读取这个文件，就应该能在回答中命中“项目简介”“RAG”“文档读取”“文档切分”等关键词。
