# 长文档示例

本文件用于验证 chunk 切分、重叠以及检索排序。

doc_qa_agent 的设计目标是把本地资料变成可检索的知识源，然后围绕问题挑出最相关的片段。为了实现这一点，系统先扫描文档目录，再把文本切成多个 chunk，然后根据关键词重合度计算分数，最后把分数最高的几个片段拼接为上下文。

在测试时，可以反复检查以下点：

- chunk 是否按固定大小切分
- chunk 之间是否保留重叠内容
- 多个 chunk 命中时，分数是否能正确排序
- 来源标签是否能显示成 `相对路径#chunkN`
