# knowledge-base-draft

这是《小売業向け AI 経営分析システム》开发与日本面试准备手册。

知识库围绕一个统一项目展开：**小売業向け AI 経営分析システムの開発**。

项目背景统一设定为日本小売業客户的经营分析业务，覆盖销售分析、库存分析、商品分析、市场调查、Research Agent、日文经营报告生成、API 设计、系统设计、Docker 和日本现场 Review。
