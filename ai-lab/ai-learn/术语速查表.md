# AI Learn 术语速查表

| 术语 | 最短解释 | 首个示例 |
| --- | --- | --- |
| Prompt / Message | 提供给模型的指令与消息 | `chat_cli` |
| Structured Output | 按 schema 返回结构化数据 | `structured_output_demo` |
| Runnable / LCEL | 可组合的 LangChain 执行单元 | `runnable_composition_demo` |
| RAG | 先检索证据，再基于证据回答 | `doc_qa_agent` |
| Tool Calling | 模型选择工具，程序负责执行 | `tool_agent_demo` |
| Workflow | 预先约束节点和执行顺序 | `workflow_agent` |
| StateGraph | 用状态、节点和边描述工作流 | `langgraph_workflow_demo` |
| Checkpoint | 保存线程运行状态以便恢复 | `enterprise_graph` |
| HITL | 在高风险步骤暂停并等待人工决定 | `enterprise_graph` |
| MCP | Client 与 Server 之间的工具/资源协议 | `mcp/client.py` |
| Multi-Agent | 多个职责节点共享状态并协作 | `graph_team_demo` |
| Deep Research | 计划、检索、证据、写作和审校闭环 | `deep_research_demo` |
| Tracing | 记录调用链、耗时、错误和成本 | `tracing_demo` |

更完整的 Agent 架构说明见 [Agent 系统框架与术语](./agent-lab/Agent系统框架与术语.md)。
